Two Types of Python Loops

GCSE CS Cheat-sheet
1. WHILE Loop (Condition-Controlled)
# Repeat while condition is TRUE
count = 0
while count < 5:
    print("Count:", count)
    count = count + 1
Runs until count reaches 5. If you forget to increment, you get an infinite loop!
2. FOR Loop (Count-Controlled)
# Repeat for a set list of items
names = ["Aisha", "Daniel", "Eva"]
for x in names:
    print("Hello", x)
Runs exactly 3 times (once for each name in the list). Safe from infinite loops.
