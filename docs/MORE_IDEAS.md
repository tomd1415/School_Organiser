*** More Ideas to include ***
1. Have a scetion to record teaching concepts and ideas to include i lessons.
1.1 Add these ideas to lessons as developed by AI where i makes sense.
1.2 Add to existing lessons without extending the length of lessons.

2. Use information gathered about where the teacher gets up to in a lesson for a particular class to help develop amount of content for new lessons with that class.

3. Allow the teacher to add specific instructions to modify the style of lessons or features of the lessons. 

4. I would like the option to turn on a feature that uses a more expensive AI LLM to spot check random lessons and lesson plans to see if they could be improved in any way and recording what they found in a way that can be used by the usual LLM models that are being used. Maybe set at 1 in 3 schemes of work and 1 in 6 lesson plans.

5. Allow the teacher to select different models from a selection of models from the provider (updated list on request) with pros and cons of each model for that feature it is being selected for.

6. Feedback from teachers was that this is a bit overwhelming so I would like a small redesign where the day to day features are clear and easy to select but all of the setup and options are hidden behind a menu option and broken up and explained in a friendly way. I see this being installed and setup for each teacher by an expert and then used by the teacher with minimum input from the teacher other than about the lessons etc.

7. I would like guidance about information for the teacher when recording general notes about the group, text boxes with prompts like 'number of pupils with visual impairment' and therefore 'Minimum font size'. 'Does this group contains pupils with very short attention spans' and similar prompts and guided questions (all are optional and can be altered as the years go on).

8. The ability to trigger a complete review of all lessons and schemes (or parts of schemes) that have not been taught to make sure that they are the best they can be for the class, given the latest information available (either for a class or for all classes). 

9. A place to upload official documents for courses like the GCSE Specification and examiners report and past exam papers. These should also be referenced when designing lessons and schemes.

10. The ability to create complete schemes of work for courses and make sure to cover everything needed without missing anything. Then to be able to make sure that none of the alterations during the course miss anything. Make sure to leave space for revision on courses that have exams.

11. The separation of the workings of the system with the user interface that the user sees. This is so that the user interface can be easily redesigned without affecting (or even risking affecting) the other parts of the system and so that different teachers with different priorities can have different UIs
