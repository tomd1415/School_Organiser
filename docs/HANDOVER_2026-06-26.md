### Privacy NON-NEGOTIABLES (carry through every phase)
- **No pupil name to AI** — generation is cohort/spec-point content; marking sends only **redacted,
  slot-lettered** answers via the wrapper's `context[]` (slot→pupil map server-side, like `services/marking.ts`).
  Inputs go in `context[]`, **never** the `system` string.
- **Safeguarding-flagged answers withheld entirely** (`guardMatch` → `disclosure=true`, never sent).
- `is_test` cohort/analytics reads filter `WHERE NOT is_test`; `wipeTestAttempts` clears Test-Lab attempts.
- **Tests never call the real provider** (key forced empty in test/integration config). The AI smoke is a
  throwaway self-cleaning dev script (`app/scripts/X-smoke.ts`), deleted after.
