# Yes or no questions

## Today we are learning
- tell a yes/no question from an open question
- make up a yes/no question about an object
- split a group of objects into two groups

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: attribute · value · yes/no question · object · group. Same routine as every lesson.

## Two kinds of question
- a YES/NO question only needs a yes or a no
- an OPEN question needs a longer answer

> 🧑‍🏫 Examples: "Is it red?" (yes/no) vs "What is your favourite colour?" (open). Pupils sort questions on the starter worksheet. Likely error: treating "What colour is it?" as yes/no — it has many answers, so it is open.

## An attribute splits a group
- an ATTRIBUTE is one thing about an object, like "can it fly?"
- "Can it fly?" splits these into: can fly / cannot fly

> 🧑‍🏫 Model the split: only the balloon can fly. One yes/no question makes two groups.
