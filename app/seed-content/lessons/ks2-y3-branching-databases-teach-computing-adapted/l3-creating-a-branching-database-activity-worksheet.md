## What we are learning
Today I will build a branching database online and test that it works.

## Build it in j2data Branch
Open j2data Branch. You will choose some objects, then split them with yes/no questions.

Put the build steps in the right order.

```order
Choose a library set of objects
Click 6 to 8 objects to add them
Press the Sort button
Type a yes/no question
Drag the objects into thumbs-up or thumbs-down
Add more questions until each object is on its own
Press Play to test it
```
