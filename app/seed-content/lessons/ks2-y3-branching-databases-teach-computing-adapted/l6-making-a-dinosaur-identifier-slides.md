# Making a dinosaur identifier

## Today we are learning
- build my dinosaur branching database online from my plan
- test a partner's identifier and check it works
- suggest real-world uses for branching databases

## Build it online
- open j2data Branch (ncce.io/branchingdb)
- choose your dinosaurs and type your yes/no questions
- screenshot your finished tree

> 🧑‍🏫 Likely error: typing a question that is not yes/no — each question can only be answered yes or no. Scaffolding: the step-by-step handout from Lesson 3. TA: prompt, do not do it for them.

## Test it
- swap with a partner
- one picks a dinosaur in secret
- the other uses their tree to identify it
- swap roles

> 🧑‍🏫 If the tree cannot reach a dinosaur, that is useful feedback — they add a question to fix it.

## Real-world uses
- identify things (a plant, an animal)
- find a fault in a machine
- help spot a health problem

> 🧑‍🏫 The activity worksheet sorts real jobs a branching database could do. Plenary: each pupil names one real-world use.
