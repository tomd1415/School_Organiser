## What we are learning
Today I will compare two branching databases and explain why the order of questions matters.

## A well-structured tree
This branching database splits the minibeasts into even groups. You never need more than a few questions to reach any minibeast.

![A minibeast branching database]({{res:l4-minibeast-branching-database.png}})

A different tree picks off one object at a time, like this:
- Q1: Is it the bee? → no
- Q2: Is it the ant? → no
- Q3: Is it the snail? → no … and so on

## Put the questions in a good order
Start with a question that splits the whole group in half. Order these questions from best-first.

```order
Can it fly? (splits all the minibeasts in two)
Does it have legs? (splits the next group)
Does it have more than 8 legs? (splits a smaller group)
Is it the spider? (picks out just one)
```
