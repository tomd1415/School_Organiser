# Conversion notes — KS4 IT and the world of work (Teach Computing — adapted)

- **Slug:** `ks4-it-world-of-work-teach-computing-adapted`
- **Course / key stage:** KS4 IT & Digital Skills · **KS4**
- **Source:** TeachComputing `KS4_non_GCSE/unit_2` (IT and the world of work, KS4 v1.2), Lessons 1–6 + the
  unit guide and the 20-question summative assessment. © Raspberry Pi Foundation, OGL v3.0.
- **Lessons authored:** 6 (each: starter worksheet + activity worksheet + slide deck + media; L6 also has the
  end-of-unit quiz worksheet that folds in the summative).
- **Self-verify:** PASS — every activity/quiz worksheet renders a screenshot (image) field; Support and
  Challenge slices differ on every level-sectioned worksheet (the end-of-unit quiz is a shared assessment
  with no level sections, by design); every slide deck parses ≥4 slides with non-empty teacher notes; every
  `{{res:}}` placeholder maps to a declared manifest file and every declared file exists on disk.
- **Question types used:** text, multiple-choice (single), multi-select, matching, fill-in-the-blank,
  card-sort, order/sequence, screenshot, checklist (9 interactive kinds — good variety for a discussion-heavy
  non-programming unit). No Parsons/label/scale (no clean unlabelled diagram in source for a label task — see
  image-gap log).
- **Adaptation:** show-your-work is office/cloud-based (this unit has no MakeCode/DB tool): pupils screenshot
  a shared doc, an email, a saved cloud file, a settings screen, or a work-space design. SEND defaults applied
  throughout (low-arousal slides, I-do/we-do/you-do, S/C/C on the same task, TA fix-words on every activity).
- **Summative fold-in:** the source unit's 20-question multiple-choice summative is delivered as L6's
  end-of-unit quiz worksheet, trimmed to 17 items across the three unit strands (workplace / communication +
  cloud / networks + well-being). The four source "tick all that apply" items are authored as real
  multi-select questions; the rest are single-correct multiple-choice — no single-radio multi-correct traps.
- **Heading-gotcha check:** no worksheet/slide uses the words support/core/challenge in a non-level heading,
  so the level slicer is not tripped. (The unit has no "Challenge"-named tasks to rename.)
