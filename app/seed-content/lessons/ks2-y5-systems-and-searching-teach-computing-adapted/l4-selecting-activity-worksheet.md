## What we are learning
Today I will learn what a web crawler does and how a search engine uses an index.

## An index
A book index lists words and the page they are on. You do not read every page — you look up the word. A search engine uses an index too.

## A web crawler
A web crawler is a PROGRAM. People also call it a 'bot' or a 'spider'. It visits web pages and writes down the words it finds in the index.

![A web crawler — a spider bot]({{res:l4-web-crawler.png}})

The index is kept in huge buildings full of computers, called data centres.

![A data centre]({{res:l4-data-centre.png}})
