# Systems

## Today we are learning
- a system is made of parts that work together
- name the input, process and output of a digital system
- computers in a system talk to other devices

> 🧑‍🏫 Read the three "I can…" aloud, slowly. Words on the board: system · part · input · process · output. Same routine as every lesson.

## Starter — a bike is a system
![A bike]({{res:l1-bicycle.png}})

A system = parts that work together to do a job.

> 🧑‍🏫 Ask pupils to name bike parts. The handlebars + front wheel work together to steer. This matches the label task on the starter worksheet.

## A digital system
![A washing machine]({{res:l1-washing-machine.png}})

- INPUT = you press a button / turn the dial
- PROCESS = the machine washes the clothes
- OUTPUT = clean clothes + the display

> 🧑‍🏫 Likely error: "the display is the input." Fix-words: "input is what YOU put in; output is what comes OUT." Pupils sort parts on the activity worksheet.

## Your turn — design a talking teddy
![A talking teddy]({{res:l1-talking-teddy.png}})

- inside parts: a button, a tiny computer, a speaker, a battery
- order the steps: squeeze paw → computer checks → speaker plays a sound

> 🧑‍🏫 Support: choose the input and the output. Core: order the three steps. Challenge: add a second job (light up) and name its output. TA: prompt, do not do it for them.

## Computers talk to each other
![A smart locker]({{res:l1-amazon-locker.jpg}})

- a smart locker has parts in different places
- you get a code → key it in → the right door opens
- the parts send messages to each other

> 🧑‍🏫 This is a bigger system than the washing machine. The locker talks to the shop's computer so the shop knows the parcel was collected.

## I can…
Tick your three "I can…". Tell me one input and one output on your screen.

> 🧑‍🏫 Note who can name an input and an output. Movement break before the next lesson.
