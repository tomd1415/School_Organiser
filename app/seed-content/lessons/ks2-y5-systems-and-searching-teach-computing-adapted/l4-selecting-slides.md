# Selecting search results

## Today we are learning
- why we need search engines to find things
- what a web crawler does
- how a search engine uses an index

> 🧑‍🏫 Read the three "I can…" aloud. Words on the board: index · crawler · bot · search engine. Have a book with an index ready.

## Starter — so many websites
- 1991: 1 website
- today: more than 1,000,000,000 websites
- far too many to look at one by one

> 🧑‍🏫 Let the scale land. Ask: how could you ever find one page in a billion? Pupils multi-select good ways on the starter worksheet.

## An index
- a book index lists words and the page they are on
- you look up the word instead of reading every page
- a search engine uses an index in the same way

> 🧑‍🏫 Pupils use a real book index to find a word fast. This is the key idea behind the whole lesson.

## A web crawler
![A web crawler — a spider bot]({{res:l4-web-crawler.png}})

- a crawler is a PROGRAM (a 'bot' or 'spider')
- it visits pages and writes the words into the index
- it is not a real spider — it is software

> 🧑‍🏫 Common misconception: a crawler is a physical thing. Stress: it is a program. The index is stored in data centres.

## Your turn — be a crawler
![A data centre]({{res:l4-data-centre.png}})

- make a small index of the classroom: item → where it is
- order the search steps: crawl → write to index → you type → look in index → show pages
