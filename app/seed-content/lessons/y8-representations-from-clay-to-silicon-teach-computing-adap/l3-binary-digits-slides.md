## Starter — storing "cat"
Computers store everything with just two symbols. Which two?

> 🧑‍🏫 Link back to L2: characters → 0s and 1s, often using ASCII.

## A bit is just another set of symbols
- letters: 26 symbols
- digits: 10 symbols
- bits: 2 symbols — 0 and 1

> 🧑‍🏫 "Binary digit" shortened to "bit". 0s and 1s, bits, binary digits — all the same thing.

## Length = number of bits
1011 has a length of 4 bits.

> 🧑‍🏫 Count the symbols, exactly like counting letters in a word.

## Each new bit DOUBLES the count
- 1 bit → 2 sequences
- 2 bits → 4 sequences
- 3 bits → 8 sequences

> 🧑‍🏫 Likely error: thinks adding a bit ADDS 2. Fix-words: "each new bit DOUBLES it — ×2, not +2." Build 2-bit → 3-bit by adding a 0 then a 1 in front.

## But why binary? — clay to silicon
Silicon chips store bits as reliable on/off. Two states are hard to mix up.
