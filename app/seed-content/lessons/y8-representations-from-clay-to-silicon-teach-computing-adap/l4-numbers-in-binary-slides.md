# Numbers in binary

## Today we are learning
- how place values let bits make a number
- convert binary to decimal
- convert decimal to binary

> 🧑‍🏫 Read the three "I can…" aloud. Keep the 16-8-4-2-1 grid on the board all lesson.

## Starter — the boins
Coins worth 16, 8, 4, 2, 1 — one of each. Which coins make 12?

![Binary coins worth 16, 8, 4, 2 and 1]({{res:l4-boins-binary-place-value-coins-teach-computing.png}})

> 🧑‍🏫 12 = 8 + 4. Picking coins is exactly picking which bits are 1.

## Decimal place value
Each column is a weight: …, 100, 10, 1 — powers of ten.

> 🧑‍🏫 We add digit × weight without thinking. Binary is the same idea with different weights.

## Binary place value
The columns are 16, 8, 4, 2, 1 — powers of two.

> 🧑‍🏫 Worked example: 11010 → 16 + 8 + 2 = 26. Point to each column. Support fills the missing weights.

## Binary → decimal (we do)
Add up the place values where there is a 1.

> 🧑‍🏫 Do 100, 110, 1011 together, then pupils continue on the worksheet.

## Decimal → binary (you do)
Biggest place value first: if it fits, write 1 and subtract; else write 0; repeat.

> 🧑‍🏫 Likely error: forgets a 0 column or reads right-to-left. Fix-words: "fill EVERY column 16-8-4-2-1, use 0 when a value does not fit." Ask "does 8 fit in what's left?" — do not convert it for them.
