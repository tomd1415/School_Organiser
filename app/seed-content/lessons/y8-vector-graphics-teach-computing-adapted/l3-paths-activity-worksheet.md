## What we are learning

## A path and its nodes
A path is a route between points. A **node** marks the start, the end, and every change of direction. Drag a node and the path bends.

## Steps — draw a closed path
Drag these steps into the right order.

```order
Choose the pen tool (Draw Bezier curves and straight lines).
Click on the canvas to place the start node.
Click again to place each corner node.
Click back on the start node to close the shape.
Choose Edit paths by nodes, then drag a node to a new place.
```
