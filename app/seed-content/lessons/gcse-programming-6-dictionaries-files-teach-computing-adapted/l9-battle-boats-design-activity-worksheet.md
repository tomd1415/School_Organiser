## Python ↔ pseudocode — match each one
The Teach Computing pseudocode is a clear, language-free way to plan code.

| Python code | Pseudocode |
|---|---|
| `score = 5` | (  ) `score <- 5` (  ) `OUTPUT score` (  ) `WHILE …` |
| `print(score)` | (  ) `score <- 5` (  ) `OUTPUT score` (  ) `WHILE …` |
| `while not_won:` | (  ) `score <- 5` (  ) `OUTPUT score` (  ) `WHILE …` |
