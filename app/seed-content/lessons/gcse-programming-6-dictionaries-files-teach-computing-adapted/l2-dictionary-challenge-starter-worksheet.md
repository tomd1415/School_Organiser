## What we are learning
Today I will use a **dictionary** as a secret code (a **Caesar cipher**) to encrypt a message.

## Starter — crack the secret message
A **Caesar cipher** shifts every letter forward by the same amount. This message used a shift of **+2** (A→C, B→D, …).

