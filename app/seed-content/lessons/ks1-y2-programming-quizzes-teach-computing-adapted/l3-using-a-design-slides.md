## A new start block
![The Start on tap block]({{res:l3-start-on-tap.png}})

- before, you used the **green flag**
- the **Start on tap** block runs when you **tap the sprite**
- same job — it starts the program

> 🧑‍🏫 I-do. Show a tiny program with Start on tap, then say "Hi". Likely error: pressing the flag instead of tapping the sprite → fix-words "this one needs a tap on the sprite."

## Read the design
![The seasons design]({{res:l3-seasons-design.png}})

- a **design** tells you what to build
- it shows the **background**, the **sprite** and the **algorithm**
- read it before you start

> 🧑‍🏫 We-do. Read one row together. The worksheet ORDER task is: tap the cat → say the season → change the background.

## Choose the blocks
- **Start on tap** to start
- **Say** to talk
- **Change page** to change the background

> 🧑‍🏫 The worksheet sort matches each block to its job. Model finding the Say and Change page blocks.

## Build it
- follow the design, one row at a time
- the seasons go spring → summer → autumn → winter
- check it works by tapping the cat

> 🧑‍🏫 You-do. Pupils build the seasons animation and screenshot it. Support: build one season. Challenge: all four.
