## What we are learning
Today I will find the start of a sequence and run my program.

## The start of a sequence
A **sequence** is things that happen in order. Every sequence has a **start**.
Put the school day in order. Drag the steps into order.

```order
The gate opens
The children line up
Everyone walks to class
The lesson begins
```

## Start the program
A program will not move until you start it. In ScratchJr you press the **green flag** to start.
