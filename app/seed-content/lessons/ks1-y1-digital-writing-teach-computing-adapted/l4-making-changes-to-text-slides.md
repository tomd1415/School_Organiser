# Making changes to text

## Today we are learning
- select a word by double-clicking
- select all the text by clicking and dragging
- change the font

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: mouse · select · font. Same routine. Open last lesson's saved poster.

## Recap — bold, italic, underline
- B = bold, I = italic, U = underline
- press the button, then type, then press it again

> 🧑‍🏫 Quick recall of the toolbar buttons from Lesson 3.

## Select a word
- double-click a word to select it
- the word goes into a blue box
- now you can change just that word

> 🧑‍🏫 Model double-click. If a triple-click selects a paragraph by accident, click outside to undo the highlight. Likely error: changing text without selecting it first — fix-words "select first, then change."

## Change the font
![A word processor toolbar showing the font box]({{res:l4-word-processor-toolbar-font-box-teach-computing.png}})

- click and drag to select ALL the text
- click the arrow by the font box
- choose a new font

> 🧑‍🏫 Model click-and-drag to select everything, then the font menu. Scrolling a list is new for this age — model it on the real software.

## Your turn — change your poster
- select all your text
- change the font of your lost toy poster

> 🧑‍🏫 Support: change the font one word at a time using double-click. TA: prompt, do not do it for them. They screenshot the new font onto the activity worksheet. Save at the end.

## I can…
- tick your three "I can…"
- show me your new font
