# Developing keyboard skills

## Today we are learning
- open my work from a file
- use the arrow keys to move the cursor
- delete letters

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: keyboard · computer · cursor · backspace. Same routine as every lesson.

## Mouse or keyboard?
- the MOUSE points and clicks and draws
- the KEYBOARD types letters and words

> 🧑‍🏫 Quick recap game: read a statement, pupils move to the printed mouse or keyboard. Calm, short.

## Open your saved file
- open the paint program
- click the open icon
- find your file by its name

> 🧑‍🏫 This matches the ORDER task on the activity worksheet. Remind them WHY a sensible name helped: it is easy to find. Files are in the Downloads folder by default.

## The keys that help you write
![A computer keyboard]({{res:l5-keyboard.png}})

- SPACE BAR — makes a space between words
- BACKSPACE — deletes a letter
- ARROW KEYS — move the cursor left and right

> 🧑‍🏫 This matches the LABEL task on the activity worksheet. Animate/point at each key. The cursor is the blinking line that shows where you are typing.

## Write and fix a sentence
- write a short sentence under your picture
- use backspace to delete a mistake
- use the arrow keys to move the cursor

> 🧑‍🏫 Model: type "my name is …", then fix a letter using arrow keys + backspace. Scaffold worksheet has ready-made sentences for pupils who need them. Likely error: holding backspace too long deletes too much — fix-words "one tap, one letter."

## I can…
Tick your three "I can…". Show me your sentence.

> 🧑‍🏫 Plenary: think-pair-share "why give a file a sensible name?" Reassure and keep calm.
