# Using a computer responsibly

## Today we are learning
- say why we have rules
- give a rule for using technology safely
- say how a rule helps us

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: computer · technology · rules · safe. Same routine as every lesson.

## Why do we have rules?
- to stay safe
- to keep everyone happy
- so we can learn
- to help us be good to each other

> 🧑‍🏫 Think-pair-share: "what rules do we have at school?" Note them on the board. Start with general rules before computer rules.

## Rules for using computers
- ask an adult if you are unsure
- tell a trusted adult if something upsets you
- take a break to rest your eyes
- be kind in messages

> 🧑‍🏫 This matches the card-sort (safe / not a safe choice) on the activity worksheet. Build the list WITH the class. Keep it calm and concrete.

## How does the rule help?
- a break keeps you healthy
- being kind keeps everyone happy
- saving your work keeps it safe

> 🧑‍🏫 This matches the Core matching task. For each rule pupils give, ask "how does that help us?"

## Make a rules poster
- pick one safe rule
- make a poster for it
- take a photo of it

> 🧑‍🏫 Groups make a poster for one rule (scaffold template available — fill in the missing words). They photograph it for the show-your-work box. TA: prompt ideas, do not do it for them.

## I can… — end of the unit
Tick your three "I can…".

> 🧑‍🏫 Recap the whole unit: technology is made by people to help us → the parts of a computer → using a mouse and keyboard → using a computer safely. Well done.
