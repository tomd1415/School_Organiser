# Using a computer keyboard

## Today we are learning
- say what a keyboard is for
- type my name on a computer
- save my work to a file

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: computer · keyboard · mouse · typing · save. Same routine as every lesson.

## What is a keyboard for?
![A computer keyboard]({{res:l4-keyboard.png}})

- a keyboard is for TYPING
- typing means writing letters, words and numbers on a screen

> 🧑‍🏫 Think-pair-share: "what do you use a keyboard for?" Keyboards can look different but do the same job, like the mice last lesson.

## Type your name
- open the paint program
- click the text tool (a T)
- click and drag a text box
- type your name

> 🧑‍🏫 Model it in the program. Note: clicking away from the text box finalises it. Capital-letter expectations are up to your setting. TA: prompt, do not type for them.

## Save your work
- click the SAVE icon
- type a sensible name
- keep the .png — click save

> 🧑‍🏫 This matches the ORDER task on the activity worksheet. The save icon looks the same in lots of programs. By default Paintz saves to the Downloads folder.

## Why a sensible name?
- so you can find it again
- so you know what it is without opening it

> 🧑‍🏫 This matches the multi-select on the worksheet. Ask: "why name a file sensibly?" Keep it concrete: "so you can find YOUR picture tomorrow."

Tick your three "I can…". Show me your typed name.

> 🧑‍🏫 Plenary: review one pupil's saved work. Which tools did they use — colour, text, anything else?
