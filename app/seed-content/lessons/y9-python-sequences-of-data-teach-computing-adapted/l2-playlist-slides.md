# Playlist — operations on lists

## Today we are learning
- add and remove items from a list
- find an item's index and count items
- reorder a list with sort and reverse
- choose the right operation for a task

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: list · index · append · remove · index · count · sort. Same routine as every lesson.

## Starter — find the item
A list is numbered from 0, so `planets[2]` is the THIRD planet.

> 🧑‍🏫 Use the starter worksheet. Zero-based counting is the usual source of confusion — keep coming back to it.

## Operations on lists
A list method is written after the list, with a dot: `planets.append("Mars")`.

> 🧑‍🏫 Use the card-sort: which operation adds, removes, finds/counts, or reorders. Point out the dot notation — the list before the dot is what the operation acts on.

## The solar system  (you do)
Tell the story of the planets by adding Uranus and Neptune, then removing Pluto.

> 🧑‍🏫 Support: pick the append line. Core: fill the method name + write the remove line. Likely error: typing `planets = append(...)` — fix-words "the list goes before the dot."
