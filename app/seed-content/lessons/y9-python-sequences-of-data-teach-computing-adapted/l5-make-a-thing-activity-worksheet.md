## Sums in a for-loop
A counter or a sum starts at a value, then changes a little each time round the loop.

`numbers = [2, 4, 8, 16]`
`total = 1`
`for number in numbers:`
`  total = total + number`
`print(total)`

## Count the vowels — for on a string  (we do)
A string can be looped over one character at a time. These jumbled lines count the vowels in a sentence. Drag them into order.

![A Python program that counts the vowels in a sentence using a for loop]({{res:l5-for-strings-code-teach-computing.png}})

```parsons
vowels = "aeiouAEIOU"
print("Enter a sentence:")
sentence = input()
vowel_count = 0
for character in sentence:
  if character in vowels:
    vowel_count = vowel_count + 1
print(vowel_count, "vowels in this sentence")
```
