# Animate your name

## Today we are learning
- choose an action to repeat for each letter
- say what the repeated action should do
- say how well my animation worked

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: animation · costume · sprite · loop · at the same time · evaluate.

## What is an animation?
![An animated name on the stage]({{res:l3-animate-name-stage.png}})

- an animation changes the look again and again
- a **loop** makes it repeat
- each letter is its own **sprite**

> 🧑‍🏫 Starter. Show the animated name. Ask: what is repeating? (the look of each letter). Keep it calm — no fast flashing on the board.

## The blocks I will use
![Looks blocks]({{res:l3-scratch-looks.png}})

- **next costume** — change the look
- **change colour effect** — change colour
- put them **inside a loop** to repeat

> 🧑‍🏫 I-do. Build one letter: green flag → forever loop → a Looks block. Likely error: forgetting the loop so it runs once → fix-words "no loop, it happens once."

## All at the same time
- every letter has its own loop
- all started by the green flag
- so they animate **at the same time**

> 🧑‍🏫 We-do. This is the L3 key idea: two or more loops running together. Show two letters animating at once.

## Your turn — build and run
- choose an action for each letter
- add a loop to each one
- click the green flag
- screenshot it running

> 🧑‍🏫 You-do. Support: choose the block that repeats. Core: match action to outcome. Challenge: explain why letters move together. TA: prompt, do not do it for them.

## Evaluate
- did it work the way you wanted?
- rate it 1 to 5
- write one change

> 🧑‍🏫 Plenary. Pupils self-rate and self-tick the three "I can…". Note who can explain the outcome of their loop. Next lesson: modifying a game.
