# Different loops

## Today we are learning
- tell a forever loop from a count-controlled loop
- change a loop to make a given outcome
- choose the right loop for a job

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: forever loop · infinite loop · count-controlled loop · repeat · at the same time.

## A forever loop
![A forever loop block]({{res:l2-forever-loop-block-teach-computing.png}})

- **forever** = never stops
- keeps doing the inside blocks
- good for things that go all game

> 🧑‍🏫 Starter. A forever loop is also called an infinite loop. Example: a fish swimming back and forth. It only stops when you click the red stop.

## A count-controlled loop
![A repeat loop block]({{res:l2-repeat-count-controlled-loop-block-teach-computing.png}})

- **repeat 3** = do it 3 times, then stop
- you choose the number
- good when you know how many times

> 🧑‍🏫 We-do. Pupils match each loop to its job, then sort jobs into forever / count a number on the activity worksheet. Likely error: using forever when you want it to stop → fix-words "if it must stop, count it."

## Change a loop
- swap **forever** for **repeat 10**
- or swap **repeat** for **forever**
- click the green flag and watch
- screenshot the change

> 🧑‍🏫 You-do. Support: choose which loop never stops. Core: fill the repeat number. Challenge: why a flashing star uses a forever loop. TA: prompt, do not do it for them.

## Two things at once
- each sprite can run its own loop
- they all run **at the same time**
- the bat flaps while the fish swims

> 🧑‍🏫 Show two sprites each with a loop. Name it: more than one process at once. This is the L2 key idea pupils often miss.
