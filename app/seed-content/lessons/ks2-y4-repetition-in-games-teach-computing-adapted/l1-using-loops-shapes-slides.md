# Using loops to create shapes

## Today we are learning
- list an everyday job as steps that repeat
- read a loop and say what it will draw
- change a loop to draw a shape

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: repeat · loop · count-controlled loop · predict · code · block. First lesson of the unit — set the routine.

## Things that repeat
![A clock]({{res:l1-clock.png}})

- a clock ticks the same way again and again
- **repeat** means do it again
- a **loop** repeats the blocks inside it

> 🧑‍🏫 Starter. Pupils sort everyday jobs into repeats / happens once. Keep examples concrete (stairs, brushing teeth). Link: a loop in code does the same — saves writing blocks over and over.

## The Scratch screen
![The Scratch screen]({{res:l1-scratch-interface.png}})

- **Blocks** on the left — drag them out
- **Code area** in the middle — build here
- **Stage** top right — where it runs
- **green flag** starts the program

> 🧑‍🏫 I-do. Name each part by pointing. Pupils label the screen on the activity worksheet. Keep it slow; let them repeat the words.

## Read the repeat block
![Repeat 4 draws a square]({{res:l1-repeat-square.png}})

- **repeat 4** = do it 4 times
- move, then turn — four times
- 4 sides = a **square**

> 🧑‍🏫 We-do. Predict together: what would repeat 3 draw? (a triangle). Likely error: thinking the number is the size → fix-words "the number is how many times, not how big."

## Your turn — change the loop
- change the **repeat** number
- change the **turn** number
- click the green flag and watch
- screenshot your shape

> 🧑‍🏫 You-do. Support: choose the shape a loop draws. Core: match number to shape. Challenge: find the turn for a triangle (120). TA: prompt, do not do it for them.

## I can…
Tick your three "I can…". Show me one shape you drew.

> 🧑‍🏫 Note who can predict before running. Next lesson: forever loops and count-controlled loops.
