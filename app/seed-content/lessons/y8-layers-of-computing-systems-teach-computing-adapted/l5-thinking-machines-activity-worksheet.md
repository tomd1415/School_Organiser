## What we are learning
Machine learning trains a model by showing it lots of examples: gather → train → test.

![Google Teachable Machine — start a new project]({{res:l5-google-teachable-machine-new-project-teach-computing.png}})

## Put the steps in order
Drag the steps to train Teachable Machine into the right order.

```order
Choose your categories (for example, apple and orange).
Gather lots of example images for each category.
Train the machine using your examples.
Test the machine with a new image.
If it gets it wrong, add more examples and train again.
```

## Sort it — programming or machine learning?
Drag each one into the right group.

```sort
Ordinary programming: you write exact step-by-step instructions, the rules are fixed by the coder
Machine learning: you show the computer many examples, the computer learns patterns from the data
```
