# Orchestra conductor

## Today we are learning
- how the processor, memory and storage work together
- put the steps of running a program in order
- what an operating system is and what it does

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary: processor · memory · storage · operating system · fetch · execute · load.

## Starter — the conductor
The processor is like an orchestra conductor: it controls and times everything.

> 🧑‍🏫 'Forbidden words' recap from last lesson. Choose the processor's job on the starter worksheet. The processor EXECUTES (runs) instructions.

## When you open a program  (I do)
![A planetarium program]({{res:l3-planetarium-night-sky-teach-computing.png}})

Storage → memory → processor. The program is copied into memory, then run one instruction at a time.

> 🧑‍🏫 Walk the journey slowly. Likely error "it runs straight from storage" → fix-words "it is COPIED into memory first." Storage = remembers; memory = the working desk.

## The operating system
The operating system is the program that controls the computer and shares the hardware between programs.

> 🧑‍🏫 It is the first program to load when you power up. It looks like many programs run at once, but ONE processor switches between them very fast.
