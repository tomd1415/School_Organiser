## What we are learning
Today I will plan a counter, then make it and test it in MakeCode.

## 1. Plan your counter (the algorithm)
An **algorithm** is a plan written in order. Put the steps for a counter in order.

```order
Decide what you will count
Make a variable called count
Set count to 0 at the start
Add 1 to count when button A is pressed
Show the number on the screen
```

## 2. Make the variable
Open MakeCode and start a **New Project**.

![The MakeCode start screen with New Project]({{res:l1-makecode-home.png}})

Click **Variables**, then **Make a Variable…** and name it **count**.

![The Variables blocks: Make a Variable, set count to 0, change count by 1]({{res:l1-variables-panel.png}})
