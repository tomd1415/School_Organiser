## What we are learning
Today I will learn what a variable is, and how the micro:bit buttons can change it.

## The micro:bit
A **micro:bit** is a tiny computer. We write programs for it in **MakeCode**. It has buttons we can press. These are called **inputs**.

## A variable holds one value
A **variable** is a box that holds **one** value. The value can change while the program runs. In this lesson our variable is called **count**. It will count how many times we press a button.
