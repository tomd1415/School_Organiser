# Making a micro:bit counter

## Today we are learning
- a variable holds one value that can change
- the micro:bit buttons (inputs) can change a variable
- we can plan, make and test a counter

> 🧑‍🏫 Read the four "I can…" aloud, slowly. Words on the board: micro:bit · MakeCode · variable · input · count. Same routine as every lesson.

## Starter — what is a micro:bit?
- a micro:bit is a tiny computer
- we write programs for it in MakeCode
- it has buttons we can press — these are inputs

> 🧑‍🏫 Pass a micro:bit round if you have one. Likely error: "a micro:bit is a phone." Fix-words: "a micro:bit is a tiny computer we program." Support pupils choose what a micro:bit is on the starter worksheet.

## A variable holds one value
- a variable is a box that holds one value
- the value can change while the program runs
- our variable is called count — it counts button presses

> 🧑‍🏫 Use a real box: put in one number card, swap it for another — the box still holds one value. Likely error: "a variable holds lots of numbers." Fix-words: "one value at a time." Challenge: what happens to the old value? (it is replaced).

## Inputs — the buttons
- an input tells the micro:bit to do something
- button A, button B, and A+B together are inputs
- we will press a button to add 1 to count

> 🧑‍🏫 Pupils tick the inputs on the starter worksheet (multi-select; the battery is the odd one out — it is power, not an input).

## Plan first, then code
![The Variables blocks: Make a Variable, set count to 0, change count by 1]({{res:l1-variables-panel.png}})

- plan the algorithm: set count to 0 → add 1 on a press → show the number
- open MakeCode → New Project → Variables → Make a Variable → count

> 🧑‍🏫 Model "Make a Variable… → count". Pupils order the plan steps, then the blocks (Parsons) on the activity worksheet. TA: prompt, do not drag the blocks for them.

## Make and test your counter
![The finished counter program in MakeCode]({{res:l1-counter-program.png}})

- set count to 0 at the start
- change count by 1 when a button is pressed
- show number count, then test with the play button

> 🧑‍🏫 Pupils make the counter and screenshot it (show-your-work + MakeCode link). Likely error: forgetting "set count to 0" so it never resets. Fix-words: "start at zero." Challenge: add an A+B reset.

## I can…
Tick your four "I can…". Tell me one thing: what does "change count by 1" do?

> 🧑‍🏫 Listen for "adds 1 to the counter". Movement break before the next lesson.
