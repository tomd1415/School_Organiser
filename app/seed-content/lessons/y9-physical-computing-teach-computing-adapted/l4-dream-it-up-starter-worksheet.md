## Starter — input, processing or output?
Every feature of a project has three parts: it takes **input**, it does some **processing**, and it gives an **output**. Drag each step of a guessing game into the right group.

```sort
Input (what it senses): a button is pressed, the board is tilted left
Processing (what it works out): make the guess smaller, check if the number is found
Output (what it shows): show the guess on the display, show a tick when correct
```
