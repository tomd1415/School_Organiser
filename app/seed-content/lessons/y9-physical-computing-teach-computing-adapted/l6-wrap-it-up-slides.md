## Starter — small board or full computer?
The micro:bit is a **microcontroller**. A Raspberry Pi is a **full computer**.

> 🧑‍🏫 Use the starter sort. Microcontroller = one program for one job, low power. Full computer = many tasks at once.

## Finishing touches  (you do)

> 🧑‍🏫 Allow 10–15 minutes to finish, then diary. Use the rubric's Implementation and Ambition rows while they work — these do not need the diary.

## Document and reflect
Write down: what you built · the problems you solved · one top tip.

> 🧑‍🏫 The diary is a big part of the project assessment. Encourage honest reflection on the gap between the design and the finished prototype.

## Other platforms
- Raspberry Pi — a full computer, the size of a card
- Arduino — a microcontroller, used with added parts

> 🧑‍🏫 Plenary research: look for similarities and differences. Microcontrollers (micro:bit, Arduino) run one job; the Pi is a full computer. A full summative quiz is set separately for individual assessment.
