# Data protection

## Today we are learning
- name ways data can be stolen
- match an attack to what it does
- spot the signs of a phishing scam
- name ways to protect my data

> 🧑‍🏫 Read the "I can…" aloud. Vocabulary: phishing · malware · ransomware · hack · brute force · keylogging. Same routine.

## How data is stolen
- phishing — a fake message that tricks you
- malware — harmful software
- ransomware — locks your files for money
- brute force — guessing passwords

> 🧑‍🏫 Use the starter matching task. Keep each one to one short line. Likely error: thinking malware and phishing are the same — fix-words: "phishing TRICKS you; malware is SOFTWARE."

## Spot the scam
- it rushes you ("URGENT", "now")
- it asks for passwords or bank details
- it has a strange link or spelling mistakes

> 🧑‍🏫 This is the spot-the-scam multi-select. Show the example message on the activity worksheet. Read the warning signs aloud together.

## What to do
1. Stop — do not click
2. Check the sender and link
3. Give NO personal information
4. Report it and delete it

> 🧑‍🏫 This matches the order task. Make "Stop" the strongest habit. Reassure: it is never your fault if you receive a scam.

## Protect your data
- strong, different passwords
- two-factor login
- check a link before clicking

> 🧑‍🏫 This is the safe/risky card sort. TA: prompt, do not do it for them. A trusted adult is always a safe first step.
