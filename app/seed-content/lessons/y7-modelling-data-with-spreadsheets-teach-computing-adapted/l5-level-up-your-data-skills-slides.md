# Level up your data skills

## Today we are learning
- sort and filter data
- use COUNTIF
- use AVERAGE
- use IF

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary: header · filter · average · criteria · condition.

## Starter — warm-up
Recap: functions start with =. Name MAX, MIN, COUNTA.

> 🧑‍🏫 Pupils complete the starter worksheet. Quick recap of cell-reference ranges too.

## Sort with headers
Sorting puts the rows in order. Keep the header row at the top.

> 🧑‍🏫 Model select-all then sort. Likely error: the header gets sorted into the data. Fix-words: "tell it my data has headers."

## Filter
A filter hides the rows that do not match your rule. The data is not deleted.

> 🧑‍🏫 We do: filter to show only the top ten.

## COUNTIF — count by a rule
= COUNTIF(B2:B20, ">10") counts the cells greater than 10.

> 🧑‍🏫 Change the range and the rule to match your sheet.

## AVERAGE — the mean
= AVERAGE(B2:B20) finds the mean (middle) value.

> 🧑‍🏫 Format the answer to one or two decimal places.

## IF — show different text
= IF(B2>10, "Hooray!", "Too low") shows one of two messages.

> 🧑‍🏫 Read it as "if this is true, show the first, otherwise show the second".
