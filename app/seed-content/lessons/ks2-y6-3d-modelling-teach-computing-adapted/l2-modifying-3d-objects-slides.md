# Modifying 3D objects

## Today we are learning
- to use the handles on a 3D object
- to resize an object in three dimensions
- to lift and lower a 3D object
- to recolour a 3D object

> 🧑‍🏫 Read the four "I can…" aloud. Words on the board: handles · resize · lift · lower · recolour. Recap: last lesson we added and moved shapes.

## Handles
![A cube with handles]({{res:l2-a-cube-with-resize-handles-teach-computing.png}})

- click a shape and small squares appear — these are handles
- corner handles change width and length
- side handles change one side
- the top handle changes height

> 🧑‍🏫 Likely error: "any handle just makes it bigger." Fix-words: "each handle changes a DIFFERENT thing." On the activity worksheet pupils label the handles; Support choose which handle changes height.

## Resizing in three dimensions
- drag a handle to make a shape bigger or smaller
- hold Shift to resize evenly in all three dimensions
- the number that appears tells you the size

> 🧑‍🏫 Core question: which handle makes a cube taller but keeps the base the same? (the top handle). Model it once, slowly.

## Lifting and lowering
![A roof lifted onto a base to make a house]({{res:l2-a-roof-lifted-onto-a-base-to-make-a-house-teach-computing.png}})

- the top cone handle lifts a shape up or down
- it does not change the size
- 0 mm means the shape is sitting on the workplane

> 🧑‍🏫 Challenge: make a triangular prism sit neatly on top of a cuboid (a house). Pupils order the steps and use the lift number to place the roof. TA: prompt, do not do it for them.

## Changing colour
![The colour picker]({{res:l2-the-tinkercad-colour-picker-teach-computing.png}})

- select a shape, then click the colour box
- choose a new colour
- you can add shapes in different colours

> 🧑‍🏫 Give pupils a moment to recolour their house. Keep choices calm — no need for many bright colours.

## 3D printing
![A 3D printed cube showing print layers]({{res:l2-a-3d-printed-cube-showing-print-layers-teach-computing.jpg}})

- models made on a computer can be printed on a 3D printer
- the printer builds the model in thin layers
- one printed object can only be one colour

> 🧑‍🏫 Plenary. Point out the thin layer lines on the printed cube. Pupils self-tick the four "I can…" and name one thing they changed about their shape.
