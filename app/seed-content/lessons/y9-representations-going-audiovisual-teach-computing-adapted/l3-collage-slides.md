# Collage

## Today we are learning
- edit a picture using software
- editing is arithmetic on the picture's numbers
- one creative benefit of editing
- one ethical drawback of editing

> 🧑‍🏫 Read the four "I can…" aloud. This is a hands-on lesson — work at your own pace.

## Spot the edit
The same photo, four ways. Which part is the original?

![One photo shown four ways: greyscale, scattered, blurred, original]({{res:l3-image-manipulation-comparison-teach-computing.jpg}})

> 🧑‍🏫 The far-right band is the original — but the answer matters less than the point: it is hard to tell what is real.

## Editing means changing numbers
A picture is stored as numbers. To edit it, the software does arithmetic on those numbers.

![The GIMP image-editing program]({{res:l3-gimp-logo-teach-computing.png}})

> 🧑‍🏫 Start everyone with the Setup and Lighthouse worksheets, then let them choose Owl / Ada / Penguin / Size-and-colours.

## Follow the steps
Open the program → open the picture → choose a part → apply a change → export.

> 🧑‍🏫 Support: a printed step strip. Challenge: combine two edits. Circulate; if someone is stuck, suggest a different task and come back.

## Creative — but careful
Editing is creative and useful. It can also be used to mislead.

> 🧑‍🏫 Think–write–pair–share: one good use and one harmful use. Link to "is this image real?" Sort task on the worksheet.

## Your turn
Edit a picture, then paste a screenshot on the activity worksheet.

> 🧑‍🏫 Ask pupils to show you each finished task before moving on.
