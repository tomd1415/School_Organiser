## Starter — predict this program
A **subroutine** is a named block of code you can run (call) whenever you need it. Read this program before you run it.

![A calculator program that calls a subroutine]({{res:l1-calculator-program-teach-computing.png}})

```python
def calculate(a, b):
    answer = a + b
    print(f"{a} + {b} = {answer}")

num1 = 10
num2 = 15
calculate(num1, num2)
```
