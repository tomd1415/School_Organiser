## What we are learning
Today I will learn the **structured approach**: each block has **one way in and one way out**.

## Starter — count the exit points
A block of code should have **one entry point** (where it starts) and **one exit point** (where it finishes). Read this function.

```python
def and_function(a, b):
    if a == True and b == True:
        return True
    else:
        return False
```
