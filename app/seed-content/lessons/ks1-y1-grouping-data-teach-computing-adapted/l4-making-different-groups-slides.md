# Making different groups

## Today we are learning
- group similar objects
- group objects in more than one way
- count how many objects share a property

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: group · object · property · value · colour · data set. Same routine as every lesson.

## One object, many labels
![A frog]({{res:l4-frog.png}})

- the frog is green AND an animal
- so it could go in a "green" group OR an "animals" group

> 🧑‍🏫 Show the slide out of Present mode if you drag objects. An object has more than one property, so it fits more than one group.

## Group by colour
![A cat]({{res:l4-cat.png}})

- put all the GREEN objects together
- the rest go in "not green"

> 🧑‍🏫 I-do: model grouping by colour. "is green" / "is not green" language helps.

## Re-group the SAME objects
- now group by what they ARE: animals and cars
- same objects, a different group

> 🧑‍🏫 We-do: the big idea of the lesson — the SAME objects can be grouped more than one way, depending on the property you pick.

## Count how many share a property
- pick a property, like "blue"
- count how many objects have that property
- write the number — this is a small DATA SET

> 🧑‍🏫 Use whole numbers, not tally charts. A data set is a collection of related data (here: the counts).

## Your turn
- group the shapes by colour
- then group the SAME shapes by size
- count each group

> 🧑‍🏫 You-do the two card sorts on the activity worksheet, then real shapes. Scaffold: offer "red / not red" if two groups is easier. TA: prompt, do not do it for them.

## I can…
Tick your three "I can…". Show me the same objects grouped two ways.

> 🧑‍🏫 Note who can re-group the same objects by a different property.
