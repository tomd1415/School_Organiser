## What we are learning
Today I will group objects and then count each group.

## Do it in order
Put the steps in order to count a messy pile of objects.

```order
Look at all the objects
Sort them into groups
Count one group
Count the next group
```

## Sort the objects into groups
Drag each object into its group.

```sort
Cars: red car, blue car
Frogs: green frog, tree frog, toad
```
