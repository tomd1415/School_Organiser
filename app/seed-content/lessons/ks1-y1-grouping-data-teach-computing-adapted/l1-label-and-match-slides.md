# Label and match

## Today we are learning
- give an object a label (a name)
- match an object to a group
- find the label for a group of objects

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: object · label · group · search · image. Same routine as every lesson.

## A name is a label
![A red car]({{res:l1-red-car.png}})

![A blue car]({{res:l1-blue-car.png}})

- a LABEL is a name we give to an object
- both cars have the label "car"

> 🧑‍🏫 Point at each car. Same label, different colour. An object can have many labels — today we use its name.

## Why do objects have labels?
- labels make objects easy to talk about
- labels make objects easy to find
- a computer search uses labels too

> 🧑‍🏫 Think-pair-share: "Why do objects have labels?" Take a few answers, then land on: easy to talk about and find.

## The computer is not clever
![A cat]({{res:l1-cat.png}})

- the computer shows cats because a person LABELLED each picture "cat"
- no label means the computer cannot find it

> 🧑‍🏫 Misconception: "the computer knows it is a cat." Fix-words: "a person typed the label — the computer just matches it."

## Match objects to a group
- objects with the SAME label go in the same group
- cars go with cars · cats go with cats

> 🧑‍🏫 We-do one together, then you-do the card sort on the activity worksheet. TA: prompt, do not sort it for them.

## Label the group
- a group has a label too (a group name)
- a group of cats has the label "cats"

> 🧑‍🏫 Challenge: an object can fit more than one group (a red car → "cars" or "red things").
