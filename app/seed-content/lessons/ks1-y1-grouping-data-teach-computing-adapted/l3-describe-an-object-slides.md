# Describe an object

## Today we are learning
- describe an object
- describe a property of an object
- find objects with similar properties

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: object · property · label · colour · shape · size. Same routine as every lesson.

## Ways to describe an object
- COLOUR — red, blue, green
- SHAPE — round, square, long
- SIZE — big, small

> 🧑‍🏫 A property is a way to describe an object. Think-pair-share words for each property. Add their words to the board.

## Describe the car — colour
![A red car]({{res:l3-red-car-teach-computing.png}})

- this car is RED
- "red" is its colour
- colour is a PROPERTY

> 🧑‍🏫 I-do: name the colour, then reveal the word "property". A property is data we collect about the object.

## Describe the ball — shape
![A tennis ball]({{res:l3-tennis-ball-teach-computing.png}})

- this ball is ROUND
- "round" is its shape
- shape is a PROPERTY too

> 🧑‍🏫 We-do: ask "what is the same about a ball and a coin?" (both round) — same property, different object.

## One object, many properties
![A parrot]({{res:l3-parrot-teach-computing.png}})

- this parrot has MANY colours
- you usually label the main colours only

> 🧑‍🏫 Misconception: you do not have to label every tiny colour. Pick the main ones. Challenge: name three properties of the parrot.

## Properties let us group
- objects with the SAME property go together
- group all the RED objects, or all the ROUND objects

> 🧑‍🏫 You-do: the "complete the property" and "sort by property" tasks on the activity worksheet. TA: prompt, do not do it for them.

## Find similar properties
- a "similar properties hunt": find objects the same colour
- this is how a computer search finds pictures with the same label

> 🧑‍🏫 Activity 3: pairs find three objects the same colour and take a photo. Drawing is just a quick sketch — not the focus.

## I can…
Tick your three "I can…". Tell me one property of an object on your table.

> 🧑‍🏫 Note who can name the property (colour/shape/size), not just the value (red/round/big).
