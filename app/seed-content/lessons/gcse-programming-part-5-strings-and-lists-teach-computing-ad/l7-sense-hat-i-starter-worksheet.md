## What we are learning
Today I will use a **list** to control the **LED matrix** on a Sense HAT, using **RGB** colours.

## Starter — array or list?
A **Sense HAT** is a board for the Raspberry Pi with an 8×8 grid of LEDs (the **LED matrix**). We use a list to set the colours.
