## What we are learning
Today I will learn what a **GUI** is and how **event-driven** programs are different from the programs I have written so far.

## Starter — two ways a program can run
A **procedural** program runs line by line, top to bottom. An **event-driven** program waits for the user to do something (like click a button) before it runs a block of code. A **GUI** (graphical user interface) is event-driven.
