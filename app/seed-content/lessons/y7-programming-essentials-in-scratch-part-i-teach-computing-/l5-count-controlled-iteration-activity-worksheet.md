## What we are learning
Today I will use a **count-controlled loop** (`repeat`) to make a program shorter, and I will **debug** a program by tracing a variable.

## Predict
A `repeat 10` loop runs the blocks inside it ten times. A variable can change each time.

## Build it — order the counting program
Drag the blocks into an order that counts from 0 to 9 using a loop.

