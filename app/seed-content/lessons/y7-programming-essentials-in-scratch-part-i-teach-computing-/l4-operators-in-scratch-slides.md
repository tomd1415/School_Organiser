## Starter — what will it output?
The program checks `score > 30`. Work out true or false.

> 🧑‍🏫 Pupils know `>` from maths. Mark true/false on the starter worksheet. Edge case: 30 is not greater than 30.

## Comparison operators
`>` greater than · `<` less than · `=` equal to. They **compare** two values.

> 🧑‍🏫 Read each example aloud; ask what it checks. Keep it concrete (ages, scores).

## Logic operators
![A condition: value = King and suit = hearts]({{res:l4-operators-comparison-and-logic-blocks.png}})

`and` needs **both** true · `or` needs **at least one** true · `not` flips true and false.

> 🧑‍🏫 Optional playing-card stand-up: read a Boolean expression, pupils move to true/false. Build up complexity slowly.
