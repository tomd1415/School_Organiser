## What we are learning
Today I will describe what a pixel is, describe colour depth and image resolution, and say what metadata is.

## Colour depth — how many bits, how many colours
The colour depth is the number of bits used for each pixel. More bits means more colours. Drag each colour depth onto the number of colours it can make.

```sort
1 bit per pixel: 2 colours
2 bits per pixel: 4 colours
3 bits per pixel: 8 colours
8 bits per pixel: 256 colours
```

## Same photo, two colour depths
The first picture uses a high colour depth (lots of bits per pixel). The second uses only 1 bit per pixel, so each pixel can only be black or white.
