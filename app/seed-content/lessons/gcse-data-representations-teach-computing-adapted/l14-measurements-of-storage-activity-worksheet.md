## What we are learning
Today I will define the units of storage, compare them to the 'bibyte' units, and convert between units.

## The units (smallest to largest)
Each unit is 1,000 times bigger than the one before it (after the byte).

| Unit | Equals |
|---|---|
| bit | a single 0 or 1 |
| nibble | 4 bits |
| byte (B) | 8 bits |
| kilobyte (kB) | 1,000 bytes |
| megabyte (MB) | 1,000 kilobytes |
| gigabyte (GB) | 1,000 megabytes |
| terabyte (TB) | 1,000 gigabytes |
| petabyte (PB) | 1,000 terabytes |
