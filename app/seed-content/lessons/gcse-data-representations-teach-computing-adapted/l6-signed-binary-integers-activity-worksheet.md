## Two's complement: decimal → binary — put the steps in order
Drag the three steps into the right order.

```order
Write the positive value of the number in binary
Flip every bit (0 becomes 1, 1 becomes 0)
Add 1 to the result
```

## Worked examples
- **Sign and magnitude**, 4-bit: `1010` → sign bit 1 (negative), magnitude `010` = 2, so **-2**.
- **Two's complement**, 4-bit (place values -8 4 2 1): `1010` = -8 + 2 = **-6**.
