# Number bases

## Today we are learning
- the difference between base-2 and base-10
- read the place values in a binary number
- convert binary to decimal
- convert decimal to binary

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: base 2 · base 10 · decimal · binary · place value. Same routine.

## Starter — tiny switches
- A **transistor** is a tiny on/off switch.
- Millions fit across a human hair.
- We group switches to represent numbers.

> 🧑‍🏫 The hair image gives a sense of scale — millions of transistors fit across it. Support: tick the digits decimal/binary use.

## Base-10 and base-2
- **Decimal** (base-10) uses ten digits: 0–9.
- **Binary** (base-2) uses two digits: 0 and 1.
- In decimal the columns ×10 each time (1, 10, 100…). In binary they ×2 (1, 2, 4, 8…).

> 🧑‍🏫 I-do. Connect to their maths: place value is the same idea, just doubling instead of ×10.

## The place-value grid
- 8-bit place values: **128 64 32 16 8 4 2 1**.
- Put a **1** where you use that value, **0** where you do not.
- Add up the 1s to get the decimal number.

![An 8-bit place-value grid showing 10110100]({{res:l2-place-value-grid.png}})

> 🧑‍🏫 We-do. The grid shows 10110100 = 128+32+16+4 = 180. On the worksheet pupils label the place values (drag) and sort digits by base.

## Binary → decimal  (we do)
- `101` = 4 + 1 = **5**
- `1001` = 8 + 1 = **9**
- `1111` = 8 + 4 + 2 + 1 = **15**

> 🧑‍🏫 Say each binary number digit by digit: "one, zero, one" — NOT "one hundred and one". Common slip: reading binary as a decimal number.

## Decimal → binary  (you do)
- Find the biggest place value that fits, put a 1, subtract.
- Repeat with what is left. Put 0 in columns you do not use.
- Example: 10 = 8 + 2 → `1010`.

> 🧑‍🏫 You-do on the worksheet. Support: which place values make 5. Core: convert 10. Challenge: 10110100 → 180, and 68 → 01000100. TA: prompt, do not do it for them. Screenshot the working.
