# Binary addition

## Today we are learning
- count in binary
- the four golden rules of binary addition
- add two binary numbers
- add three binary numbers

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: binary · decimal · addition · carry. Same routine. Always say "one, zero" not "ten".

## Starter — decimal to binary
- 5 = `101`
- 9 = `1001`
- Count up: 0, 1, 10, 11, 100, 101…

> 🧑‍🏫 Retrieval from last lesson. Support: tick 5 = 101. The pattern when counting up sets up the carrying idea.

## The four golden rules
- `0 + 0 = 0`
- `0 + 1 = 1`
- `1 + 1 = 10`  (zero, carry 1)
- `1 + 1 + 1 = 11`  (one, carry 1)

> 🧑‍🏫 I-do. Say each rule slowly. Biggest misconception: reading `10` as ten. Fix-words: "one, zero — that means carry."

## Add a column at a time  (we do)
- Start at the **right**.
- Add the column with the rules.
- Two-digit answer → write the right digit, **carry** the left.

> 🧑‍🏫 We-do `101 + 11`: right 1+1=10 write 0 carry 1; middle 0+1+1=10 write 0 carry 1; left 1+0+1=10. Answer 1000 = 8. Check 5+3=8.

## Your turn  (you do)
- Support: order the column steps; tick 1+1 = 10.
- Core: `100 + 1`, `110 + 11`, `1010 + 1111`.
- Challenge: add three numbers, e.g. `111 + 111 + 111`.

> 🧑‍🏫 You-do on the worksheet. Likely error: forgetting to add the carry into the next column. Fix-words: "carry first, then add." TA: prompt, do not do it for them. Screenshot the working.
