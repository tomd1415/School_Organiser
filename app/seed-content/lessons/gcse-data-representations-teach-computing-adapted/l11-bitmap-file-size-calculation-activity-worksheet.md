## What we are learning
Today I will calculate the file size of a bitmap image, and describe how the number of pixels and the colour depth change the file size.

## The formula
The file size of a bitmap image in bits is:

**width in pixels × height in pixels × colour depth**

To change the answer from bits into bytes, **divide by 8**.

## Worked example
An image is 8 × 8 pixels with a colour depth of 2.
- 8 × 8 × 2 = 128 bits
- 128 ÷ 8 = 16 bytes
