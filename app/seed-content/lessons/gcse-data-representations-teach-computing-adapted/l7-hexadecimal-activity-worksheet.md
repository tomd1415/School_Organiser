## Where is hexadecimal used?
Hex is a short, tidy way to write long binary numbers. You see it in colour codes, network (MAC) addresses, and memory dumps.

## The nibble method — put the steps in order
To convert a hex number to decimal the fast way:

```order
Split the hex number into its two digits
Turn each hex digit into a 4-bit nibble
Join the two nibbles into one 8-bit binary number
Add up the place values to get the decimal number
```

## Worked example
`A2` → A = `1010`, 2 = `0010` → `10100010` → 128 + 32 + 2 = **162**.
