## What we are learning
Today I will explain what compression is and why it is used, and define lossy and lossless compression.

## Two types of compression
- **Lossy** compression makes a file smaller by **permanently removing** some data. The file can never be made exactly the same again. Used for photos (JPEG) and music (MP3).
- **Lossless** compression makes a file smaller with **no data lost**. The file can be made exactly the same again. Used for PNG images, ZIP files and text.
