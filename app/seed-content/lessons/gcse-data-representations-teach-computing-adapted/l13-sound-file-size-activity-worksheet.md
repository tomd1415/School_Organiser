## What we are learning
Today I will calculate the file size of a sound file, and describe how sample rate, duration, and sample resolution change its size and quality.

## The formula
The file size of a sound file in bits is:

**sample rate (Hz) × sample resolution (bits) × duration (seconds)**

Remember: 1 minute = 60 seconds, 2 minutes = 120 seconds, 3 minutes = 180 seconds.

## Worked example
A 2-second clip, sample rate 1,000 Hz, sample resolution 4 bits.
- 1,000 × 4 × 2 = 8,000 bits
