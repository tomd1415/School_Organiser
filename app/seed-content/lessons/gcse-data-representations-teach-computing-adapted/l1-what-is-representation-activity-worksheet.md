## What we are learning
Today I will tell **data** from **instructions**, and work out how many things a group of **on/off** switches can represent.

## Key words — match each to its meaning
Drag each answer to the right word.

| Word | What it means |
|---|---|
| Data | (  ) numbers or text that on their own have no meaning (  ) information on how to do a task (  ) a tiny switch that is on or off |
| Instructions | (  ) numbers or text that on their own have no meaning (  ) information on how to do a task (  ) a tiny switch that is on or off |
| Transistor | (  ) numbers or text that on their own have no meaning (  ) information on how to do a task (  ) a tiny switch that is on or off |
