## Why Unicode?
ASCII can only code English keyboard characters. **Unicode** uses more bits per character, so it can code many languages, symbols and emoji. The first 128 Unicode codes are the same as ASCII.

## How to calculate file size
**File size = number of characters × bytes per character.** Example: 10 characters in ASCII = 10 × 1 = **10 bytes**.
