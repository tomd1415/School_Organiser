## How to add a column — put the steps in order
Drag these steps into the right order for adding one column.

```order
Add the two digits in the column using the golden rules
If the answer is two digits, write the right-hand digit down
Carry the left-hand digit to the next column on the left
Move left to the next column and add again, including any carry
```

## Worked example
`101 + 11`: right column 1+1 = 10 (write 0 carry 1); middle 0+1+1 = 10 (write 0 carry 1); left 1+0+1 = 10. Answer = **1000**. Check: 5 + 3 = 8. ✔
