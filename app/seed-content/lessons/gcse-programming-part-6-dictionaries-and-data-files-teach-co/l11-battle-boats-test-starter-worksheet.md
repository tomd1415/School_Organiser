## 🟢 Support — match the test type
The grid coordinates are 0 to 7.

| Test data type | What it means |
|---|---|
| Normal | (  ) data we expect (e.g. row 4) (  ) data right at the edge (e.g. row 0 or 7) (  ) data that should be rejected (e.g. row 9 or "cat") |
| Boundary | (  ) data we expect (e.g. row 4) (  ) data right at the edge (e.g. row 0 or 7) (  ) data that should be rejected (e.g. row 9 or "cat") |
| Erroneous | (  ) data we expect (e.g. row 4) (  ) data right at the edge (e.g. row 0 or 7) (  ) data that should be rejected (e.g. row 9 or "cat") |
