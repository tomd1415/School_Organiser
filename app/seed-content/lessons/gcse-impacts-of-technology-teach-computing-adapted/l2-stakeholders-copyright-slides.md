## What is a stakeholder?
- a stakeholder is **anyone affected by a technology**, or who can change how it is used
- example: a hacked hospital — patients, staff, the public are all stakeholders

> 🧑‍🏫 Misconception: a stakeholder is NOT only the owner. Tick all the stakeholders on the worksheet — the hackers and "the weather" are the traps.

## The right to be forgotten
- part of GDPR / the Data Protection Act 2018
- a person can ask an organisation to **erase all their personal data**

> 🧑‍🏫 Only applies in some cases — e.g. when the data is no longer needed. Keep the definition short and exact.

## Copyright and Creative Commons
- copyright protects work you create (pictures, music, software source code)
- a Creative Commons licence lets the owner **share** while **keeping** copyright

> 🧑‍🏫 Worked example (I do): I made this drawing, so I own the copyright. We do: when is it legal to use someone's work? (you own it / have permission / it is out of copyright). You do: worksheet.

## Open source vs proprietary
- **open source** — source code shared, you can copy and change it
- **proprietary** — source code hidden, vendor controls it

> 🧑‍🏫 Sort task on the worksheet. Misconception to break: open source is NOT always free; proprietary is NOT always paid. Stretch: why is a "no commercial use" clause still possible on open source?
