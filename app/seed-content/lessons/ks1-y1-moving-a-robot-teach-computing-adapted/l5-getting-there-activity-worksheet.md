## What we are learning
Today I will plan an algorithm, turn it into a program, and debug it if needed.

## We do — plan, then program
Turn over a start card and a destination card. Point out the route first. Then choose your command cards.

## 🟢 Support
Put the steps of making a program in order. Drag them into order.

```order
Point out the route from start to end
Put the command cards in order
Step through it with the paper-bot
Press the buttons on the robot
```
