# Routes

## Today we are learning
- find several possible routes
- plan two programs
- use two different programs to reach the same place

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: route · plan · program. Same routine as every lesson.

## Which way?
- when we travel there is often more than one route
- think of two ways to get across school
- today we plan different routes for the robot

> 🧑‍🏫 Slide 3. Link to a familiar journey: more than one way to the hall. There is not always one "right" route.

## Going this way and that way
- pick a start square and an end square
- draw two routes between them
- both start and end on the same squares

> 🧑‍🏫 Activity 1. Pairs draw two or three routes on the planner. No robot yet — this is design. Support: start with two squares close together.

## Design your programs
- choose how the robot will face on the start square
- make an algorithm with command cards for each route
- both programs start from the same square, facing the same way

> 🧑‍🏫 Activity 2. Plan the algorithm at the side of the mat if they can; place cards on the mat if they need to. Test and debug each program. Likely error: changing the start facing between the two programs — fix-words: "same start, same facing, both times."

## Even more routes
- try a longer route with turns
- a route can go back over squares
- the robot does not have to face the way it is going

> 🧑‍🏫 Activity 3 + plenary. Challenge: from the woods to the mushroom using only Left turns and Forwards, starting facing the owl. Calm, independent. TA: prompt, do not do it for them.

## I can…
Tick your three "I can…". Show me two programs that reach the same place.

> 🧑‍🏫 Note who can plan two different working routes to the same end square.
