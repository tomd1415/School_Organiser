## What we are learning
Today I will read a program with up to four commands and predict where the robot ends up.

## We do — work it out
Put the robot on the start square, facing the top of the mat. Step through the program one command at a time.

## 🟢 Support
A program to go forward, turn, then forward again has its commands in order. Drag them into order.

```order
Forwards
Right turn
Forwards
Go
```

## 🟡 Core
The robot starts facing the top of the mat.

| Question | Choose |
|---|---|
| Program: Forwards, Right turn. After it runs, which way is the robot facing? | (  ) to the right (  ) to the left (  ) backwards |
| In the program Forwards, Right turn, Forwards — how many times does the robot MOVE to a new square? | (  ) 2 times (  ) 3 times (  ) 1 time |
