- predict where a program will stop

## Start from the same square
- every group starts on the SAME square
- this makes the robot's moves predictable
- one Forwards command = one square forwards

> 🧑‍🏫 Activity 1. Same start position is the key idea — it makes prediction possible later. Likely error: starting anywhere — fix-words: "same square, every time."

## Forwards and backwards
- one Forwards command moves one square forwards
- one Backwards command moves one square backwards
- try Forwards then Backwards — where does it end up?

> 🧑‍🏫 Activities 2–3. Backwards moves one square back. Forwards then Backwards returns to the start — let them discover it. Clear between every program.

## Where will it get to?
- put the robot on the start square
- read the program
- mark where you think it will stop
- run it and check

> 🧑‍🏫 Activity 4 — the prediction task on the activity worksheet. Use the paper-bot to step through by hand first, then run the real robot. TA: prompt, do not move it for them.
