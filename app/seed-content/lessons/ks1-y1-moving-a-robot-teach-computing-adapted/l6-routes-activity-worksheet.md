## What we are learning
Today I will write two different programs that both get the robot to the same place.

## We do — two routes, one end
Choose a start card and an end card. Draw two different routes between them. Then make a program for each route.

## 🟢 Support
Put the steps in order to plan and test a route. Drag them into order.

```order
Choose a start square and an end square
Draw a route between them
Make a program with the command cards
Test the program and fix any bugs
```
