## What we are learning
Today I will read a forwards and backwards program and predict where the robot will stop.

## We do — run a program
Put the robot on the start square. Clear first. Enter the commands. Press Go.

## 🟢 Support
Put the steps in order to run ANY program on the robot. Drag them into order.

```order
Put the robot on the start square
Press Clear (X)
Press the command buttons
Press Go
```

## 🟡 Core
The robot starts on square 0 and faces the higher numbers.

| Question | Choose |
|---|---|
| Program: Forwards, Forwards, Forwards. Which square does it stop on? | (  ) square 3 (  ) square 2 (  ) square 1 |
| Program: Forwards, Forwards, Backwards. Which square does it stop on? | (  ) square 1 (  ) square 2 (  ) square 3 |
