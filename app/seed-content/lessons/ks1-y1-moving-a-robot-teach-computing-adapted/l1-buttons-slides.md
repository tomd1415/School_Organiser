# Buttons

## Today we are learning
- predict what a command will do
- match a command to what the robot does
- run a command on the robot

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: forwards · backwards · turn · clear · go · commands. Same routine as every lesson.

## These are all robots
- a robot is a machine that does a job
- a robot does its job because a human told it to
- we tell a robot what to do with commands

> 🧑‍🏫 Show the school floor robot itself, not just the slide. Keep it calm: pass it round one table at a time. Ask: what jobs do robots do, and why?

## The buttons on the robot
![The buttons on top of the floor robot]({{res:l4-floor-robot-buttons-teach-computing.png}})

- the arrows are direction commands: forwards, backwards, left turn, right turn
- Go (green) runs the program
- Clear (X) makes the robot forget

> 🧑‍🏫 Point at each button. This matches the label task on the starter worksheet. Stress Clear (X) — it is hard to see working but very important.

## Clear, then Go
- always press Clear (X) first
- then press a command button
- then press Go to run it

> 🧑‍🏫 Demonstrate: Clear → Forwards → (pause, ask them to predict) → Go. The eyes light up when it is on. Likely error: forgetting Clear, so old commands are still in memory — fix-words: "Clear first, every time."

## Your turn — try the buttons
- press Clear (X)
- press ONE button
- press Go and watch
- did it do what you expected?

> 🧑‍🏫 One robot between two. Support: pair them and try one button at a time. TA: prompt, do not press it for them. No need for sound — turn robot volume down if it suits the class.

## Match the move to the button
- the teacher presses one button in secret
- watch the robot move
- point to the button you think was pressed

> 🧑‍🏫 Activity 2: everyone in a circle with the button sheet. Move round so every pupil gets a turn behind the robot. This is the activity worksheet matching task.

## I can…
Tick your three "I can…". Show me a button and tell me what it does.

> 🧑‍🏫 Note who can name Forwards, Backwards, Go and Clear and match a button to a movement.
