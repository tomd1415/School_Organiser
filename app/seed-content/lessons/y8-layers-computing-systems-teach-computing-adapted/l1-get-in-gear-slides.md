# Get in gear

## Today we are learning
- a computer is a device for running programs
- a program is a list of instructions that work on data
- the difference between a general-purpose computer and a purpose-built machine
- examples of programs (software) we use

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson.

## Starter — old machine or modern computer?
![An old calculating machine — the Analytical Engine]({{res:l1-analytical-engine.png}})

This old machine was built to do ONE job. Choose on your starter worksheet what a modern computer can do that it cannot.

> 🧑‍🏫 Think · pair · share. Both old machines and computers take input → process → output. The big difference: a computer can RUN DIFFERENT PROGRAMS, so it can change its job.

## A program is a list of instructions
A program tells the computer the exact steps to follow, working on data.

> 🧑‍🏫 Keep it concrete: a recipe is a list of instructions; the ingredients are the data. Vocabulary: program · instructions · data.

## Purpose-built or general-purpose?  (we do)
![A calculator — built for one job]({{res:l1-calculator.jpg}})
![A computer running a game]({{res:l1-general-purpose-computer.png}})

A calculator only does sums. A computer runs many programs.

> 🧑‍🏫 Sort everyday things together: washing machine / digital watch / microwave = purpose-built; phone / laptop / tablet = general-purpose. Likely error "a phone is just a phone" → "a phone runs MANY programs."

## Your turn — program or data?  (you do)
On the activity worksheet, sort each thing into PROGRAM (software) or DATA.

> 🧑‍🏫 Support: single-choice "which is a program?". Challenge: why is a song file data, not a program? Data is what programs work ON.

## Unplugged — the intelligent piece of paper
Follow the printed instructions like a machine to play noughts and crosses.

> 🧑‍🏫 Pupils "execute" the instructions by hand — a first feel for running a program mechanically. They screenshot/keep the finished grid.
