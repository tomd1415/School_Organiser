## Today we are learning
- name the main hardware parts of a computer
- describe what the processor, memory, storage, input/output and communication do
- all computers have the same basic parts

## The universal parts
![A processor (CPU) chip]({{res:l2-processor-chip.png}})

Every computer has: processor · memory · storage · input/output · communication.

> 🧑‍🏫 I-do: name each part and its one job. Processor = runs instructions. Memory (RAM) = holds data while working. Storage = remembers files when off. Communication = talks to other computers.

## Match the part to its job  (we do)

> 🧑‍🏫 Likely error: memory vs storage. Fix-words "memory FORGETS when the power is off; storage REMEMBERS." Keep saying it.

> 🧑‍🏫 Input = the computer senses/receives. Output = the computer shows/sends. Camera and microphone are inputs; screen, printer, speaker are outputs.
