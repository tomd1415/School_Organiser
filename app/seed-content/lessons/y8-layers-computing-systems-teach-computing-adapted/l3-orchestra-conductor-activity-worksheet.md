## What we are learning
When you open a program, it is copied from storage into memory, then the processor runs the instructions one by one.

![A planetarium program shows the night sky]({{res:l3-planetarium-night-sky.png}})

## Put the steps in order
Stella opens her image-editing app. Drag what happens INSIDE the computer into the right order.

```order
Stella clicks the app to start it.
The program is copied from storage into the main memory.
The processor fetches the instructions one by one from the memory.
The processor runs (executes) each instruction.
Stella saves her work, so the data is copied back to storage.
```
