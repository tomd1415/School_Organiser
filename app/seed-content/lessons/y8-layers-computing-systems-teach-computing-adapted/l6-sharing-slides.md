## See inside

"See inside" lets you read how a Scratch program works.

> 🧑‍🏫 The 'Naughty elf' game is secretly cheating — pupils who "see inside" find the comment that proves it. You can only trust a program if you can read its code.

## Remix
"Remix" makes your OWN copy you can change. The original is not touched, and the first author is credited.

> 🧑‍🏫 Likely error "remix changes the original" → fix-words "remix makes a COPY and credits you." Reassure: no one can meddle with your original.

## Free and open source software
Open source = you can see, use and change the code. Many everyday programs are open source.

> 🧑‍🏫 Card-sort benefit vs risk on the activity worksheet. Then pupils name one open source program for an everyday task.
