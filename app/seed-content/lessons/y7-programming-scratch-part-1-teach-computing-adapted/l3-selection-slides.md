# Selection in Scratch

## Today we are learning
- say if a condition is true or false
- what selection (an If statement) does
- where selection is needed
- change a program to use selection

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on board: condition · expression · evaluate · selection · If statement.

## Starter — true or false?
A **condition** is checked by the computer. It is either **true** or **false**.

> 🧑‍🏫 Stand-up option: true side / false side of the room. Use 3 statements. Pupils mark true/false on the starter worksheet. Settle them before the next slide.

## If… else…
![If birthday = yes then say Happy Birthday, else say A very happy unbirthday]({{res:l3-selection-if-else-blocks-teach-computing.png}})

If the condition is **true**, the **If** part runs. If it is **false**, the **else** part runs.

> 🧑‍🏫 Misconception M21/M22: the Else is NOT optional and is NOT always run — exactly one path runs. Both are written because we don't know the answer in advance.

## Predict — Chat with Big Ed  (I do → we do)
Predict what Big Ed says for "yes" and for "no".

> 🧑‍🏫 PRIMM predict then run. Investigate: try different answers. Likely error: thinks both messages show → "only one path runs each time."
