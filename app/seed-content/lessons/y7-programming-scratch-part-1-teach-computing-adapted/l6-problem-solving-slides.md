## Be a problem-solver
Re-read · check earlier work · ask a partner to prompt · then ask the teacher.

> 🧑‍🏫 If a pupil asks for help, first ask what they have already tried. Prompt, do not do it for them. A calm "planning" role is fine for anyone who finds the open task hard — plan the algorithm in blocks on paper.

## Pick the right construct  (you do)
- Repeat a move a set number of times → a **loop**
- Do one thing if a key is pressed, else another → **selection**
- Check two keys are both pressed → the **and** operator

> 🧑‍🏫 Tasks 1–4 build up: sequence → Parson's subroutine → loop → full subroutine. Support: copy the shape of the subroutine above. Challenge: the score + feedback explorer tasks.
