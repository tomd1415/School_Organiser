## What we are learning
Today I will use **comparison operators** (`>`, `<`, `=`) and **logic operators** (`and`, `or`, `not`) to build conditions.

![A condition: value = King and suit = hearts]({{res:l4-operators-and-blocks-teach-computing.png}})

## Predict

## True or false?
Decide if each condition evaluates to true or false.

| Condition | Choose |
|---|---|
| `7 > 6` | (  ) true (  ) false |
| `9 = 9` | (  ) true (  ) false |
| `10 < 9` | (  ) false (  ) true |
| `(30 < 50) or (30 > 50)` | (  ) true (  ) false |
| `(20 = 20) and (15 < 15)` | (  ) false (  ) true |
| `not (20 = 20)` | (  ) false (  ) true |
