## Build it — order the quiz code
The plenary program asks a quiz question. If the answer is right it adds to the score, otherwise it says "Incorrect". Drag the lines into a working order.

```parsons
ask [What is the capital city of Spain?] and wait
if <answer = Madrid> then
change score by 1
say [That is correct]
else
say [Incorrect]
```
