## Starter — true or false?
A **condition** is something a computer checks. It is either **true** or **false**.

| Question | Choose |
|---|---|
| The condition `7 > 6` is… | (  ) true (  ) false |
| The condition `the card is a heart`, when the card is a spade, is… | (  ) false (  ) true |
