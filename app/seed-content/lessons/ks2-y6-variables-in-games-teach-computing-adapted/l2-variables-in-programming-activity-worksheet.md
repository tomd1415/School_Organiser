## A variable has a name and a value
Every variable has two parts:
- a **name** — what the variable is called (like Score)
- a **value** — what it is holding right now (like 5)

Sort these into the name and the value.

```sort
Name: Score, Favourite_food, Home_team
Value: 5, chocolate cake, Japan
```

## A good name says what it holds
A variable is like a labelled drawer. A good label tells you what is inside. A good variable name is **short**, **says what it holds**, and has **no spaces** (use _ instead).

| Question | Choose |
|---|---|
| Which is the **best** name for a variable that holds the score? | (  ) score (  ) thing (  ) x |

## Make and change a variable in Scratch
Open the **Scoreboard** project. You will make a score and change it when a flag is clicked.

![A Scratch scoreboard with the USA and Japan flags]({{res:l2-a-scratch-scoreboard-with-two-flags-teach-computing.png}})

Put these Scratch blocks in the right order to make a score go up when the sprite is clicked.

```parsons

```
