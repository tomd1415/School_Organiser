## Make your own variable
To add a new variable in Scratch, you follow these steps. Put them in order.

```order
Click on the Variables blocks
Click Make a Variable
Type a name for your variable, like Score
Click OK to add the variable
```

## Variables hold numbers or letters
A variable can hold a **number** (like a score: 5) or **letters** (like a team name: Japan). Both of these are called the **value** of the variable.

```sort
Number: 5, 162, -2
Letters: Japan, chocolate cake, Score
```
