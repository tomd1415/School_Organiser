## What we are learning
Today I will learn that a **variable** is something that can change.

## Watching a football match
In a football match, some things stay the same and some things change. The **score** changes when a team scores a goal. The **time** changes as the game goes on.

## A thing that can change is a variable
A thing that can change, like the score, is called a **variable**. The score is a variable because it goes up and down.
