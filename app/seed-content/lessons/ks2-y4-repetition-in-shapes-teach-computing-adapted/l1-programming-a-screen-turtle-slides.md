# Programming a screen turtle

## Today we are learning
- type Logo commands to move the turtle
- explain what changing a number does
- write a code snippet to draw something

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: program · turtle · command · value · FD · BK · RT · LT. Same routine as every lesson. Logo software ready and projected.

## This is Logo
![The FMSLogo screen: a drawing area, the turtle, and a command box]({{res:l1-fmslogo-screen-drawing-area-turtle-and-command-box-teach-computing.png}})

- the **turtle** is the arrow in the middle
- you type **commands** in the **command box**
- the turtle draws a line in the **drawing area**

> 🧑‍🏫 Point at each part on the real software, not just the slide. This matches the label task on the starter worksheet. Pass control round so each pupil types one command.

## Forwards and backwards
- `FD 100` = forwards 100 steps
- `BK 50` = backwards 50 steps
- a bigger number = a longer line

> 🧑‍🏫 Demonstrate live: FD 100, then FD 200, so they see the longer line. The number is the **value**. Then type CS to clear the screen.

## Turning: right and left
- `RT 90` = turn right 90 degrees
- `LT 90` = turn left 90 degrees
- turns use **degrees**, not steps

> 🧑‍🏫 Recap 90 degrees from maths (a square corner). Likely error: forgetting the number after the command — fix-words: "every move and turn needs a number."

## Three special commands
- `CS` = clear screen (start again)
- `PU` = pen up (move without drawing)
- `PD` = pen down (draw again)

> 🧑‍🏫 These three have NO number after them. Demonstrate PU → move → PD so they see the gap in the line.

## We do — draw the digit 7
- `FD 100  RT 90  FD 200`
- start at the top-left, facing right
- what number have we drawn?

> 🧑‍🏫 Draw it on the board with a pen as you read each command. Answer: 7. Then run it in Logo to confirm. This is the Challenge order task on the activity worksheet.

## Your turn
- press Execute to run your commands
- try FD, BK, RT, LT with different numbers
- type CS to start again

> 🧑‍🏫 Support: one command at a time. Core: a short snippet. Challenge: draw a digit. TA: prompt, do not type it for them. Save a screenshot for the show-your-work box.

## I can…
Tick your three "I can…". Show me one command and tell me what it does.

> 🧑‍🏫 Note who can type a command accurately and explain what a value change does.
