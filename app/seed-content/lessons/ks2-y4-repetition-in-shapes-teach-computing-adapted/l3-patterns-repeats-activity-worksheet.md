## What we are learning
Today I will spot the repeat inside a shape and use a **count-controlled loop** to draw it.

## The long way to draw a square
To draw a square the "long" way you type the SAME two commands four times:

`FD 100 RT 90   FD 100 RT 90   FD 100 RT 90   FD 100 RT 90`

| Question | Choose |
|---|---|
| Which two commands keep repeating? | (  ) FD 100 RT 90 (  ) FD 200 LT 90 (  ) BK 100 RT 45 |

## The short way — a repeat loop
A **count-controlled loop** repeats commands a set number of times:

`repeat 4 [FD 100 RT 90]`

The number after `repeat` is **how many times**. The commands to repeat go in the **square brackets**.

| Question | Choose |
|---|---|
| In `repeat 4 [FD 100 RT 90]`, what does the 4 mean? | (  ) repeat 4 times (  ) move 4 steps (  ) turn 4 degrees |
