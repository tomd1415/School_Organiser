# Patterns and repeats

## Today we are learning
- spot repetition in everyday things
- spot patterns in a sequence
- use a count-controlled loop to draw a shape

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: pattern · repeat · repetition · count-controlled loop. Same routine as every lesson.

> 🧑‍🏫 "Repeat means to do something again." Brushing your teeth repeats the same brush stroke many times — that is repetition too.

## A square the long way
- `FD 100 RT 90` … then again … then again … then again
- you type the same two commands **four times**
- that is a lot of typing!

> 🧑‍🏫 Type it out the long way on screen so they SEE the repetition. Ask: is there a quicker way? Likely answer: do it once and say "4 times".

## A square the short way — a loop
- `repeat 4 [FD 100 RT 90]`
- the **4** = how many times
- the commands to repeat go in **[square brackets]**

> 🧑‍🏫 This is the **count-controlled loop**. Run both versions — same square, much shorter code. This matches the activity worksheet.

## We do — find the loop
- long: `FD 100 LT 120` three times
- short: `repeat 3 [FD 100 LT 120]`
- spot the repeat → count it → write the loop

> 🧑‍🏫 Model the three steps: spot, count, write. This is the Challenge order task on the activity worksheet.
