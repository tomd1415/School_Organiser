# Using loops to create shapes

## Today we are learning
- say what changing the repeat number does
- trace a loop and predict the shape
- choose which numbers to change to draw a shape

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: count-controlled loop · trace · value · sides. Same routine as every lesson.

## Changing the repeat number
- `repeat 4 [...]` repeats 4 times
- a bigger repeat number = more repeats
- to double a tower's height, double the number

> 🧑‍🏫 Starter. Answer: 8. Keep it concrete — count the blocks. This matches the starter worksheet.

## Tracing code
- read the loop one step at a time
- draw what each step does
- this predicts the shape BEFORE you run it

> 🧑‍🏫 Model tracing repeat 4 [FD 100 LT 90] on the board: line, turn, line, turn — a square appears. Pupils trace on whiteboards.

## Shapes all follow a pattern
![A table of shapes with their turn angles: triangle 120, pentagon 72, hexagon 60, octagon 45, decagon 36]({{res:l4-regular-shapes-with-their-turn-angles-and-loop-code-teach-computing.png}})

- `repeat (sides) [FD 100 RT (angle)]`
- change the **number of sides** and the **turn angle**
- the table gives you the angle for each shape

> 🧑‍🏫 Don't worry pupils with the maths of 360÷sides — give them the angle from the table. This matches the activity worksheet.

## We do — predict the shape
- `repeat 3 [FD 100 RT 120]`
- 3 sides, turn 120 — what shape?
- a **triangle**

> 🧑‍🏫 Read it aloud, predict, then run it in Logo to confirm. This is the activity worksheet sort task.

## Your turn — draw the shapes
- start with the square: `repeat 4 [FD 100 RT 90]`
- change the numbers for a triangle, then a hexagon
- use the table for the angle

> 🧑‍🏫 Support: match shape to loop. Core: change one shape. Challenge: write an octagon loop. TA: prompt, do not type it for them. Likely error: wrong angle — fix-words: "check the table." Save a screenshot.
