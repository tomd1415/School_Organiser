## What we are learning
Today I will write SMART goals for the Delicious Desserts project, and learn iteration and interaction.

## From aims to objectives
An **aim** is a big thing the project must do (like "make the integrated system"). **Objectives** are the smaller tasks that get you there.

## What does SMART mean?
Drag each letter onto its meaning.

| Meaning | Match the letter |
|---|---|
| Says exactly what to do | (  ) Specific (  ) Measurable (  ) Achievable (  ) Realistic (  ) Timely |
| You can check when it is done | (  ) Specific (  ) Measurable (  ) Achievable (  ) Realistic (  ) Timely |
| It can actually be done with what you have | (  ) Specific (  ) Measurable (  ) Achievable (  ) Realistic (  ) Timely |
| It makes sense for this project | (  ) Specific (  ) Measurable (  ) Achievable (  ) Realistic (  ) Timely |
| It has a time or a deadline | (  ) Specific (  ) Measurable (  ) Achievable (  ) Realistic (  ) Timely |
