## What we are learning
Today I will make a Gantt chart and a PERT chart, label the parts, and choose a planning tool.

## A Gantt chart
A Gantt chart shows each task as a **bar** across time. Longer bar = more time.

![A Gantt chart drawn as bars]({{res:l4-gantt-graphic.png}})

This is what a real Gantt chart looks like on paper.

![A real printed Gantt chart]({{res:l4-gantt-photo.png}})

## Label the PERT chart
A PERT chart uses **circles (nodes)** for tasks and **arrows** for the order. Drag each label onto the chart.

```label
image: {{res:l4-pert-chart.png}}
tasklist (12%, 28%): Task list
node (37%, 56%): A node (one task)
time (42%, 42%): Time in minutes
last (89%, 58%): The last task
```

## 🟢 Support
Drag each fact into the right tool.

```sort
Gantt chart is good for: short simple projects, seeing time as bars
PERT chart is good for: large projects, showing the order of tasks
```

## 🟡 Core
Put the steps to build a PERT chart in order.

```order
List your tasks in the order they happen
Draw a circle (node) for each task
Number the nodes
Join the nodes with arrows
Write the time in minutes on each arrow
```
