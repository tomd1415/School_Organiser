## What we are learning
Today I will follow a plan to build the interactive dessert selector and create the posters.

## Build the dessert selector
The selector is a slide deck. Each question slide has buttons that link to the next slide. Put the build steps in order.

```order
Open a blank slide deck
Set the house style on the master slide
Add a logo, an image and a question to each slide
Add button shapes to each question slide
Add a hyperlink from each button to the right slide
Test that every button goes to the right slide
```

## 🟢 Support
