# Start your own project

## Today we are learning
- find the user requirements of a new project
- spot the constraints
- sort activities into the right stage
- start the planning documents

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: initiate · plan · feasibility report · Gantt chart · PERT chart. Same routine as every lesson.

## Starter — sort the activities
- match each activity to its stage: initiate, plan, execute, evaluate

> 🧑‍🏫 This recalls the whole life cycle before pupils run their own project.

## The project — Team Outdoor Obstacles
- TOO runs inflatable obstacle courses
- they want a sales sheet, a poster, and a business card

> 🧑‍🏫 Similar to Delicious Desserts but simpler. Pupils now run it themselves.

## Initiate it
- analyse the brief
- find the user requirements and constraints
- start the feasibility report

> 🧑‍🏫 Likely error: jumping to making things. Fix-words: "initiate first — what do they want?" Give 25 minutes of calm work.

## Plan it
- write SMART goals
- choose a Gantt or PERT chart

> 🧑‍🏫 Both templates are provided. Most pupils pick the Gantt chart.

## Track and feedback
