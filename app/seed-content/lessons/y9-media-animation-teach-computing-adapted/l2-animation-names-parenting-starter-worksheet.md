## Two kinds of animation
- **Stop motion:** move a real object a tiny bit, take a photo, repeat.
- **Keyframe:** set a few key positions, and the computer fills in the frames between them.

Sort each card into the right kind of animation.

```sort
Stop motion: move a real object a little bit, take a photo of each tiny move, used in clay films like Wallace and Gromit
Keyframe: set a start and end position, the computer fills in the in-between frames, used in computer 3D films
```
