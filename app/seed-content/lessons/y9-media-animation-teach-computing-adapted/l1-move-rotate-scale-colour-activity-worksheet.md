## 🟢 Support
Choose the right key.

| Question | Choose |
|---|---|
| To **move** an object you press… | (  ) G (  ) X (  ) I |
| To **scale** (resize) an object you press… | (  ) S (  ) R (  ) A |
| To **delete** an object you press… | (  ) X (  ) S (  ) G |

> Tip: G = grab/move, S = scale, R = rotate, X = delete.

Put the snowman steps in the right order.

```order
Delete the cube (press X)
Add a sphere for the body
Add a sphere for the head and move it on top
Scale the head smaller (press S)
Add a cone and rotate it for the nose (press R)
Give each part a colour (a material)
```
