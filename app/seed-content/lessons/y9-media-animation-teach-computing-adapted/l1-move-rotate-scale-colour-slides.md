# Move, rotate, scale, colour

## Today we are learning
- add, delete, and move objects
- scale and rotate objects
- use a material to add colour
- make a snowman model

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson. Everyone has Blender open.

## Starter — how important is 3D animation?
![A still from the Blender film Agent 327]({{res:l1-agent327-blender-film.png}})

Where do we see 3D animation? Tick all that apply on your starter worksheet.

> 🧑‍🏫 Games, films/TV, adverts/posters — not the plain letter. Agent 327 was made in Blender, the same free software we use. Mention green screen: the green is replaced by 3D graphics later.

## This is Blender  (I do)
Blender is free, open-source 3D software. The grey box is the default cube. We delete it with **X**.

> 🧑‍🏫 Show the viewport. Switch to rendered mode so colour and light show. Delete the cube (X), then Add ▷ Mesh.

## Four moves — G, S, R, X  (we do)
- **G** = grab/move
- **S** = scale (resize)
- **R** = rotate
- **X** = delete

> 🧑‍🏫 Demo each on one object. Likely error: scaling shoots off — fix-words "press Esc, move the mouse away from the centre, then press S again."

## Your turn — build a snowman  (you do)
Two spheres (body + head), a cone for the nose. Move, scale, rotate.

> 🧑‍🏫 Support: the steps are in order on the worksheet. Challenge: add a scarf/hat/arms (explorer task). TA: prompt, do not do it for them.

## Colour with materials
A **material** is a colour you make once, name, and reuse — like a **variable** in programming.

![A coloured snowman model in Blender]({{res:l1-snowman-coloured.png}})

> 🧑‍🏫 Make one black material, reuse it on both eyes + buttons. Stress the variables link: name it once, use it many times.
