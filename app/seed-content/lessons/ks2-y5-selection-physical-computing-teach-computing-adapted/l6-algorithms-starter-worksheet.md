## What we are learning
Today I will write a clear, precise algorithm for my carousel, build it, then test and debug it.

## Order the algorithm
A good algorithm is a precise set of ordered steps. Drag these steps into order for a carousel that starts when the switch is pressed.

```order
Check for ever: is the switch pressed?
If the switch is pressed, then start
Spin the motor forwards
Flash the Sparkle 3 times
Wait 2 seconds
Turn the motor and Sparkle off
```
