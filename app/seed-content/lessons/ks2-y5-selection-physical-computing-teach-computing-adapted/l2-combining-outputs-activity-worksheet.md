## Build the program
A **count-controlled loop** ("do 4 times") repeats the blocks inside it a set number of times. Put the blocks in order.

![A count-controlled loop program]({{res:l2-count-loop-code.png}})

```parsons
program start
do 4 times
set sparkle 0 to red
wait 1 seconds
turn sparkle 0 off
wait 1 seconds
loop
turn sparkle 0 off
```

## Order the algorithm
Drag the steps into order to make the disco dancer move and flash.

```order
Spin the motor forwards at full speed
Light the Sparkle green
Wait 0.5 seconds
Turn the Sparkle off
Turn the motor off
```
