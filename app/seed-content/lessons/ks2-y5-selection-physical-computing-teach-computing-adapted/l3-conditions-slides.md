# Controlling with conditions

## Today we are learning
- explain that a condition is either true or false
- design a conditional (do until) loop
- program a microcontroller to respond to an input

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary: condition · true · false · switch · input · do until loop. Same routine.

## What is a condition?
- a **condition** is a sentence that can only be **true** or **false**
- "It is raining" — true or false. "What is the weather?" — not a condition

> 🧑‍🏫 This matches the sort task on the starter worksheet. Turn statements into yes/no questions to test them.

## A switch is an input

## Do until loop
![A do until A is HI program]({{res:l3-do-until-code.png}})

- a **do until** loop keeps repeating UNTIL its condition is true
- here it flashes until button A is pressed

> 🧑‍🏫 Likely error: mixing up while/until. Fix-words: "do UNTIL the condition is true, then stop."
