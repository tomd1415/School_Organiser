## Build the program
A **do until** loop keeps repeating UNTIL its condition is true. Put the blocks in order. This loop flashes until button A is pressed.

![A do until A is HI program]({{res:l3-do-until-code.png}})

```parsons
program start
do until A is HI
set sparkle 0 to red
wait 1 seconds
turn sparkle 0 off
wait 1 seconds
loop
turn sparkle 0 off
```
