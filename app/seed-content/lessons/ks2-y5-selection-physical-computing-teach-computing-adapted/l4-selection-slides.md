# Starting with selection

## Today we are learning
- explain that a condition being met can start an action
- identify a condition and an action
- use selection (an 'if…then…' statement) to direct a program

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary: selection · if…then · condition · action · input · output. Same routine.

## Condition and action
- "**if** it is raining, **then** put up your umbrella"
- the **condition** comes after "if" · the **action** comes after "then"

> 🧑‍🏫 This matches the sort task on the starter worksheet. The condition starting the action is the big idea today.

## What is selection?
- **selection** means: IF a condition is true, THEN do an action
- if the condition is false, the action does not run

> 🧑‍🏫 Last lesson a condition STOPPED a loop. Selection CHOOSES whether to do an action. Keep that contrast clear.

- put the if…then inside a **do forever** loop

> 🧑‍🏫 This is why infinite repetition is needed — to keep checking if the condition has been met.
