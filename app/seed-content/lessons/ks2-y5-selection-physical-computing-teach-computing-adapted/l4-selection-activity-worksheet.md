## What we are learning
Today I will use **selection** (an 'if…then…' statement) to direct a program. The action only happens when the condition is true.

## Selection in code
**If** the condition is true, **then** the actions inside the block run. Here: if button A is pressed (A is HI), then flash.

![An if…then selection program]({{res:l4-if-then-code.png}})

## Build the program
Put the blocks in order. The "if…then…" sits inside a "do forever" loop so the condition is checked again and again.

![Selection inside a forever loop]({{res:l4-if-then-loop-code.png}})

```parsons
program start
do forever
if A is HI then
set sparkle 0 to green
wait 1 seconds
turn sparkle 0 off
end if
loop
```
