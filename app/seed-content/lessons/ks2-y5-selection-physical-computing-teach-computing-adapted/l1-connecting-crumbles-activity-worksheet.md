## Order the build steps
Drag the steps into the right order to connect the Sparkle.

```order
Connect the battery box to the power pads
Connect the Sparkle + and – to the Crumble power pads
Connect the Sparkle D pad to the Crumble D pad
Switch the battery box on
Watch the Sparkle flash white 6 times
```

## Build the program
Put the program blocks in order. The "do forever" loop makes the Sparkle flash again and again — this is an **infinite loop**.

![An infinite loop program]({{res:l1-infinite-loop-code.png}})

```parsons
program start
do forever
set sparkle 0 to red
wait 1 seconds
turn sparkle 0 off
wait 1 seconds
loop
```
