## What is HTML?  (I do)
A web page is just a plain text file. HTML uses tags to mark up parts of it.

A tag sits inside angle brackets: `<h1>` opens, `</h1>` closes. The `/` marks the closing tag.

> 🧑‍🏫 `<h1>` = big heading, `<b>` = bold, `<em>` = italic. Only the text BETWEEN the tags is changed.
