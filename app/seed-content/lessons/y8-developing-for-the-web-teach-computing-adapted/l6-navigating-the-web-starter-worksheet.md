## Starter — working hyperlink or not?
A hyperlink uses the `<a href="...">` tag and must be closed with `</a>`.

| Question | Choose |
|---|---|
| Which hyperlink is written correctly? | (  ) `<a href="home.html">Home</a>` (  ) `<a href="home.html">Home<a>` (  ) `<a link="home.html">Home</a>` |
