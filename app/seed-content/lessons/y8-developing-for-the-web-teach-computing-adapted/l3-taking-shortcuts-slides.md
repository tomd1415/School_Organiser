## What is CSS?  (I do)
CSS keeps all the styling in one separate file.

A rule like `h2 { color: red; }` styles EVERY `<h2>` on the page at once.

> 🧑‍🏫 Change the CSS once → the whole site updates. The styling moves out of the HTML and into the CSS file.
