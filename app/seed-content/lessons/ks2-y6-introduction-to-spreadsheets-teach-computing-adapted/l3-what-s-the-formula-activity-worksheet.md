## What we are learning
Today I will build a formula, and see that changing an input changes the output.

## Which data can be used in a calculation?
A spreadsheet can only do maths with **numbers** (this includes currency, which is a number with a £). It cannot do maths with **plain text** (words).

Sort each cell into "can be used in a calculation" or "cannot".

```sort
Can be used: a number like 86, a currency like £0.12
Cannot be used: a word like London, a name like Alton Towers
```

## Building a formula
A **formula** tells the spreadsheet what to calculate. It always starts with **=**. Instead of typing numbers, you point to **cells** by their cell reference.

To find the journey cost, multiply **Distance in miles** (column D) by **Cost per mile** (column E). For row 2 the formula is **=D2\*E2**.

## Changing inputs and outputs
The numbers you put in are the **inputs**. The answer the formula gives is the **output**. If you change an input cell, the output updates by itself.
