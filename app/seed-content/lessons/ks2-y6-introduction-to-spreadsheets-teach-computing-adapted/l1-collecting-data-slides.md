# Collecting data

## Today we are learning
- collect data
- structure data in a table with headings
- enter data into a spreadsheet

> 🧑‍🏫 Read the three "I can…" aloud, slowly. Words on the board: data · collecting · table · structure · spreadsheet. Same routine as every lesson.

## Starter — rolling for data
![A dice]({{res:l1-dice.png}})

- data means facts and numbers we collect
- roll the dice, write down each score
- each score is a piece of data

> 🧑‍🏫 Pupils roll a dice five times and record the scores on the starter worksheet. Likely error: "data is only numbers." Fix-words: "data is any fact we collect." Support pupils choose what data means.

## Structure — a table with headings
- a pile of numbers is hard to read
- a table gives data a structure
- headings (Name, Roll 1, Total…) say what each number means

> 🧑‍🏫 Model one table with headings on the board. TA: prompt pupils to choose headings, do not write them for them. Core: pupils write a sensible heading.

## Putting data in a spreadsheet
![Data in a spreadsheet grid]({{res:l1-spreadsheet-grid.png}})

- a spreadsheet stores data in a grid of boxes
- type the headings along the top
- type the data underneath

> 🧑‍🏫 Pupils order the "make a spreadsheet" steps on the activity worksheet, then type their dice data in and screenshot it. Challenge: why keep data in a spreadsheet, not on paper?

## I can…
Tick your three "I can…". Tell me one thing: what is a heading for?

> 🧑‍🏫 Listen for "it says what the column is for". Movement break before the next lesson.
