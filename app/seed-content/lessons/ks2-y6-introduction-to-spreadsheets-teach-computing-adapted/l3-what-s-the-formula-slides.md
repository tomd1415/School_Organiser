# What's the formula?

## Today we are learning
- which data can be used in calculations
- build a formula in a spreadsheet
- changing inputs changes outputs

> 🧑‍🏫 Read the three "I can…" aloud. Words on the board: formula · calculation · input · output · cell reference. Recap: every cell has an address like A1.

## Starter — the maths symbols
![A calculator with maths symbols]({{res:l3-a-calculator-with-maths-symbols-teach-computing.png}})

- + add, − subtract
- \* multiply, / divide

> 🧑‍🏫 Stress \* = multiply and / = divide (different from × and ÷ in maths). Pupils tick the symbols on the starter worksheet.

## Numbers, not words
- a spreadsheet can do maths with numbers (and currency)
- it cannot do maths with words (plain text)

> 🧑‍🏫 Likely error: "you can add a word to a number." Fix-words: "only numbers can be calculated." Pupils sort cells into can / cannot be calculated.

## Building a formula
![A journey spreadsheet ready for a formula]({{res:l3-a-journey-spreadsheet-ready-for-a-formula-teach-computing.png}})

- a formula always starts with =
- point to cells, not numbers: =D2\*E2
- this finds the journey cost (distance × cost per mile)

> 🧑‍🏫 Model typing =D2*E2 in cell F2. Pupils build their own on the activity worksheet, then screenshot. Support pupils pick which cell can be calculated.

## Inputs and outputs
- inputs = the numbers you put in
- output = the answer the formula gives
- change an input → the output updates by itself

> 🧑‍🏫 Change £0.12 to £0.15 live; the answer changes. Challenge: why does the output update when you used cell references? TA: prompt, do not do it for them.

## I can…
Tick your three "I can…". Tell me one thing: what does every formula start with?
