# Can you edit it?

## Today we are learning
- change font size, colour, and style
- edit text on a page
- explain how changing text makes a message clearer

> 🧑‍🏫 Read the three "I can…" aloud. Same routine. Vocabulary on the board: font · font style · communicate · template. Have the invitation template open and ready.

## Desktop publishing
- page layout software that puts text and images together
- used for invitations, magazines, posters
- a ready-made page is called a **template**

> 🧑‍🏫 We use desktop publishing (e.g. Canva) this unit. A template means you do not start from a blank page. Keep it concrete: show the invitation template now.

## Three things we change about a font
- **size** — bigger or smaller
- **colour** — red, blue, any colour
- **style** — the look of the letters

> 🧑‍🏫 I-do then we-do: change one word's size, then its colour, then its style. This matches the "Sort the change" task on the worksheet. Likely error: changing the page background instead of the text — fix-words "click the words first, then change them."

## Editing keys
- **Shift** — hold for a CAPITAL letter
- **Backspace** — removes the letter to the left
- **Return / Enter** — starts a new line

> 🧑‍🏫 Builds on Year 1 typing. TA: prompt the key, do not press it for them. Likely error: typing in the wrong text box — fix-words "click into the box first."
