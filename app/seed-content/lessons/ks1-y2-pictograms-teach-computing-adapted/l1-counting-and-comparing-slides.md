# Counting and comparing

## Today we are learning
- count a group of objects
- record data in a tally chart
- write the tally as a total
- compare totals using more than and less than

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: data · count · tally chart · total · more than · less than. This is the first lesson of the unit — set the routine.

## Count the animals
![A cartoon elephant]({{res:l1-a-cartoon-elephant-teach-computing.png}})

- point to each object as you count
- 4 elephants, 6 leopards, 8 tigers, 3 zebras
- counting tells us how many — this is our **data**

> 🧑‍🏫 Show groups of objects and count together. Keep the pace slow. Ask "which has more?" / "which has fewer?".

## Why use a tally chart?
- if we just count, we forget the numbers
- a tally chart keeps a **record**
- one mark for each object

> 🧑‍🏫 Key idea: a tally chart organises data so we can count and compare it later without recounting.

## Marks in fives
- one mark for each object
- at **five**, draw a line across — like a **gate**
- groups of five are easy to count: 5, 10, 15…

> 🧑‍🏫 Model a few tallies on the board. Likely error: counting marks as you draw them → fix-words "make the marks first, count the total at the end."

## Read a tally chart
![Favourite pets tally chart]({{res:l1-favourite-pets-tally-chart-teach-computing.png}})

- find the **most** and the **fewest**
- use **more than** and **less than**

> 🧑‍🏫 You-do. The activity worksheet asks pupils to ORDER the steps, then read this chart. Then they tally objects in the room and write the totals.

Tick your four "I can…". Tell me one "more than" sentence.
