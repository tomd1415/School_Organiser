## What we are learning
Today I will say what data, a record and a field are, and start my own card database.

## Starter — what is data?
**Data** is facts, like words and numbers. A **database** is a tidy way to keep lots of data so we can find it again.

| Question | Choose |
|---|---|
| A database is… | (  ) a tidy store of data (  ) a kind of animal (  ) a paint program |

## 🟢 Support
A **record** is one card (all the facts about one animal). A **field** is one fact on the card.

| Question | Choose |
|---|---|
| One whole card about a bat is a… | (  ) record (  ) field (  ) chart |
| "Can fly" is one fact, so it is a… | (  ) record (  ) field (  ) chart |
