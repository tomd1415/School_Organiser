## AND or OR?
Drag each fact into the right box.

```sort
AND — fewer results (narrower): both rules must be true, Male AND first class, Female AND died
OR — more results (wider): either rule can be true, first class OR second class, boarded at Belfast OR Queenstown
```

## Build an OR search
This is an OR search being built. Put the steps in order.

![An OR search: Class = 1st OR Class = 2nd]({{res:l4-or-search.png}})

```order
Choose the field Class
Choose the value 1st
Add OR on the first row
On the next row choose Class again
Choose the value 2nd
```
