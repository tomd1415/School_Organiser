## Two ways to see the data
A computer database can show data as a **form** (one card at a time) or as a **table** (all rows together).

![A record shown in form view]({{res:l2-countries-form.png}})

Match each view to what it shows.

| View | What it shows |
|---|---|
| Form view | (  ) one record at a time (  ) all records in rows |
| Table view | (  ) one record at a time (  ) all records in rows |
