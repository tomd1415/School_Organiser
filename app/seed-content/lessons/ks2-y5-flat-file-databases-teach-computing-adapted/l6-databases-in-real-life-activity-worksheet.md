## Search the flight database
This is a flight-finder website. You type where you fly from, where you fly to, and the dates.

![A flight search website with From, To and date fields]({{res:l6-expedia-search.png}})

Put the steps in order to search for a flight.

```order
Type where you are flying from
Type where you are flying to
Choose the dates
Press search
Read the list of flights
```
