# Think before you link!

## Today we are learning
- explain the implications of linking to others' work
- create hyperlinks to other people's work
- evaluate the user experience of a website

> 🧑‍🏫 Read the three "I can…" aloud. Words on the board: hyperlink · external link · implication · evaluate · embed. This is the last lesson — recap the whole unit at the end.

## Starter — is it OK to link?
![Is the site secure?]({{res:l6-a-locked-and-unlocked-padlock-is-the-site-secure-teach-computing.png}})

- a link to someone else's site is an external link
- it is OK to link, but think first

> 🧑‍🏫 Ask "is it OK to link to content owned by others?" Link back to copyright. Support pupils choose what an external link is.

## Implications of external links
- the page might be deleted → link breaks
- the content might change → you can't control it
- the site might not be secure or trustworthy
- give credit · open links in a new tab

> 🧑‍🏫 Go through each implication calmly. Pupils tick what to think about (multi-select). Fix-words: "open in a NEW tab so the user can get BACK to your site." Brief online-safety chat: only link to sensible sites; report concerns.

## Add an external link
- search for a sensible, trustworthy site
- link to it and give credit

> 🧑‍🏫 Show the add-external-link video. Pupils add a link tied to their topic. Early finishers: embedding content (show the video). Avoid sites with adverts/suggested content.

## Evaluate a website
![A finished website to evaluate]({{res:l6-a-finished-website-to-evaluate-teach-computing.png}})

- evaluate = judge how good it is
- user experience = what it is like to use
- look for: spelling, pictures that fit, links that work

> 🧑‍🏫 Model the evaluation on the example site (spot the spelling error, the broken link). Agree the 1–5 scale (5 = great). Pupils rate it, then evaluate their own site. Show your work: screenshot the site + link.

## I can… — end of unit
Tick your three "I can…". Tell me one thing to think about before linking.

> 🧑‍🏫 Recap the whole unit: good websites → web designer → copyright → look on devices → navigation paths → linking safely. Celebrate the finished websites. Movement break.
