# What makes a good website?

## Today we are learning
- explore a website
- name the media used on websites
- websites are written in HTML

> 🧑‍🏫 Read the three "I can…" aloud, slowly. Words on the board: website · web page · browser · media · HTML. Same routine as every lesson.

## Starter — websites we use
![The San Diego Zoo Kids website]({{res:l1-zoo-website.png}})

- a website is a set of pages on the internet
- a web page is one page of a website
- a browser is the program we use to look at them

> 🧑‍🏫 Ask "which websites do you use?" Keep it age-appropriate. Likely mix-up: website vs web page. Fix-words: "a web page is ONE page; a website is the WHOLE set." Support pupils choose what a browser is.

## Quick quest — explore
- click the menu, buttons and search to find facts
- there is more than one way to find an answer

> 🧑‍🏫 Model one search live (menu → animals → tigers). Pupils find one fact and say how they found it. TA: prompt with "try the menu" — do not click for them.

## The media on a page
- media = text, pictures, video and sound
- good sites use media to help and to look interesting

> 🧑‍🏫 Pupils tick all the media they find (multi-select on the worksheet). Draw out that a page is more than just words.

## Websites are written in HTML
![The HTML code behind a web page]({{res:l1-html-code.png}})

- HTML = Hypertext Markup Language
- it tells the browser what the page should look like
- you can see the HTML behind ANY website

> 🧑‍🏫 Likely error: "the website is programmed." Fix-words: "HTML is not run — the browser READS it to draw the page." Show View page source. Challenge: why is it useful you can see it? Show your work: screenshot the code.

## I can…
Tick your three "I can…". Tell me one thing: what does HTML stand for?

> 🧑‍🏫 Listen for "Hypertext Markup Language" or "the code that makes a website". Movement break before next lesson.
