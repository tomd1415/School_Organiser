# Final project

## Today we are learning
- build a three-page website using HTML and CSS
- use a rubric to evaluate a website
- give and act on peer feedback

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on board: HTML · CSS · div · class · box model.

## Starter — write the HTML and CSS
On the starter worksheet, write the HTML for a list inside a `div`, and the CSS to make it red and monospace.

> 🧑‍🏫 Recall task — they only write what goes inside `<body>`. Swap and check against the answers together.

## The project
Build a three-page website on a theme you choose. Reuse your code from earlier lessons.

> 🧑‍🏫 Hand out the rubric now so they know what "good" looks like. Encourage reusing and adapting working code.

## What "good" looks like — the rubric
Your site is judged on HTML/CSS, pages, images, accessibility, navigation and design.

> 🧑‍🏫 Walk through each row. Self-rate on the worksheet — it focuses their next steps. Reassure: the site need not be finished today.

## Plenary — peer feedback
Swap with the person next to you. Give two things that work and one thing to improve.

> 🧑‍🏫 Model kind, specific feedback. Feedback is acted on next lesson, so keep it constructive.
