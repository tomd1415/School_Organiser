## What is CSS?
CSS (Cascading Style Sheets) sets the **look** of a page — colours, fonts and sizes. One stylesheet can style every page at once.

## 🟢 Support
A CSS rule is built from three parts: a **selector**, a **property** and a **value**.

```
h1 { color: red; }
```

| Question | Choose |
|---|---|
| In `h1 { color: red; }`, the part `h1` is the… | (  ) selector (it chooses what to style) (  ) value (  ) bracket |
| In `color: red;`, the part `red` is the… | (  ) value (  ) selector (  ) tag |
