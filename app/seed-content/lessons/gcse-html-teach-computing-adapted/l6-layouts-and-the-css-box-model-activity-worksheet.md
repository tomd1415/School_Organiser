## The CSS box model
Every element sits inside a box with four layers.

![The CSS box model: content, then padding, then border, then margin]({{res:l6-css-box-model-teach-computing.png}})

## Match each box-model layer to what it is
Drag each answer to the right layer.

| Box-model layer | What it is |
|---|---|
| content | (  ) the text or image itself (  ) space inside the border, around the content (  ) the line around the element (  ) space outside the border, between elements |
| padding | (  ) the text or image itself (  ) space inside the border, around the content (  ) the line around the element (  ) space outside the border, between elements |
| border | (  ) the text or image itself (  ) space inside the border, around the content (  ) the line around the element (  ) space outside the border, between elements |
| margin | (  ) the text or image itself (  ) space inside the border, around the content (  ) the line around the element (  ) space outside the border, between elements |
