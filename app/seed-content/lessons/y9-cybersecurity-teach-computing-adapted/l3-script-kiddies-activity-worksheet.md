## How a DDoS attack works
![A DDoS attack card]({{res:l3-attack-ddos-card.jpg}})

Drag these steps into the right order.

```order
The attacker controls a large group of computers (a botnet).
All those computers send requests to one server at the same time.
The server gets far more requests than it can handle.
The server slows down or stops responding.
Real users can no longer reach the website.
```
