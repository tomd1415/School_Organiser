## What we are learning

## Getting your message to the right place
To send a letter to the right house, you need the **address**: the name, the street, the town, and the postcode.

![A postbox]({{res:l1-postbox.png}})

Computers are the same. Every device on the internet has an address. It is called an **IP address**. IP means Internet Protocol. An IP address looks like this: `93.184.216.34`.

## A website name becomes an address (DNS)
IP addresses are hard to remember (93.184.216.34). Names are easier (bbc.co.uk). A **Domain Name Server (DNS)** is like an address book. You give it a name, and it finds the number (the IP address).

![An address book with A–Z tabs]({{res:l1-address-book.jpg}})

Put the steps in order. What happens when you type a website name?

```order
You type a website name (like bbc.co.uk)
The DNS looks up the name in its address book
The DNS finds the IP address (the number)
Your computer uses the IP address to open the website
```
