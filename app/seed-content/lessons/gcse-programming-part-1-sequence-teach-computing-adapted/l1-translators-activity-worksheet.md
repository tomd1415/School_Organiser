## What we are learning
Computers can only run **machine code** (0s and 1s). A **translator** turns the code people write into machine code.

![The first Fortran reference manual — an early high-level language]({{res:l1-fortran-reference-manual-teach-computing.png}})

## Order the journey
Programming languages changed over time. Drag the steps into the right order — oldest first.

```order
Programmers wrote programs directly in machine code (0s and 1s)
Assembly language was invented to make the code easier to read
An assembler translated the assembly language into machine code
Grace Hopper had the idea of writing programs in plain English
High-level languages like Fortran were created
A translator turns the high-level code into machine code so the computer can run it
```
