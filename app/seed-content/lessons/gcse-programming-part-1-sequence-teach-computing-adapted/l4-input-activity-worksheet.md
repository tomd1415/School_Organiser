## What we are learning
`input()` lets the user type an answer. Today I will collect data and learn the five **data types**.

## The data collection program
![The mini data collection program code]({{res:l4-data-collection-code-teach-computing.png}})

```python
print("What is your first initial?")
initial = input()
print("What is your age?")
age = int(input())
print(f"Well hello {initial}.")
print(f"You are {age} years old.")
```
