## Predict
Now look at the "Twinkle" program.

```python
from time import sleep
def twinkle():
    print("Twinkle, twinkle little star")
    sleep(3)
    print("How I wonder what you are")
    sleep(3)
def upabove():
    print("Up above the world so high")
    print("Like a diamond in the sky")
twinkle()
```
