## Starter — will this code run?
Here is a program written in **pseudocode**, not Python.

```
WHILE pass_mismatch
    OUTPUT "Enter your password:"
    password <- USERINPUT
    pass_mismatch <- stored_pass != password
ENDWHILE
OUTPUT "Access granted"
```
