## 🟢 Support
Put the FizzBuzz rules **in the order** the program checks them, from 1 to 15.

```order
Start the number at 1
Check if the number divides by both 3 and 5
If it divides by both, output FizzBuzz
Otherwise if it divides by 5, output Buzz
Otherwise if it divides by 3, output Fizz
Otherwise output the number
Add 1 to the number
Stop when the number passes 15
```
