## The interactive dessert selector
- a slide deck of either/or questions
- buttons link to the next slide
- e.g. "Do you prefer chocolate or fruit?"

> 🧑‍🏫 Still in the EXECUTE stage. Demonstrate one set of buttons.

## Hierarchy diagram
- shows how the slides link, level by level
- plan the links BEFORE you build

> 🧑‍🏫 Pupils plan their selector on the hierarchy diagram first.

## Build it
- set the **house style** on the master slide (saves time)
- add a logo, image and question to each slide
- add button shapes, then hyperlink each to the right slide

> 🧑‍🏫 Likely error: linking the button to the wrong slide. Fix-words: "read the slide number before you link." Then test every button.

## The posters

- keep the house style the same on both
