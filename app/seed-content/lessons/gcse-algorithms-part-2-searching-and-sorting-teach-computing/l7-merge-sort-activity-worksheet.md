## What we are learning

## Merging two ordered lists
To merge two ordered lists, compare the FRONT item of each list and move the smaller one into a new list. Keep going.

![Merging two ordered lists of cards: compare the front of each list, move the smaller into the merged list, repeat until done]({{res:l7-merging-two-ordered-lists-teach-computing.png}})

## Put the merge steps in order
Drag the steps for merging two ordered lists into order.

```order
Start with two lists that are already in order
Make a new empty list for the merged items
Compare the first item of each list
Move the smaller item into the new list
Repeat until one list is empty
Add the rest of the other list to the new list
```

## 🟡 Core
A merge sort has two stages: **split** the list down to single items, then **merge** pairs back together in order.

```order
Split the list in half again and again until each item is alone
Merge pairs of lists back together in order
Keep merging pairs until there is one sorted list
```
