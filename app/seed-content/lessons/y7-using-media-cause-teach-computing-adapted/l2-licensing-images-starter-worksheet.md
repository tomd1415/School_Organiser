## What we are learning
Today I will choose images that suit a document, and learn the rules for using other people's images.

## Starter — is this image a good choice?
A good image suits the **purpose** and the **audience** of the document.

| Question | Choose |
|---|---|
| Your document is about saving trees. Which image suits it best? | (  ) a forest of trees (  ) a birthday cake (  ) a football |
| The audience is young children. Which image suits them best? | (  ) a clear, friendly picture (  ) a page of small text (  ) a scary photo |
| Where should an image go on the page? | (  ) next to the text it matches (  ) over the top of the words (  ) off the edge of the page |
