# Features of a word processor

## Today we are learning
- choose the right software for a task
- name word-processor tools (features)
- format a document with the tools
- explain why we format documents

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson. Vocabulary: application software · word processor · formatting · font · icon.

## Starter — the right software for the job
Which software is best for: writing an essay? showing slides? sending a document?

> 🧑‍🏫 Use the starter worksheet. Single best answer each time. Word processor = writing; presentation = slides; email = sending. Likely error: "any software will do" → fix-words: "we pick the tool that fits the job."

## Features of a word processor  (I do)
Bold · font · font colour · alignment · bulleted list.

> 🧑‍🏫 I-do: open the software, hover over each icon, show what it does. Then pairs find the tools on the activity sheet. Support: word-bank of tool names. Challenge: find one extra tool.

## Why do we format?  (we do)
A document should be easy for the reader to understand.

![A poorly formatted "help save the earth" document]({{res:l1-poorly-formatted-document.png}})

> 🧑‍🏫 Show the messy document. Ask: what is wrong? (no headings, hard-to-read font, mixed alignment.) Link to the AUDIENCE — we format for the reader.

## Your turn — format the document  (you do)
Open "Help save the Earth" and use the tools to make it clear.

> 🧑‍🏫 Support sheet has fewer items to fix and less writing. Core: format the whole document. Challenge: investigate one new tool. TA: prompt, do not do it for them. They paste a link + screenshot on the activity worksheet.

## I can…
Tick your four "I can…". Show me your formatted document.
