# Collecting data

## Today we are learning
- recap formulas
- explain data and information
- explain primary and secondary sources
- help collect class data

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary: data · information · source · primary source · secondary source.

## Starter — formulas recap
Write the formula, not the answer. For example = A1 * C1.

> 🧑‍🏫 Pupils complete the starter worksheet. Recap: formulas start with =, use * for multiply and / for divide.

## Data vs information
Data = raw facts and figures. Information = data organised so it has meaning.

> 🧑‍🏫 Example: "27, 46, 12" is data; "the USA won the most golds" is information.

## Primary and secondary sources
Primary source = you collect the data yourself. Secondary source = someone else collected it.

> 🧑‍🏫 Examples: your class survey (primary); a Wikipedia table (secondary).

## Design a survey (we do)
As a class, write a few survey questions. No names or personal details.

> 🧑‍🏫 Keep questions optional. Save the spreadsheet of responses — we use it in the next lessons.

## Word match (plenary)
Match each word to its meaning on your worksheet.

> 🧑‍🏫 Circulate, or display the answers for self-assessment.
