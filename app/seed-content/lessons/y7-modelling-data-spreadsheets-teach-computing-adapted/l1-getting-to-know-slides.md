# Getting to know a spreadsheet

## Today we are learning
- name rows, columns and cells
- write a cell reference like B3
- select a range like A1:F5
- change fill colour and borders

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson. Vocabulary: data · cell · cell reference · row · column · range · select.

## Starter — Olympic medals
![Olympic rings]({{res:l1-olympic-rings.png}})

The medal data is written as a list. Is it easy or awkward to read?

> 🧑‍🏫 Pupils answer on the starter worksheet (tick). Draw out: a list is awkward; a spreadsheet grid is easier. USA won the most golds (46).

## Rows and columns
![Seats in a stadium]({{res:l1-rows-columns-stadium-seats.png}})

A spreadsheet is a grid. Columns go across the top (A, B, C). Rows go down the side (1, 2, 3).

> 🧑‍🏫 Ask where else they have seen rows and columns (seats, chessboard, a table). Columns = letters, rows = numbers.

## Cell references (I do → we do)
A cell reference is the column letter then the row number. B3 = column B, row 3.

> 🧑‍🏫 Model finding B3 on screen. Likely error: writes 3B. Fix-words: "letter first, then number."

## Bingo — find the cell (you do)
![Cell reference bingo card]({{res:l1-cell-reference-bingo.png}})

I call out a cell like B3. You find it on your bingo sheet.

> 🧑‍🏫 Support: point to the column, then slide down to the row. Challenge: a pupil calls a cell for the class.

## Pixel art — select a range
A range like A1:F5 is a block of cells. Select it, then change the fill colour and add borders.

> 🧑‍🏫 Model select-range → fill colour → border. Pupils make a flag, then paste a screenshot on the activity worksheet.

## I can…
Tick your four "I can…". Show me your flag.

> 🧑‍🏫 Note who selected a range and changed the fill colour unaided.
