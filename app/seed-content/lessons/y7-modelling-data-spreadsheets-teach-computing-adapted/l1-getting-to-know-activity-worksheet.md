## Cell references (we do)
A spreadsheet is a grid of rows and columns, a bit like seats in a stadium.

![Seats in a stadium]({{res:l1-rows-columns-stadium-seats.png}})

A cell reference is the **column letter** then the **row number**. B3 means column B, row 3.

| Question | Your answer |
|---|---|
| What is the reference for the cell in column D, row 5? | (  ) D5 (  ) 5D (  ) DD |
