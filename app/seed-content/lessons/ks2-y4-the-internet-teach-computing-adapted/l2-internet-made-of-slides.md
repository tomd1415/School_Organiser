# What is the internet made of?

## Today we are learning
- name networked devices and how they connect
- say the internet gives us many services
- recognise the World Wide Web is part of the internet
- tell the difference between a website and a web page

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: switch · server · router · internet · World Wide Web · website · web page. Same routine as every lesson.

## The devices that make the internet
![A network switch]({{res:l2-switch.png}})
![A server]({{res:l2-server.png}})
![A router]({{res:l2-router.png}})

- **switch** — passes a message inside a network
- **server** — stores files for everyone
- **router** — joins one network to another

> 🧑‍🏫 Recap from last lesson and Year 3. Memory hooks: "Switch Sends, Server Stores, Router Reaches another network." The starter is the device→job match.

## The internet carries services
- the internet is the wires, routers and devices
- it carries **services**: the World Wide Web, files, email

> 🧑‍🏫 Analogy: the internet is the roads; the services are the different journeys (post, shopping, visiting). Keep it concrete.

## The World Wide Web is PART of the internet
- the **World Wide Web** = websites and web pages
- it is only ONE part of the internet
- the internet and the World Wide Web are NOT the same

> 🧑‍🏫 Key misconception to fix: "the internet = the web". Fix-words "the web is one part of the internet, not all of it." You-do: the `sort` task (internet vs World Wide Web).

## Website and web page
![A website]({{res:l2-website.png}})
![A web page]({{res:l2-web-page.png}})

- a **website** = a collection of pages under one name
- a **web page** = a single page from that site

> 🧑‍🏫 Show a home page (website) then click to one page (web page). Likely error: using the two words to mean the same — fix-words "a website holds many web pages."

## Explore a website
- open the website your teacher chooses
- find the home page and one other page
- screenshot it for show-your-work

> 🧑‍🏫 You-do. Calm, structured browsing of ONE chosen site. TA: prompt, do not click for them. Movement break is routine.

Tick your four "I can…". Tell me one service the internet gives us.
