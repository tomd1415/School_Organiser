# What is a website?

## Today we are learning
- name the parts of a website
- recognise that I can add content to the web
- use a web tool to create content online

> 🧑‍🏫 Read the "I can…" aloud. Vocabulary on the board: website · web page · content · logo · links · media. Same routine as every lesson.

## The parts of a website
![A website with a logo, search box, menu and pictures]({{res:l4-website.png}})

- **logo** — shows whose site it is
- **search box** — find things on the site
- **menu links** — click to go to other pages
- **pictures**, **text** and **video** — the content

> 🧑‍🏫 I-do: point to each part on the board. Starter is the `label` task — drag Logo / Search box / Menu links / Picture onto the right spot.

## People make the content
- websites are made by **people**
- on some sites, anyone can **add content** (a blog, a comment, a project)

> 🧑‍🏫 Good: easy to share knowledge. Tricky: not everything added is true or kind. Keep this cohort-level — no personal posting stories. Sets up lessons 5 and 6.

## Make your own content
![Chrome Music Lab — a web tool]({{res:l4-music-lab.png}})

- a **web tool** lets you create content online
- today: use Chrome Music Lab to make a sound or pattern

> 🧑‍🏫 Model the Kandinsky tool briefly, then you-do. Note for SEND: Music Lab can play sound — keep volume low and let pupils use headphones if they prefer. No flashing visuals.

## Show what you made
- you used a web tool to create your own content
- screenshot it for show-your-work

> 🧑‍🏫 Low floor: one shape that makes a sound. High ceiling: a pattern or tune. TA: prompt, do not make it for them. Movement break is routine.

## I can…
Tick your "I can…". Tell me one part of a website.

> 🧑‍🏫 Check pupils can name a logo, links or a picture, and say "people make websites." Next lesson: who owns the content on the web.
