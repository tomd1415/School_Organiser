# Clean it up

## Today we are learning
- explain why data needs cleansing
- find and fix errors in a data set
- make a visualisation from a data set

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on board: data cleansing · error · analysis · visualisation. We are in the Data + Analysis steps of PPDAC.

## Starter — spot the problem
![A zoo spreadsheet with a row highlighted]({{res:l5-dirty-data-spreadsheet.png}})

This data was scanned in from paper. Which values look wrong?

> 🧑‍🏫 The meerkat at 3 m is the obvious error (should be ~0.3 m). Think · write · pair · share. Change a cell's colour to flag a problem.

## Garbage in, garbage out
![A bar chart of average animal heights]({{res:l5-heights-chart.png}})

If we chart dirty data, the chart lies.

> 🧑‍🏫 Show that one wrong value makes the meerkat bar wrong. Big idea: clean BEFORE you analyse.

## Clean your own data  (you do)
Open your litter data. Check it. Fix or remove anything wrong.

> 🧑‍🏫 Support: point to the cells to check. Challenge: decide which odd values are real vs errors. TA: prompt only. Movement break is routine here.

## Make a visualisation  (you do)
Upload your cleaned data to the tool and build a chart that answers one question.

> 🧑‍🏫 Bring pupils needing it to the front for a second demo of making a chart, while others continue.
