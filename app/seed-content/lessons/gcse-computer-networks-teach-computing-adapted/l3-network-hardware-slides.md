PAN = around a person · LAN = one site · WAN = a large area.

## The hardware  (I do)
- **Router** — joins one network to another (e.g. to the internet).
- **Switch** — sends data only to the right wired device.
- **Hub** — copies data to every device (broadcasts).
- **NIC** — lets a computer join a network; holds the MAC address.
- **WAP** — gives devices a wireless way to join.
- **Bridge** — joins one LAN to another.

## What is a MAC address?  (we do)
Every **NIC** has a unique **MAC address** built in. It is used to find a device on the network — like a permanent name tag.

> 🧑‍🏫 Plain line: MAC address = a device's unique built-in name tag. Not a web address, not a password. Likely error: confusing MAC with IP — fix-words "MAC is built in and stays."

## Choose the hardware
Different jobs need different devices: a switch for tidy wired traffic, a router to reach the internet, a WAP for WiFi.
