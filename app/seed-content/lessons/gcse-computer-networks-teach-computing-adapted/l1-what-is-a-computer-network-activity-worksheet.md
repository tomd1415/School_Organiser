## What we are learning
Today I will define a computer network, give its advantages and disadvantages, and describe how a computer behaves in a peer-to-peer network.

![A network of people connected to each other]({{res:l1-a-network-of-connected-people.png}})

A **computer network** is two or more computers connected together, sharing resources.

## Is it a network?
A network is things that are **connected and sharing**. Something on its own is not a network. Drag each one into the right box.

```sort
A network (connected and sharing): computers in a school sharing one printer, the mobile phone network, friends who all message each other
Not a network (on its own): a single calculator, one laptop with no connections, a games console that is switched off and unplugged
```

## Drawing a network — graphs
We draw networks as a **graph**. The dots are called **nodes** and the lines that join them are called **edges**.

| Question | Choose one |
|---|---|
| In a graph, what is a node? | (  ) a dot that stands for a connected thing (  ) the line between dots (  ) the name of the network |
| In a graph, what is an edge? | (  ) the line that joins two nodes (  ) a dot (  ) a printer |
