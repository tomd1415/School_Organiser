## A protocol is a set of rules
A **protocol** is a set of rules for communication. Both the sender and receiver follow the same rules so the message gets through.

## 🔴 Challenge — design a protocol
Your group must send a message that is too long to fit in one go, so it is split into small chunks. The chunks might arrive out of order. Put the rules of a good sending protocol in order.

```order
Split the message into small numbered chunks.
Send each chunk across the network.
The receiver puts the chunks back in number order.
The receiver checks none are missing and asks again for any that are.
The receiver confirms the whole message has arrived.
```
