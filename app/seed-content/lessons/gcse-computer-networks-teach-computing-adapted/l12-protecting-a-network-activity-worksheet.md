## Two kinds of attack
Attacks fall into two groups: **social engineering** (tricking people) and **malicious software** (harmful code). Drag each attack into the right group.

```sort
Social engineering (tricks a person): phishing, blagging, shouldering
Malicious software (harmful code): a virus, spyware, a DDoS attack
```

## How a pharming attack works
In **pharming**, malware quietly changes where a web address points, sending the victim to a fake copy of a site. Put the steps in order.

```order
Malware changes the DNS record for a real website.
The user types in the correct web address as normal.
The DNS sends them to the attacker's fake website instead.
The user enters their details into the fake site, and the attacker steals them.
```
