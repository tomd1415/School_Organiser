# What is a computer network?

## Today we are learning
- define a computer network
- give the advantages and disadvantages of a network
- describe a computer's role in a peer-to-peer network

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary: network · graph · node · edge · computer network · client · server · peer-to-peer. Hand out both worksheets.

## Starter — networks are everywhere
A **network** is just things connected together: friends who message, train stations on a map, the phone network.

> 🧑‍🏫 Take two or three everyday examples from the class. Keep it concrete and calm. No writing — pupils choose on the starter sheet.

## What is a computer network?  (I do)
A **computer network** is two or more computers connected together, **sharing resources** (files, printers, an internet connection).

![A network of people connected to each other]({{res:l1-a-network-of-connected-people.png}})

> 🧑‍🏫 Plain line: connected + sharing = a network. One computer on its own is not a network. Then the "is it a network?" card-sort.

## Drawing a network — graphs  (we do)
We draw a network as a **graph**. The dots are **nodes** (the connected things). The lines are **edges** (the connections).

> 🧑‍🏫 Draw three dots and join them on the board. Point: node = a thing, edge = a connection. They will see graphs all unit.

## Good and bad points
- Good: share files, share printers, central backups.
- Bad: costs money to set up, and if it breaks, lots of people are affected.

> 🧑‍🏫 Pupils tick the advantages on the activity sheet, then write one advantage + one disadvantage (Core). Likely error: thinking a network is always free — fix-words "connecting costs money."

## Peer-to-peer  (you do)
▶ **Your turn — open your activity worksheet now.** Put the peer-to-peer steps in order, then answer the questions for your level.

In a **peer-to-peer** network every computer is **equal**. There is no main server — each computer can **ask for** files and **send** files.

> 🧑‍🏫 Use the card-passing idea: one computer asks another for a file, that computer sends it back. Challenge pupils order the four steps. Next lesson: the client–server model.

## I can…
Tick your three "I can…". Tell me one thing a computer network lets you share.
