# Hosting services

## Today we are learning
- describe how servers host services across the internet
- describe what a web server and a client do
- explain what cloud computing is
- list advantages and disadvantages of the cloud

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary: web server · hosting · client · data centre · cloud computing · software as a service (SaaS). Hand out both worksheets.

## Starter — an internet minute
Every minute, millions of searches, messages and videos travel the internet. Where is all of it stored?

> 🧑‍🏫 Take a couple of guesses for the search number (about 6 million). Lead to: it is all stored on servers. Recap last lesson: browser shows pages, DNS finds the IP address.

## Where do web pages live?  (I do)
A **web server** is a powerful computer that stores web pages and sends them out when a device asks for one. **Hosting** means a server is connected to a client and giving it a service.

> 🧑‍🏫 Plain line: the server keeps the page, your device (the client) asks for it. Stress server = sends, client = asks.

## Data centres
Big companies keep thousands of servers together in a **data centre** — a building built to power and cool them.

> 🧑‍🏫 Optional teacher-played hook: the Google data-centre tour video (ncce.io/TheTour) — motion/sound, your choice. Point out the cooling and the rows of servers in the photo.

## The cloud  (we do)
**Cloud computing** means software and storage you use **through the internet**, not on your own device. Using software this way is called **software as a service (SaaS)**.

> 🧑‍🏫 Examples pupils know: Google Docs, iCloud photos, Netflix. Class sorts "in the cloud" vs "only on this device" on the activity sheet.

## Good and bad points  (you do)
▶ **Your turn — open your activity worksheet now.** Tick the advantages of cloud hosting, then give a disadvantage.

- Good: reach your files from any device, automatic backups, no big install.
- Bad: you need an internet connection, and someone else holds your data.

> 🧑‍🏫 Pairs do the "tick all the advantages" task, then the Core disadvantage. Likely error: thinking the cloud works offline — fix-words "no internet, no cloud."

## Is the cloud green?
Data centres use a lot of electricity — but sharing servers can be more efficient than everyone owning their own.

> 🧑‍🏫 Challenge discussion: good or bad for the environment? Accept either view with a reason. Keep it calm and balanced.
