## Web server or client?
A **web server** stores web pages and sends them out when asked. A **client** is the device that asks for them and shows them to you. Drag each one into the right box.

```sort
The web server (stores and sends): keeps the web pages, waits for requests, sends the page back
The client (asks and shows): your laptop or phone, types in the web address, displays the page to you
```

## In the cloud or only on this device?
"The cloud" means software and storage you use **through the internet**, kept on servers in a data centre. Drag each example into the right box.

```sort
In the cloud (used through the internet): editing a document in Google Docs, photos backed up to iCloud, streaming a film from Netflix
Only on this device (no internet needed): a game saved only on this PC, a file on a USB stick, a photo stored only on your phone
```
