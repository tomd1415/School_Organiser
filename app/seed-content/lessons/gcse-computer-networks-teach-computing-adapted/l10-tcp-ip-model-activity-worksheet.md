## What we are learning
Today I will name the four layers of the TCP/IP model in order, say what each layer does, and match protocols to their layer.

## The four layers, top to bottom
Data starts at the top with the application and works its way down to be sent. Drag the layers into the right order, from the top layer to the bottom layer.

```order
Application layer — the app (like a web browser) prepares the data.
Transport layer — splits the data into segments and adds port numbers.
Internet layer — adds the sender and receiver IP addresses (makes packets).
Link layer — sends the bits across the wire or air to the next device.
```
