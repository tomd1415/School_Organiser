## The internet vs the Web  (I do)
- **The internet** = a global **network of networks** (the cables, routers and connections).
- **The World Wide Web** = the **web pages** you view on it.
They are often confused — the Web rides *on* the internet.

![Networks linked all around the world]({{res:l7-internet-world.png}})

> 🧑‍🏫 Plain line: internet = the roads, the Web = the shops you visit. Then the internet/Web card-sort. Brief history: ARPA built a distributed network so it could survive damage.

## IP addresses  (we do)
Every device has a unique **IP address** — a number that locates it on the network, so data reaches the right place.

> 🧑‍🏫 Plain line: IP address = a device's address, like a house number. Likely error: confusing IP with MAC — fix-words "MAC is built in; IP is given on the network."

## DNS — the internet's phone book
You type a **URL** (www.example.com). The **DNS** looks it up and returns the **IP address**, so the browser knows where to go.

> 🧑‍🏫 Plain line: DNS turns the name you type into the number the network needs. Optional: the Packet Tracer "investigating DNS" video — motion/sound, your choice.
