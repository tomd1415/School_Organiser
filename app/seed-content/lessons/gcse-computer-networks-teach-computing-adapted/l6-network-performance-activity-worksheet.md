## What we are learning
Today I will describe the factors that affect network performance, calculate how long a file takes to send, and find the lowest-cost route.

Four things change how well a network performs: **bandwidth**, **latency**, **range** and the **number of devices**.

![A crowd at a concert all using their phones at once]({{res:l6-crowd-bandwidth.png}})

> When lots of devices share one connection — like a crowd at a concert — each one gets less bandwidth, so everything feels slow.

## 🟡 Core — file transfer time
The formula is: **time = file size (in bits) ÷ speed (in bits per second)**.
Worked example: 50 bits ÷ 10 bits per second = **5 seconds**.

## 🔴 Challenge — the lowest-cost route
