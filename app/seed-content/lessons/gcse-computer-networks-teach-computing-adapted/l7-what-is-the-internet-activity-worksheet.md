## What we are learning
Today I will describe the internet as a network of networks, describe an IP address and DNS, and describe what a web browser does.

The **internet** is a global **network of networks**. The **World Wide Web** is the web pages you view on it — they are not the same thing.

## 🔴 Challenge
Put the steps of visiting a website in the right order.

```order
You type a URL like www.example.com into the web browser
The browser asks the DNS for that website's IP address
The DNS finds the IP address and sends it back
The browser uses the IP address to request the page from the web server
The web server sends the HTML back to your browser
The browser builds the page on your screen
```
