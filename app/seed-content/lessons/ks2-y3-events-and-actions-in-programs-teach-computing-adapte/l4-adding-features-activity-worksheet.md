## What we are learning
We add a feature by joining a Pen block to an **event** (a key press). The key press turns the feature on.

## Look at the code
Pressing the u key lifts the pen. Pressing the d key puts it down.

![Two stacks: when u key pressed pen up; when d key pressed pen down]({{res:l4-pen-up-and-pen-down-on-keys-teach-computing.png}})
