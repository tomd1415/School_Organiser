# Adding features

## Today we are learning
- name extra Pen blocks
- choose a key to turn a feature on
- join the event and action to make it work

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: pen up · change pen colour · change pen size · event · action. Recap: last lesson we set up a drawing program. Same routine today.

## The Pen blocks
![Pen blocks: set pen colour to red, change pen size by 1, change pen size by -1]({{res:l4-pen-feature-blocks-teach-computing.png}})

- set pen colour to… changes the colour
- change pen size by 1 makes the line thicker
- change pen size by -1 makes the line thinner

> 🧑‍🏫 I-do. Name each block and what it does. This matches the match-the-block task on the starter worksheet. No flashing — point calmly at each block.

## Pen up and pen down
![Two stacks: when u key pressed pen up; when d key pressed pen down]({{res:l4-pen-up-and-pen-down-on-keys-teach-computing.png}})

- pen up stops the sprite drawing
- pen down starts it drawing again

> 🧑‍🏫 We-do. Press u (pen up) and d (pen down) together and watch the line stop and start. Likely error: mixing up pen up with erase all → fix-words "pen up stops drawing; erase all clears the screen."

## Add a feature to your project
- pick a Pen block
- choose a key to turn it on
- join the event to the action

> 🧑‍🏫 You-do. Each pupil adds one feature with a key. Support: add pen up on the u key. Core: add a colour or size change. Challenge: add two features and explain your key choices. TA: prompt, do not do it for them.

## Show me
Tick your three "I can…". Show me your new feature working.

> 🧑‍🏫 Plenary. Each pupil shows one feature and names the key that turns it on.
