## What we are learning
A program should **set up** the same way every time it runs. We use **when green flag clicked** to put the sprite in place, set the pen colour, and clear old lines.

## Look at the setup code
This setup runs every time the green flag is clicked.

![Setup blocks: when green flag clicked, go to x 0 y 0, set pen colour to blue, erase all]({{res:l3-program-setup-blocks-teach-computing.png}})

## Put the setup in order
Drag the setup steps into a sensible order.

```order
when green flag clicked
go to x: 0 y: 0
set pen colour to blue
erase all
```
