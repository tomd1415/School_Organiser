## How a DDoS attack works
Drag the steps into the right order, from first to last.

```order
The attacker secretly infects many computers with malware, making a botnet of 'zombies'.
The attacker tells all the zombie computers to send requests at the same time.
The target server is flooded with far more requests than it can cope with.
The server slows down or stops, so real users cannot get through.
```
