# The cost of cybercrime and hacker motivation

## Today we are learning
- describe how cybercrime affects people and businesses
- tell cybersecurity and network security apart
- say why a network can be attacked
- name types of hacker and what motivates them

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson. Vocabulary today: cybersecurity · network security · hacker · ethical hacker · hacktivist.

## Starter — how big is the problem?
![UK cyber security breaches survey]({{res:l1-uk-cyber-security-breaches-survey.png}})

> 🧑‍🏫 Let them read the numbers. Point: this is real and it costs a lot of money. Pair-talk: "who could this hurt?" Tick the answers on the starter worksheet.

## Two words that sound the same
- Cybersecurity = keeping ALL systems and data safe from attack.
- Network security = keeping the NETWORK and its data safe by managing who gets in.

> 🧑‍🏫 Likely mix-up: the two terms. Fix-words: "cyber = everything, network = the network." They do the matching task on the activity sheet.

## Why can a network be attacked?  (I do)
A network lets many computers share data. Every shared link is also a door an attacker could try.

> 🧑‍🏫 Worked example: one weak password on one computer can let an attacker onto the whole network.

## Who attacks, and why?  (we do)
- Malicious hacker — for money or to cause harm.
- Hacktivist — to send a message or protest.
- Ethical hacker — paid to find weak spots and help fix them.

> 🧑‍🏫 Read the three short stories together, then pairs sort the rest on the activity sheet. Stress: an ethical hacker has permission.
