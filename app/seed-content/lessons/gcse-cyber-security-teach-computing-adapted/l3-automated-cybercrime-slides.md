# Automated cybercrime

## Today we are learning
- name types of malware
- sort malware from network attacks
- put the steps of a network attack in order
- explain a weakness that let a real attack happen

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary: malware · virus · worm · trojan · spyware · ransomware · DoS · DDoS · zombie · botnet.

## Starter — a risky download
A free game from an unknown site asks you to turn off your antivirus first. What should you do?

> 🧑‍🏫 Draw out the safe choice: do not download it. This is how malware tricks its way in.

## What is malware?  (I do)
Malware is short for malicious software. It is code written to harm computers or steal data.

> 🧑‍🏫 List the family on the board: virus, worm, trojan, spyware, ransomware. They tick the malware ones on the activity sheet.

## Types of malware  (we do)
- Virus — hides in a program and spreads when you run it.
- Worm — copies itself and spreads on its own.
- Trojan — pretends to be safe to trick you.
- Spyware — hides and records what you do.

> 🧑‍🏫 Keep each one short. Pairs do the matching task. Common mix-up: virus vs worm — fix-words "a worm needs no help to spread."

## Network attacks — DoS and DDoS  (we do)
A DDoS attack floods a server with so many requests that it cannot answer real users.

> 🧑‍🏫 Optional active demo: the paper-ball / maths-sum DoS game from the teacher handout. Then pairs order the four steps on the sheet.
