## What is encryption?  (I do)
Encryption scrambles a message so that only someone with the key can read it.

> 🧑‍🏫 Plain example: a secret note only your friend can unlock. Now show the Caesar cipher as the simplest key.

## The Caesar cipher  (we do)
![A Caesar cipher wheel]({{res:l4-caesar-cipher-wheel.png}})

Pick a shift number (the key). Move each letter that many places along the alphabet.

> 🧑‍🏫 Work HELLO with +1 → IFMMP together. Then pairs fill the gaps and order the steps on the activity sheet. Likely error: forgetting to wrap Z back to A.

## Two keys: asymmetric encryption
Asymmetric encryption uses a public key to lock and a private key to unlock. The padlock and https show it is in use.

> 🧑‍🏫 Keep it simple: one key locks, a different key unlocks. No need for the maths.

## More software defences  (you do)
- Keep software updated — fixes known weak spots.
- Input sanitisation — cleans what users type so harmful commands fail.
- Code review — check the program for mistakes.

> 🧑‍🏫 Link back: WannaCry got in through software that was not updated. Pairs do the sort and matching tasks.
