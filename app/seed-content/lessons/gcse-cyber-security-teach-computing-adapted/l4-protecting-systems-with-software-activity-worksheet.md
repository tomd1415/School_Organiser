## How to encrypt with a Caesar cipher
Drag the steps into the right order, from first to last.

```order
Choose a shift number. This is the key, for example +3.
Find the first letter of your message on the wheel.
Move that many places along the alphabet to the new letter.
Write down the new letter, then do the same for every other letter.
```

## Sort the software defences
Drag each example into the box for the method it shows.

```sort
Encryption (scramble data so only the right person can read it): customer data is disguised and needs a key to read, the padlock and https in a web address
Keeping software up to date (fix known weak spots): the IT team always installs the latest updates, applying the patch that stops a known attack
Input sanitisation (clean what users type in) and code review (check the code): cleaning a website form so an SQL injection cannot work, regularly checking existing programs for mistakes
```
