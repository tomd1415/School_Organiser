## Plan an attack on 'Ranter'
You are a penetration tester checking a social media company called Ranter. Sort each test idea into the right type of attack.

```sort
Physical attack: follow a worker through a locked door, plug in a USB stick left in the car park
Software attack: try common passwords on the login page, look for a server that has not been updated
Social engineering attack: phone a worker pretending to be from IT, send a fake email asking for a password
```

## The steps of a penetration test
Drag the steps into the right order, from first to last.

```order
Plan the test and gather information about the target.
Scan the system to find weak spots.
Try to break in using the weak spots you found.
Write a report telling the company how to fix the weak spots.
```
