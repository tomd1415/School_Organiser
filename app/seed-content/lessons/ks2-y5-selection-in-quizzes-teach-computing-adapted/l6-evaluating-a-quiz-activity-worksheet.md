## What we are learning
Today I will add setup so my quiz starts the same every time, and improve it.

## Why setup?
Setup blocks run BEFORE the quiz starts. They put the sprite back to the same place, size, direction and colour every time — like setting up a board game before you play.

![Setup blocks: go to x 0 y 0, set size to 100, point in direction 90, set colour effect to 0]({{res:l6-setup-blocks.png}})
