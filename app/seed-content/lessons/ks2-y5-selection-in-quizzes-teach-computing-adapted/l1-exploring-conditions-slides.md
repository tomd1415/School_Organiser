# Exploring conditions

## Today we are learning
- say what a condition is
- find the condition in a Scratch program

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: selection · condition · true · false. Same calm routine as every lesson.

## Starter — true or false?
A condition is something the computer checks. It is TRUE or FALSE.

> 🧑‍🏫 Use the true/false sort on the starter worksheet. Keep it light — statements that are clearly true or false for the class.

## What is a condition?
- a CONDITION is something the computer checks
- it is either TRUE or FALSE
- example: "key r pressed?"

> 🧑‍🏫 Literal language: a condition is a yes/no check. Link back to selection from Programming A.
