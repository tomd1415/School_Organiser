# Selecting outcomes

## Today we are learning
- use a forever loop so the condition is checked again and again
- find the condition and the two outcomes in if… then… else…
- make a program with two outcomes

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: condition · true · false · outcome · if… then… else…. Same routine.

## Starter — keep checking
- without a loop, the condition is checked ONCE
- with a forever loop, it is checked again and again

> 🧑‍🏫 Use the two starter images. Likely error: pupils think a key press always works. Fix-words: "no loop = checked once."

## Two outcomes: if… then… else…
- THEN = the TRUE outcome
- ELSE = the FALSE outcome
- one of the two always happens

> 🧑‍🏫 I-do: a conditional statement links one condition to two outcomes. Act it out: "if teacher touches nose → thumbs up, else → thumbs down."
