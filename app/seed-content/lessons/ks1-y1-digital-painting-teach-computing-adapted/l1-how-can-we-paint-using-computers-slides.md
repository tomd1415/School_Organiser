# How can we paint using computers?

## Today we are learning
- name some paint tools
- use a tool to make marks on the screen
- say what a tool does
- draw a picture using the paint tools

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: paint program · tool · paintbrush · pencil · erase · fill · undo. Same routine as every lesson.

## A paint program
![A paint pot and a paintbrush]({{res:l1-paint-tools-pot-and-brush-teach-computing.png}})

- a paint program makes pictures on a computer
- the white area is the page
- the buttons are the tools — each does a different job

> 🧑‍🏫 Open the paint program (paintz.app or Microsoft Paint). Point to the page and the tools. Ask who has used a paint app before.

## The paint tools
![A pencil]({{res:l1-pencil-tool-teach-computing.png}}) ![A spray can]({{res:l1-spray-can-tool-teach-computing.png}}) ![An eraser]({{res:l1-eraser-tool-teach-computing.png}})

- paintbrush and pencil — make marks
- fill — colours a shape
- eraser — rubs out
- undo — goes back a step

> 🧑‍🏫 Model the pencil first: "What will this do?" Make a mark. Name each tool slowly. The starter worksheet asks pupils to name these tools.

## I do — draw a face
- draw a circle for the face
- fill it with colour
- add eyes and hair

> 🧑‍🏫 Model drawing a face. Show what happens if the circle is not joined up — the fill leaks. Likely error: "the fill went everywhere" → fix-words "close the shape, then undo and try again."

## Your turn — be a tool detective
- try each tool
- watch what it does
- then draw your own picture

> 🧑‍🏫 You-do. Support: offer 3 tools only. Challenge: offer all the tools. TA: prompt, do not do it for them. They screenshot their picture onto the activity worksheet.

## I can…
Tick your four "I can…". Show me one tool you used.

> 🧑‍🏫 Note who can name a paintbrush, eraser, fill and undo. Movement break before the next task.
