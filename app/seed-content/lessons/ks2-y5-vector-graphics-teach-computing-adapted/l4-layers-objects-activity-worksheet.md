## Fix the layers
These emoji faces are layered in the wrong order. You fix them by changing which object is in front.

![Emoji made by layering objects]({{res:l4-emoji-layering.png}})

## Send a shape to the back
Put the steps to move a shape to the back in order.

```order
Click the shape to select it
Click Arrange in the top menu
Choose Order
Choose Send to back
```

## 🟢 Support
Sort each one to where it is in the picture.

```sort
At the front: the last object you drew
At the back: the first object you drew
```
