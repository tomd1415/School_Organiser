# The drawing tools

## Today we are learning
- vector drawings are made from shapes and lines
- name the main drawing tools
- use the shape and line tools to make a picture
- say how a vector drawing is different from paper

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: vector · shape · line · object · toolbar · tool. Drawing software (Google Drawings or PowerPoint) open on each screen.

## What is a vector drawing?
![A logo made from shapes]({{res:l1-logo-globe.png}})

- a picture made on a **computer**
- built from **shapes** and **lines**
- logos, signs and app icons are vector drawings

> 🧑‍🏫 Starter slide. Point out the shapes inside the logo. Ask "what do all logos have in common?" — shapes and lines.

## Made from shapes
![Many shapes to choose from]({{res:l1-shapes-menu.png}})

- click the **shape** tool to open the shapes
- each shape you add is called an **object**
- put shapes together to make a picture

> 🧑‍🏫 I-do. Model opening the shape menu and drawing a rectangle. Say "this shape is one object."

## The toolbar
- **select** — pick an object (arrow)
- **line** — draw a straight line
- **shape** — make a new shape
- **zoom** — make the view bigger to add detail

> 🧑‍🏫 We-do. Point to each tool on the real software. The worksheet asks pupils to label these four tools.

## Which shape is selected?
![A selected shape with handles]({{res:l1-rect-circle-selected.png}})

- a **selected** object has a blue box
- and small squares (handles) round it
- now you can move it or change its colour

> 🧑‍🏫 Likely error: clicking empty space deselects the shape → fix-words "click the shape once to select it; the blue handles show it is selected."

## Your turn
- use the **shape** and **line** tools
- make a simple picture (a house, a robot)
- save it — we use it next lesson

> 🧑‍🏫 You-do. Support: 2–3 shapes. Core: a house from shapes. Challenge: add a line detail and change a colour. Nobody's screen looks less done.
