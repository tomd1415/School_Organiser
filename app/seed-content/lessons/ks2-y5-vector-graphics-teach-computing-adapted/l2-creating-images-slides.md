# Creating images

## Today we are learning
- find the shapes used to make a vector drawing
- each shape is an object
- move, resize and rotate an object
- duplicate (copy) an object to save time

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: object · move · resize · rotate · duplicate/copy. Drawing software open.

## Last lesson
- vector drawings are made from shapes and lines
- each shape is an **object**
- we found the drawing tools

> 🧑‍🏫 Quick recap. Ask one pupil to name a drawing tool.

## Copy to make more
![Many circles made by copying]({{res:l2-nine-circles.png}})

- you do NOT have to draw every shape
- **copy** one shape, then **paste** it
- Ctrl + C to copy, Ctrl + V to paste

> 🧑‍🏫 I-do. Model copy and paste. Duplicate is the same idea — one copy of an object.

## Resize with handles
![A selected object with handles]({{res:l2-resize-handles.png}})

- click an object to get the **handles**
- drag a **corner** handle to resize
- drag the round handle at the top to **rotate**

> 🧑‍🏫 Likely error: dragging a side handle stretches the shape out of shape → fix-words "use a corner handle to keep it neat."

## Move, resize, rotate
- **move** — drag the object to a new place
- **resize** — make it bigger or smaller
- **rotate** — turn it round
- **duplicate** — make a copy

> 🧑‍🏫 We-do. Demonstrate each on one circle. The worksheet sorts each tool to what it does.

## Your turn
- make a simple picture from circles and rectangles
- **duplicate** a shape instead of drawing it twice
- move, resize and rotate to get it right

> 🧑‍🏫 You-do. Support: copy one shape. Core: build a small picture (an owl, a snowman). Challenge: rotate a shape to add a detail.
