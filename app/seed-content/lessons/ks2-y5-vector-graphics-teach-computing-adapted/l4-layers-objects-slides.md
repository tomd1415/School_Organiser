# Layers and objects

## Today we are learning
- each object goes on a new layer
- the newest object is on top
- change the order of layers
- use layering to make a picture

## Each object is a layer
![Each object added is placed on a new layer]({{res:l4-new-layer.png}})

- every shape sits on its own **layer**
- the newest object is on **top**
- the first object is at the **back**

> 🧑‍🏫 I-do. Stack three shapes and point out which is on top. Key idea.

## Change the order
- use the **Arrange** menu
- **Send to back** or **Bring to front**
- move one step at a time, or all the way

> 🧑‍🏫 We-do. Model Arrange → Order → Send to back. The worksheet puts these steps in order.

## Fix the emojis
![Emoji made by layering objects]({{res:l4-emoji-layering.png}})

- the faces are layered in the wrong order
- bring the right parts to the front
- so each face can be seen

> 🧑‍🏫 Likely error: deleting a hidden object instead of re-ordering → fix-words "do not delete it — send the front shape to the back."
