## Made from shapes
A vector drawing is built from simple shapes.

![A picture made from shapes]({{res:l1-rect-circle-selected.png}})

When a shape is **selected** it has a blue box and small squares (handles) around it.

## 🟡 Core
Drag each tool to what it does.

```sort
Makes a new shape: shape tool
Draws a straight line: line tool
Picks an object to change it: select tool
Makes the picture look bigger to add detail: zoom tool
```
