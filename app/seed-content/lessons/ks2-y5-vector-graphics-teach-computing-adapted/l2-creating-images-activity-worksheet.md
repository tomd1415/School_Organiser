## What we are learning
Today I will move, resize, rotate and duplicate objects to make a vector drawing.

## The handles
When you click an object you get **handles** (small squares). Drag a corner handle to **resize** it.

![A selected object with handles]({{res:l2-resize-handles.png}})

## Copy a shape
Put the steps to **copy** a shape in order.

```order
Click the shape to select it
Hold Ctrl and press C to copy
Hold Ctrl and press V to paste
Drag the new shape to a new place
```

## 🟢 Support
Drag each tool to what it does.

```sort
move: drag the object to a new place
resize: make the object bigger or smaller
rotate: turn the object round
duplicate: make a copy of the object
```
