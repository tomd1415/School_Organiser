# Make a thing — sums, strings and mini-projects

## Today we are learning
- build up a sum or count inside a loop
- use a `for` loop over the characters of a string
- combine selection, iteration and list/string operations
- choose and start a mini-project

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: accumulate · sum · for · string · character. Same routine as every lesson.

## Starter — sums
`total = total + number` adds the number onto a running total. Do it again and again and the total builds up.

> 🧑‍🏫 Use the starter worksheet. Walk through it on the board, writing the value of `total` each step, so the build-up is visible.

## Sums in a for-loop  (we do)
Instead of repeating the line by hand, put `total = total + number` inside a `for` loop.

> 🧑‍🏫 Support: pick the accumulator line. Core: fill the gap. Likely error: putting `total = 0` inside the loop, so it resets each time — fix-words "set the total ONCE, before the loop."
