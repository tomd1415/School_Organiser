## What we are learning
Today I will add, remove, find, count and reorder items in a list, and choose the right operation for a task.

## Sort the list operations  (we do)
Drag each operation into the job it does.

```sort
Add an item: append, insert
Remove an item: pop, remove
Find or count: index, count
Reorder the whole list: reverse, sort
```
