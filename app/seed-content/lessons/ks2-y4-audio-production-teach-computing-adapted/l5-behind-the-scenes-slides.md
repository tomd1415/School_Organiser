# Behind the scenes

## Today we are learning
- open my saved project to keep working
- add music and sound effects as layers
- set the volume so voices are heard over music
- explain the difference between saving and exporting

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: import · save · export · MP3 · layer · audio engineer. Same routine as every lesson.

## Open your project
- find your saved project
- open it to keep working
- all your tracks are still there

> 🧑‍🏫 Starter. Model opening a saved project. Check everyone can find theirs. If a project is missing, pair them up so no one is stuck.

## Add music and sound effects
![Layers of music, voice and a sound effect]({{res:l5-layered-tracks-music-voice-sound-effect-teach-computing.png}})

- **import** music or a sound effect on a new track
- move it to the right place
- this is layering, like an audio engineer does

> 🧑‍🏫 I-do → we-do. Model import → move → listen. The added sound should help the podcast, not drown out the voice.

## Set the volume
- the voice must be the loudest part
- turn the music down so the voice is clear
- listen after every change

> 🧑‍🏫 You-do. Likely error: music too loud. Fix-words: "voice on top, music underneath." TA: prompt, do not do it for them.

## Save AND export
- **save** the project — keeps your tracks to edit later
- **export** an MP3 — one file that plays on any device
- an exported MP3 cannot be edited easily, so save first

> 🧑‍🏫 Stress the difference: save = editable project; export = finished file to share. Everyone saves, then exports an MP3.

## Listen back
- play a finished podcast
- what works well? what could be better?

> 🧑‍🏫 Plenary. Play one short example podcast (teacher-played, low volume). Gather what worked. Tick the four "I can…".
