# Putting it all together

## Today we are learning
- combine a `while` loop with `if`/`else`
- use a Boolean flag (True/False) to control a loop
- count and limit the number of guesses
- finish and test a complete guessing game

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: iteration · selection · Boolean · flag · and. Same routine as every lesson.

## Starter — which parts repeat?
Pick the lucky number once; ask for a guess and give feedback over and over.

> 🧑‍🏫 Use the card-sort on the starter worksheet. The "pick a number" line must stay OUTSIDE the loop, or the secret changes every guess.

## Flags — True and False  (I do)
![A guessing game that uses a Boolean flag called guessed]({{res:l6-lucky-number-code-teach-computing.png}})

> 🧑‍🏫 A flag is a True/False variable, like the flag at a palace: down (False) or up (True). The loop runs `while guessed == False`. If we never raise the flag, the game never ends — show this, then fix it.

## Your turn — finish the game  (you do)
Raise the flag when the guess is right, count the guesses, and limit the tries.

> 🧑‍🏫 Support: fill the gaps (True, False). Core: raise the flag + count guesses. Challenge: limit to 3 guesses with `and`, reveal the number. Likely error: the flag is set to True every round — fix-words "only raise the flag inside the correct `if`."

## Show what you built
Test your game: win it, and lose it. Save a screenshot.

> 🧑‍🏫 Pairs swap roles. Everyone pastes a link + screenshot on the worksheet. Celebrate finished games — low floor, high ceiling.

## Summative assessment
A short quiz covers the whole unit: input, output, variables, maths, selection and loops.

> 🧑‍🏫 Use your chosen ~20 closed questions (online quiz if possible for instant feedback). This is separate from the worksheet. Keep it calm and low-stakes.

## I can… and reflect
Tick your four "I can…". Write one thing you learnt and rate your confidence.

> 🧑‍🏫 Use the reflection slider on the worksheet. Gather one takeaway each — useful feedback on the unit.
