# Make a change

## Today we are learning
- find blocks that have numbers
- change the value
- say what happens when I change a value

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: value · change · effect · block. Recap: last lesson we joined blocks and ran a program.

## Long programs are tricky
- some programs use the same block over and over
- they get very long
- there is an easier way

> 🧑‍🏫 Show how five move-right blocks is a lot. Set up the problem before the solution.

## The number is the value
![A move block with its value and a number pad]({{res:l3-change-value.png}})

- some blocks have a NUMBER underneath
- this number is the VALUE
- tap it and use the number pad to change it

> 🧑‍🏫 Hunt activity first: pupils go through each block colour and find which blocks have a number. This matches the label task on the starter worksheet.

## One block can do the work of many
![A program with a move-right block set to 5]({{res:l3-value-program.png}})

- one move-right block with value 5
- moves the cat 5 steps

> 🧑‍🏫 Instead of five blocks, use one block and change the value to 5. Same result, fewer blocks. We-do this together.

## What happens if we change it?
- value 5 → moves a long way
- value 2 → moves a shorter way
- bigger number = further

> 🧑‍🏫 You-do: change a value and watch what happens. Likely error: adds more blocks instead of changing the number → fix-words "don't add a block, change the number." Predict-then-run. TA: prompt, do not do it for them.

## I can…
Tick your three "I can…". Show me a value you changed.

> 🧑‍🏫 Note who can find a number block and say what changing it does.
