# Joining blocks

## Today we are learning
- join more than one block together
- use a Start block in a program
- run my program

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: block · join · Start block · End block · run · program · background · algorithm · predict. Recap: last lesson we moved the cat with one command.

## First — add a background
![A classroom background behind the cat]({{res:l2-classroom-background.png}})

- tap the picture button at the top
- choose the Classroom background

> 🧑‍🏫 Model adding the Classroom background. Give time for everyone to add it. Keep it calm — one step at a time.

## Join the blocks
- yellow Start block goes first
- blue move blocks in the middle
- red End block goes last
- they click together like a jigsaw, left to right

> 🧑‍🏫 The pieces only click side to side, not stacked. You can read a program left to right like a book. This matches the order task on the starter worksheet.

## A joined program
![A joined program: Start, three up, five right, End]({{res:l2-joined-program.png}})

- Start → up, up, up → right ×5 → End

> 🧑‍🏫 Read it together left to right. Ask: which way first? (up). This is the predict question on the activity worksheet.

## Run it with the green flag
- tap the green flag to run
- use full-screen so nothing else moves

> 🧑‍🏫 You-do: build a program with a Start block, five move blocks, and an End block, then run it. Likely error: blocks left with a gap, so only the first runs → fix-words "push them together until they click." TA: prompt, do not do it for them.

## Predict before you run
- look at the blocks
- say where the cat will go
- then run it and check

> 🧑‍🏫 Predict-then-run builds reading-code skills. Support: predict the first move only. Challenge: predict the whole path.
