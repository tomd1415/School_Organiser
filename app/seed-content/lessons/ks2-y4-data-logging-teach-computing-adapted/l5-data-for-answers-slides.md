# Data for answers

## Today we are learning
- write a question a data logger could answer
- choose the right sensor
- plan where to put the logger and how long to log for
- set up a data logger to collect data

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: question · sensor · plan · collect. Same routine as every lesson.

## Data answers questions
- people collect data for a reason
- a step counter collects steps to help you keep fit
- we will collect data to answer our OWN question

> 🧑‍🏫 Think-pair-share: who collects data, and why? Keep it short.

## Asking a question
- a good question can be answered with data
- "when is the room brightest?" → light
- "when is it loudest?" → sound

> 🧑‍🏫 Help filter questions so they are achievable with sound, light or temperature. Write a class list of good questions.

## Which sensor?
- brightest / darkest → LIGHT sensor
- loudest / quietest → SOUND sensor
- warmest / coldest → TEMPERATURE sensor

> 🧑‍🏫 This matches the starter. Likely error: choosing a sensor that can't answer the question. Fix-words: "match the sensor to the question word."
