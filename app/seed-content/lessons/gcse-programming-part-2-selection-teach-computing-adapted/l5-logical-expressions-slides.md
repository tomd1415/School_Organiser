# Logical expressions — and / or

## Today we are learning
- put scrambled code in the right order
- use the Boolean operators and / or
- work out if an and / or condition is True or False
- write conditions that use and / or

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: Boolean / logical expression · Boolean / logical operator · and · or.

## Starter — Parson's puzzle
Put the jumbled password-checker lines in the right order.

> 🧑‍🏫 Pupils order the lines on the worksheet. Stress indentation: the prints sit under `if` / `else`. The solution can be self-checked.

## and / or
- `and` → True only when **both** parts are True.
- `or` → True when **at least one** part is True.

> 🧑‍🏫 This is the core idea. Pupils fill the gaps: both / at least one.
