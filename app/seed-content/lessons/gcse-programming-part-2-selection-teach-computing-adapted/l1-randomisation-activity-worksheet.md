## True or pseudo? — sort the cards
A **true** random number comes from the physical world. A **pseudorandom** number is made by a computer using maths.

```sort
True random: atmospheric noise, lightning in the sky, rolling a real dice
Pseudorandom: Python's random module, a monster in a video game, a shuffle in a computer card game
```
