## Starter — comparison operators
We use **comparison operators** to compare two values in a condition.

![Keyboard keys showing the less-than and greater-than symbols]({{res:l4-comparison-operator-keys-teach-computing.jpg}})

Drag the meaning onto each operator.

| Operator | What it means |
|---|---|
| == | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
| != | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
| < | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
| > | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
| <= | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
| >= | (  ) equal to (  ) not equal to (  ) less than (  ) greater than (  ) less than or equal to (  ) greater than or equal to |
