## A flowchart with a decision
A **decision** symbol (a diamond) asks a yes/no question. The arrow you follow depends on the answer.

> 🧑‍🏫 Draw the diamond on the board if you can — the source diagram is a PowerPoint shape, so it is logged as an image gap. Input 10: walk the True and False arrows.

## What is a condition?
A condition uses a **logical expression** that is True or False, e.g. `age > 25`.

> 🧑‍🏫 Arithmetic expressions make a number; logical expressions make True or False. This controls which code runs.

## if · elif · else
![A program using if, elif and else]({{res:l3-what-will-be-the-output-program-teach-computing.png}})

- `if` checks the first condition
- `elif` checks another if the first was False
- `else` runs when none were True
