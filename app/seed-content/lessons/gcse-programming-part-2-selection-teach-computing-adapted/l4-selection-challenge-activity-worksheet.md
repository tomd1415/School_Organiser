## What we are learning
Today I will use selection to **make** a joke machine, working in a pair.

## Pair programming
One person is the **driver** (types). One person is the **navigator** (reads the steps and spots mistakes). **Swap every 5 minutes.**

## Build the joke machine — put the steps in order
```order
print a welcome message and the start of the joke
ask the user to guess the punchline
store the guess in a variable
use an if statement to check the guess
print "well done" and add 1 to the score if correct
use else to give feedback if the guess is wrong
print the final score at the end
```
