## Match if, elif and else
| Keyword | What it does |
|---|---|
| if | (  ) checks the first condition (  ) checks another condition if the first was False (  ) runs when none of the conditions were True |
| elif | (  ) checks the first condition (  ) checks another condition if the first was False (  ) runs when none of the conditions were True |
| else | (  ) checks the first condition (  ) checks another condition if the first was False (  ) runs when none of the conditions were True |
