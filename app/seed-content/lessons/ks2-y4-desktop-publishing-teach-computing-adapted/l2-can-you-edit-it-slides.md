# Can you edit it?

## Today we are learning
- change font style, size and colour for a purpose
- edit text using the keyboard keys
- explain that changing text makes a message clearer

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: font · style · size · colour · edit · template. Same routine.

## What is desktop publishing?
- desktop publishing means making documents with text and images
- it uses page layout software (we will use one app)
- examples: invitations, posters, magazines

> 🧑‍🏫 Keep it concrete. Today we edit a ready-made invitation called a template — we do not start from blank.

## Three ways to change a font
- SIZE — make the text bigger or smaller
- COLOUR — change the colour of the text
- STYLE — change the shape of the letters (the font)

> 🧑‍🏫 This is the starter sort. Likely error: mixing up "size" and "style". Fix-words: "size = how BIG, style = the SHAPE of the letters."

## Edit the invitation

Click the text, then change the size, colour and style.

> 🧑‍🏫 I-do: model clicking the title and changing it. We-do: change one piece together. You-do: pupils edit on the activity worksheet task. TA: prompt, do not do it for them.

## The keyboard keys
- SHIFT — makes a capital letter
- BACKSPACE — deletes the letter before the cursor
- RETURN — starts a new line

> 🧑‍🏫 Show each key on a big keyboard image or the real keyboard. This is the fill-the-gap question on the activity worksheet.
