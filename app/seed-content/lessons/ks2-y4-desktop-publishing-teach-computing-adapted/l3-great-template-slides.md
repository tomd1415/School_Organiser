## What is a template?
- a template is a ready-made layout
- it shows where text and pictures go
- you do not start from a blank page

> 🧑‍🏫 Link back: the party invitation last lesson was a template. Today we make our OWN template for a magazine.

## Page orientation
- PORTRAIT — the page is tall (like a magazine)
- LANDSCAPE — the page is wide (like a postcard)

> 🧑‍🏫 Hold up an A4 sheet both ways. Quick "show me portrait / show me landscape" check. This matches the starter sort.

## Placeholders
![A template with empty placeholders]({{res:l3-placeholders.png}})

- a PLACEHOLDER is an empty box
- it shows where the title, picture or text will go
