# Words and pictures

## Today we are learning
- explain the difference between text and images
- say how text and images send a message clearly
- name good and bad points of text and images
- use emojis kindly online

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: text · images · communicate · advantages · disadvantages · emoji. Same calm routine as every lesson.

## Text and images
- TEXT means words you read
- an IMAGE means a picture
- we use both to send a message (to communicate)

> 🧑‍🏫 "Text" = writing, typing, words. "Image" = picture, graphic, drawing. We use the words text and images all unit. Plain, literal language.

## Guess the sign

Which signs use a picture? Which use words? Which use both?

> 🧑‍🏫 Point at a few. Roundabout / no-smoking = picture only. STOP / speed-limit / Detour = words and a picture. This matches the sort on the starter worksheet.

## Easy or hard to understand?
- a clear picture is quick to understand from far away
- words can add detail, but you must be able to read them
- a sign in another language is hard if you do not speak it

> 🧑‍🏫 Use the "Douane / Zoll" (customs) example: text does not help if you cannot read the language. So pictures are not always best, and words are not always best.

## Good points and bad points
- IMAGES: quick, no reading, seen from far — but can be guessed wrong
- TEXT: more detail, exact — but you must read it

> 🧑‍🏫 This is the card sort on the activity worksheet. Support: choose one good point. Challenge: pick text/image/both for a sign and justify.

## Using emojis kindly
- emojis are tiny pictures that show feelings
- use them to be friendly and kind
- never use them to upset or annoy someone

> 🧑‍🏫 Link to online safety. Likely error: thinking lots of emojis is funny when it can annoy. Fix-words: "a kind emoji helps the message."
