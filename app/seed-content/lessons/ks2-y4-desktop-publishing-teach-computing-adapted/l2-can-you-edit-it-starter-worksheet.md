## What we are learning
Today I will change the size, colour and style of text, and edit a party invitation.

## Starter — which tool does which job?
Desktop publishing software has tools to change text. Drag each tool into the right group.

```sort
Changes the colour: a paint-palette button, a colour drop-down arrow
Changes the size: a slider you drag, a number box like 20
Changes the style: a font-name list like Comic Sans, the T text button
```
