# Lay it out

## Today we are learning
- name different layouts
- match a layout to its purpose
- choose a suitable layout for a message

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: layout · purpose. Same routine.

## What is a layout?
- a LAYOUT is the way text and pictures are arranged on a page
- you can often tell what a document is without reading it
- the layout depends on the PURPOSE (the job it does)

> 🧑‍🏫 Show a document and ask "what is it?" before reading. We know from the layout.

## Magazines and more
![Some magazine covers]({{res:l5-magazine-layout-teach-computing.png}})

- magazine, letter, postcard — each has its own layout

> 🧑‍🏫 This is the matching task on the activity worksheet: layout ↔ purpose.

## Choose the best layout
- quick message for everyone → a poster
- private message to one family → a letter
- detailed news → a newspaper

> 🧑‍🏫 You-do: pupils choose a layout for a scenario and justify it. Likely error: choosing by "what looks nice" not by purpose. Fix-words: "match the layout to the job."
