# Why desktop publishing?

## Today we are learning
- give real-world uses of desktop publishing
- say why desktop publishing is helpful
- compare desktop publishing with work made by hand

> 🧑‍🏫 Read the three "I can…" aloud. Vocabulary on the board: desktop publishing · benefit · real world. Same routine.

## What is desktop publishing?
- making documents with text and images
- using page layout software
- for example: invitations, posters, magazines

> 🧑‍🏫 Recap from the whole unit. Ask pupils to say it in their own words first (the starter fill-the-gap).

## The benefits
- FAST — print many copies quickly
- EASY TO CHANGE — fix a mistake or a time in seconds
- NEAT — every letter is the same and tidy

> 🧑‍🏫 Use a real example: change "2pm" to "3pm" on a saved file in seconds vs redrawing by hand.

## By hand vs desktop publishing
- by hand: draw every copy again, one change means starting over
- desktop publishing: copy, change and reprint easily

> 🧑‍🏫 The "race" idea: pupils draw a poster by hand while you make one on the computer. Compare kindly. This is the card sort on the activity worksheet.
