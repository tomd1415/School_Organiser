# Networking hardware

## Today we are learning
- name network hardware
- say what each one does
- choose hardware to build a network
- say what an ISP is

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on board: network cable · hub · server · router · ISP · ports.

## Starter — what am I?
Four pieces of hardware: network cable, hub, server, router. Match each NAME to what it does on your worksheet.

> 🧑‍🏫 Can pupils spot any network cables in the room? Keep it concrete.

## The network cable
Carries the messages between devices. Often an Ethernet cable, made of copper wires.

> 🧑‍🏫 Cables can send data both ways. Without hardware to connect them, every computer would need a cable to every other computer.

## The hub
![A network hub]({{res:l2-network-hub-teach-computing.png}})

Connects computers in one room or building. It has PORTS for cables. A message coming in is sent on to ALL connected computers.

> 🧑‍🏫 The hub cuts down cables: 4 PCs to a hub is 4 cables, not 6. Ask "what about 20 PCs?".

## The server
![A network server]({{res:l2-network-server-teach-computing.png}})

A powerful computer that provides a service. A FILE server stores shared files in one central place.

> 🧑‍🏫 Link to the school shared drive: the files live on a server, not on each computer. If one PC is off, its files would be lost — a server fixes that.

## Your turn — build the network  (you do)
Add a hub, then a file server, then a router so four computers can share files and reach the internet.

> 🧑‍🏫 Support: a printed block-order strip to follow. Challenge: count how many cables the hub saved. TA: prompt, do not build it for them.

## The router and the ISP
A router joins different networks together — e.g. your home network to the internet. That internet connection comes from an ISP (internet service provider), such as Virgin, Sky or TalkTalk.

> 🧑‍🏫 A router has ports: one to the internet, the rest to your devices.
