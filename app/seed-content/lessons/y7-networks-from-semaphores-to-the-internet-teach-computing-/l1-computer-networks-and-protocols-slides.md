# Computer networks and protocols

## Today we are learning
- say what a computer network is

- say what a protocol is

> 🧑‍🏫 Read the four "I can…" aloud. Same routine as every lesson. Vocabulary on board: network · protocol · relay · stand-alone.

## Starter — sending messages long ago
![A semaphore signal tower]({{res:l1-semaphore-signal-tower-teach-computing.png}})

Before computers, how did people send a message a long way?

> 🧑‍🏫 Semaphore towers were built in a line, each in sight of the next. The message was relayed tower to tower. Use the starter worksheet — tick the ways that do NOT use a computer.

## What is a computer network?
Two or more computers connected together so they can share data.

> 🧑‍🏫 Ask for things we do that use a network (logging on, printing, YouTube). A computer on its own is "stand-alone".

## A message to Australia
A message is PASSED ON device to device — like a letter going depot to depot. It does not jump straight there.

> 🧑‍🏫 Misconception: email "goes straight" or "into the cloud". It hops through many devices. Same idea as the semaphore towers.

## What is a protocol?
A protocol is a set of RULES that both sides follow so the message gets through.

> 🧑‍🏫 Everyday example: meeting someone new — we say hello and shake hands. Rules we both already know.

## Your turn — the climber and the belayer  (you do)
A climber and belayer use fixed commands in a set order to stay safe. Match each command to its meaning on your worksheet.

> 🧑‍🏫 This is a non-networking protocol. Support: match the command to its meaning first. Likely error: guessing the order — fix-words "what must happen FIRST to be safe?". TA: prompt, do not do it for them.

## Computer protocols
Email and web addresses follow rules too: an email needs an @ ; a web address starts http://

> 🧑‍🏫 These are computer protocols. More protocols come later in the unit.
