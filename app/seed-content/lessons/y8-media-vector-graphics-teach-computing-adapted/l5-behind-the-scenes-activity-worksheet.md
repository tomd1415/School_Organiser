## What we are learning
Today I will look at how a vector image is stored as markup, and change an object by editing it.

## Inside an SVG
A vector image is saved as readable text called **markup**. This line of markup describes a square:

```
<rect x="100" y="100" width="200" height="200" />
```

The shapes below are described by markup like this.
