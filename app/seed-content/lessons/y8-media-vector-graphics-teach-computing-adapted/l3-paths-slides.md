## What is a path?
- a path is a route between two points
- it can be straight or curved
- a node marks the start, end, and each turn

> 🧑‍🏫 Use the real-world "path/route" idea. Every line and shape is made of paths you can edit.

## Three kinds of line
- straight — click to place nodes, double-click to finish
- curved — click and drag to bend the line
- freehand — drag the pencil freely

> 🧑‍🏫 Model each once. Closing a path back on its start node makes a shape you can fill. Likely error: not closing the path — fix-words "click back on the first node".

## Editing paths
- choose Edit paths by nodes
- drag a node to move it
- on a curve, drag the handles to change the bend

> 🧑‍🏫 Tip: remove fill/stroke to see nodes more clearly while editing.

## Convert a shape to a path
- draw a shape, then Path ▷ Object to Path
- now its corners are nodes you can drag

> 🧑‍🏫 The shape looks the same until you edit a node. Use this to make the head outline, then send it to the back.
