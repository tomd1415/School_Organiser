## Inside a vector graphic — markup
A vector is stored as readable text called markup:

```
<rect x="100" y="100" width="200" height="200" />
```

![Shapes drawn from SVG markup]({{res:l5-shapes-drawn-from-svg-markup.png}})

> 🧑‍🏫 They have met markup before — the "M" in HTML. x, y = top-left position; width, height = size. Colour is missing, so a default is used.

## Scalable — just change the numbers
- a 200 × 200 square, doubled, becomes 400 × 400
- because it is numbers, it scales perfectly

> 🧑‍🏫 Ask what they would change to double a square. Reveal width/height × 2. "Scalable" = made bigger or smaller without losing quality.

## Edit markup in Inkscape
- Inkscape can show and edit the markup
