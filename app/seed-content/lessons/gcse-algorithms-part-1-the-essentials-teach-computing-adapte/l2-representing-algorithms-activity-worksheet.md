## Order the dice-roll algorithm
A game rolls two dice and checks for a double (both the same). Put the steps in order.

```order
Roll the first dice (a random number 1 to 6)
Roll the second dice (a random number 1 to 6)
Compare the two dice values
If they are the same, tell the player they rolled a double
If they are not the same, tell the player it is not a double
```
