## Sort the algorithms  (I do)
Two searches (linear, binary) and three sorts (bubble, insertion, merge). Which is which?

> 🧑‍🏫 Worksheet has a drag-to-sort task. Recap: searches find an item; sorts put items in order so searching is faster.

## Search and sort the songs  (we do)
The song list is unordered, so only a linear search works — until we sort it.

> 🧑‍🏫 Bubble sort the songs, then a binary search becomes possible. This links sorting and searching together — the big idea of the unit.

## Compare bubble sort and merge sort  (you do)
Bubble sort is simple but slow on big lists. Merge sort is faster (divide and conquer).

> 🧑‍🏫 Support: sort-the-algorithms. Core: which sort does what + bubble pass order. Challenge: which sort for a large list, with a reason. Use thumbs up/down for a quick whole-class check.
