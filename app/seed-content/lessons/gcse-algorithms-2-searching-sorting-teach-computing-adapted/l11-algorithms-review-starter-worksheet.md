## Starter — mystery code
Read this code. The variable names have been hidden, but the steps are familiar.

```python
def mystery(my_list, thing):
    i = -1
    num = 0
    while num < len(my_list):
        if my_list[num] == thing:
            i = num
        num = num + 1
    return i
```
