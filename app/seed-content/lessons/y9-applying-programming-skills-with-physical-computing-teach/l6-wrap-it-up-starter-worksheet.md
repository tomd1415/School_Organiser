## Starter — small board or full computer?
The micro:bit is a **microcontroller**: it runs one program at a time to do a job. A Raspberry Pi or a laptop is a **full computer**: it can do many things at once. Sort each device into the right group.

```sort
Microcontroller (runs one program for a job): the micro:bit, an Arduino
Full computer (does many things at once): a Raspberry Pi, a laptop
```
