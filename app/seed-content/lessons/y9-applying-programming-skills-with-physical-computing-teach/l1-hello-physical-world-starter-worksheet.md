## Starter — input or output?
An **input** is something a computer **senses or receives**. An **output** is something a computer **shows, plays, or sends out**. Drag each one into the right group.

```sort
Input (senses or receives): button, light sensor, microphone, accelerometer
Output (shows or plays): LED display, speaker, vibration motor
```
