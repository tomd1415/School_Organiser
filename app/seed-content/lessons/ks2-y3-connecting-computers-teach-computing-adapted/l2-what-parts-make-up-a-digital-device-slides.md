# What parts make up a digital device?

## Today we are learning
- name the parts of a computer
- sort devices into input and output
- describe a simple process
- design my own digital device

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: input · output · process · device. Same routine as every lesson.

## The parts of a computer
![A desktop computer]({{res:l2-computer-parts-to-label-teach-computing.png}})

- screen · base unit · keyboard · mouse
- you will label these on the starter worksheet

> 🧑‍🏫 I-do: point to each part and name it. The `label` task on the starter has the same image. TA: prompt the name, do not drag it for them.

## Input or output?
- **input** = information goes IN (keyboard, mouse, microphone)
- **output** = information comes OUT (screen, speaker, printer)

![A screen is an output]({{res:l2-a-screen-an-output-device-teach-computing.png}})

> 🧑‍🏫 We-do: hold up or name a device, pupils say "input" or "output". Some fit BOTH (a touchscreen) and some fit NEITHER (a cable) — save those for Challenge.

## Sort the devices
- drag each device into INPUT or OUTPUT
- on the activity worksheet

> 🧑‍🏫 You-do: the `sort` task. Likely error: calling a speaker an input because "you hear it" — fix-words "the sound comes OUT, so it is an output." TA: prompt, do not sort it for them.

## One input, one process, one output
- press a button (input)
- the device does its job (process)
- a light comes on (output)

> 🧑‍🏫 Model the full sentence. Core pupils write the process. Reuse the input/process/output words from last lesson.

## Design your own device
- think of a job it will do
- give it an input and an output
- draw it or build it on screen

> 🧑‍🏫 You-do: the design task in show-your-work. Low floor: one input + one output is enough. High ceiling: many inputs/outputs. Keep ideas calm and concrete.

## I can…
Tick your four "I can…". Show me your device's input and output.

> 🧑‍🏫 Check each pupil can name an input and an output on their design. Next lesson: how digital devices help us.
