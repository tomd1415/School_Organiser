# How are computers connected?

## Today we are learning
- name the devices in a network
- say what a switch does
- say what a server does
- say what a wireless access point does

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: switch · server · wireless access point. Same routine as every lesson.

## The switch
![A network switch]({{res:l5-switch.png}})

- the switch passes messages to the right computer
- every computer joins to the switch

> 🧑‍🏫 Recap from last lesson. Quick check: "what does the switch do?" before adding new devices.

## The server
![A server]({{res:l5-server.png}})

- a **server** is a strong computer
- it stores files so EVERYONE on the network can use them

> 🧑‍🏫 Keep it simple: "the server is where shared files live." It joins to the switch, like the computers do.

## The wireless access point
![A wireless access point]({{res:l5-wap.png}})

- a **wireless access point** sends and receives Wi-Fi
- it lets tablets and laptops join with NO cable

> 🧑‍🏫 Point out the Wi-Fi waves. There is usually one in each room. It still connects to the network by a cable behind the scenes.

## Match each device to its job
- switch → passes messages
- server → stores files
- wireless access point → joins devices with no cable
- do the matching on the activity worksheet

> 🧑‍🏫 We-do the first row together, then pupils finish. Likely error: mixing up server and switch — fix-words "Server Stores, Switch Sends." TA: prompt, do not match it for them.

## Be the network
- role-play switch, server and computers
- send a file to the server through the switch
- order the file journey on the worksheet

> 🧑‍🏫 You-do: calm role-play with string for cables. Challenge pupils order the file journey. Screenshot/photo for show-your-work.

## I can…
Tick your four "I can…". Tell me what the server does.

> 🧑‍🏫 Check pupils can name all three devices and one job each. Next lesson: find these devices on our real school network.
