## What we are learning
Today I will use a **trace table** to walk through a while loop, step by step.

## Starter — what is the output?
Read this loop carefully before you predict.

```python
from time import sleep
count = 5
while count != 0:
    print(count)
    count = count - 1
    sleep(1)
```

## The steps a while loop repeats
Put the steps a while loop takes **in the right order**.

```order
Set the start value of the variable
Check the condition
If the condition is True, run the block
Change the variable
Check the condition again
If the condition is False, stop the loop
```
