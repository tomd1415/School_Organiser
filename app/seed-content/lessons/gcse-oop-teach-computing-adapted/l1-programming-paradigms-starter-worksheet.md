## Starter — good names
A **convention** is an agreed way of doing something so other people understand it.
In code, one convention is giving variables clear, relevant names.

Look at this program on the board. It runs a times-tables quiz, but two variables have unclear names.

| Question | Choose |
|---|---|
| A good name for the variable that holds the answer the user types is… | (  ) d (  ) answer (  ) zzz |
| A good name for the variable that holds the correct result is… | (  ) c (  ) correct_result (  ) x9 |
