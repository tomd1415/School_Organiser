## Starter — what is an animal?
Imagine describing an animal to someone who has never seen one.
Some things are true of **all** animals. Some are true of **only some** animals.

Drag each feature into the right group.

```sort
All animals: it is alive, it can move, it needs food
Only some animals: it has fur, it can fly, it lives in water
```

> Tip: a class only stores what is true for **every** object made from it.
