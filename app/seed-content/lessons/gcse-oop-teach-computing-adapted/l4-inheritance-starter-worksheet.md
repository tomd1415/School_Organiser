## Starter — spot the OOP parts
Here is a small piece of object-oriented code:

```
from monster import Monster
goblin = Monster("Grix", 20, "You shall not pass!")
goblin.speak()
```

Drag each name into what it is.

```sort
Class: Monster
Object: goblin
Method: speak
Attribute: name
```

> Tip: the class is imported and used like a function. The object is what the constructor makes.
