## Mixing light — RGB
On a screen, every colour is made by mixing **red**, **green** and **blue** light. Each one is set by an 8-bit number (0 = none, 255 = full).

## Representation size — worked example
Rule: **resolution (number of pixels) × colour depth (bits per pixel) = size in bits**.

An image is 800 × 600 pixels with 24-bit colour:
- pixels = 800 × 600 = 480,000
- bits = 480,000 × 24 = 11,520,000 bits
