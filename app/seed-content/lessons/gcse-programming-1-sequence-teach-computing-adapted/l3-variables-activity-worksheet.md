## What we are learning
A **variable** is a named box that holds one value at a time. Today I will use variables in a silly sentences program.

## The silly sentences program
![The silly sentences program code]({{res:l3-silly-sentences-code-teach-computing.png}})

```python
noun = "Car"
adverb = "gently"
adjective = "loud"
print(f"The {noun} was {adjective} when it {adverb} went to school")
noun = "Zebra"
adverb = "aggressively"
adjective = "giant"
print(f"The {noun} was {adjective} when it {adverb} went to school")
```
