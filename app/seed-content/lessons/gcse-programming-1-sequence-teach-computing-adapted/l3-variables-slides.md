## A variable is a named box
A variable holds ONE value at a time. `noun = "Car"` puts "Car" in the box called noun.

> 🧑‍🏫 I-do. Stress the misconception fix: a box holds only one value — assigning a new value replaces the old one. It does not keep a history.

## Naming conventions
Good names are clear: `first_name`, `total_score`. Avoid `x`, `a1`, `thingy`.

> 🧑‍🏫 Use the good/bad name card-sort on the worksheet. Python uses lower_case_with_underscores.
