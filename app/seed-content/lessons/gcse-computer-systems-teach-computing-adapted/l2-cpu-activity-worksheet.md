## Sort the jobs
Each part of the CPU has a job. Drag each job onto the right part.

```sort
ALU: does the maths, does the logic (compares values)
Control unit: tells the other parts what to do, sends out the control signals
Registers: tiny, very fast stores inside the CPU
```

## von Neumann architecture
John von Neumann had a big idea: store the PROGRAM and the DATA together in memory as binary. Fill in the gaps.

In von Neumann architecture, both the [[ ]] and the data are stored in memory. The instructions are fetched and run one after another, which we call [[ ]] processing.
