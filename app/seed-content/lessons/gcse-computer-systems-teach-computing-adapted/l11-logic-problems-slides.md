# Logic problems

## Today we are learning
- complete a truth table for a three-input logic circuit
- write a Boolean expression for a logic circuit
- describe how logic gates combine to add binary numbers

> 🧑‍🏫 Read the "I can…" aloud. Vocabulary on the board: Boolean · AND · OR · NOT · logic circuit · expression · truth table. Same routine as every lesson.

## Starter — recap and recall
- add two binary numbers (carry when 1 + 1)
- remember the three gates: AND, OR, NOT

> 🧑‍🏫 Recall from last lesson plus binary addition. Model one addition; convert to denary to check. Common error: starting the truth table at 001 → fix-words "always start at 000."

## A bigger problem
- "A picture is taken if motion OR proximity, AND the light is on"
- inputs: motion, proximity, light — output: picture

> 🧑‍🏫 I-do, dialogic. Work left to right. The big question: "What is new here?" Answer: it needs TWO gates, and the output of one gate feeds into the next.

## Build it methodically
- inputs → output → gates → circuit → truth table → expression
- count the three inputs in binary order: 000, 001, 010 …

> 🧑‍🏫 Model the whole method on the board. Pupils copy the circuit and truth table on PAPER — they must draw gates in the exam. The order task on the worksheet matches this.

## Turn it into a Boolean expression
- Picture = (motion OR proximity) AND light
- brackets show the OR happens before the AND

> 🧑‍🏫 Emphasise the brackets — like maths, they set the order. Then pupils do the worksheet problems (problem 2 adds a NOT gate).

## Gates do maths
![A circuit board built from logic gates]({{res:l11-a-circuit-board-built-from-logic-chips-teach-computing.png}})
- join gates together and a computer can ADD binary
- a half adder is just a few gates working together

> 🧑‍🏫 Links back to the FDE cycle: the arithmetic logic unit is made of gates. This is why "logic gates are the building blocks of computers."
