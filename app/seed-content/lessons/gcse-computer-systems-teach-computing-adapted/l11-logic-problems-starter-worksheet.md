## Remember the three gates
Pick the rule for each gate.

| Gate | What it does |
|---|---|
| AND | (  ) output is 1 only if both inputs are 1 (  ) output is 1 if at least one input is 1 (  ) gives the opposite |
| OR | (  ) output is 1 only if both inputs are 1 (  ) output is 1 if at least one input is 1 (  ) gives the opposite |
| NOT | (  ) output is 1 only if both inputs are 1 (  ) output is 1 if at least one input is 1 (  ) gives the opposite |
