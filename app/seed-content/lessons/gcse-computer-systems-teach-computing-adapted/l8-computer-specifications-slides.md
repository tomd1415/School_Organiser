## Starter — the clock is ticking
![A clock — the CPU keeps time with a steady pulse]({{res:l8-a-clock-the-cpu-keeps-time-with-a-steady-pulse-teach-computing.png}})
- the clock sends a steady pulse, like a heartbeat or a metronome
- an instruction runs on each pulse

> 🧑‍🏫 Single-correct tick on the starter worksheet. Likely error: "the clock measures how long the computer has been on" → fix-words "the clock keeps the BEAT, it does not measure time." Faster pulse = more instructions per second.

## Three things that affect CPU speed
- clock speed — how many instructions per second (measured in GHz)
- cache — a small, very fast store the CPU uses a lot
- cores — how many jobs the CPU can work on at the same time

> 🧑‍🏫 I-do each one with an everyday picture: clock speed = how fast you read; cache = the notes on your desk vs a book in another room; cores = more people sharing the work.

## Compare two processors
- old CPU: 1.5GHz, 8MB cache, 2 cores
- new CPU: 3GHz, 16MB cache, 4 cores

> 🧑‍🏫 We-do on the activity worksheet. Likely error: "doubling cores = twice as fast" → fix-words "only if the job can be SPLIT; one job that can't be split won't speed up." Bigger cache = fewer slow trips to RAM.

## The motherboard
- the main board everything plugs into
- it connects the CPU, memory and storage so they can talk

> 🧑‍🏫 Define the key word: motherboard. Point to one in a real PC if you have one to tear down.
