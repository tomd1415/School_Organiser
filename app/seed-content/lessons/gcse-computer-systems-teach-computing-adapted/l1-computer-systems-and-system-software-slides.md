## General purpose vs embedded
- a GENERAL PURPOSE computer does many jobs — the user chooses (laptop, phone)
- an EMBEDDED system is built into one machine to do ONE job (washing machine, coffee maker)

> 🧑‍🏫 A phone runs any app; a coffee maker only makes coffee — but does it better. Likely error: "a phone is embedded because it's small" → fix-words "embedded = ONE fixed job, not about size."

## Inside the computer — the hardware
![A computer chip (the CPU)]({{res:l1-a-computer-chip-the-cpu-teach-computing.png}})

- CPU = the brain (processes instructions)
- RAM = fast memory while programs run
- ROM = keeps the start-up instructions
- Hard drive = keeps your files when the power is off

> 🧑‍🏫 This sets up the start-up matching task. RAM empties when you switch off; ROM and the hard drive do not.

## System software
- SYSTEM SOFTWARE runs the computer itself
- the OPERATING SYSTEM (OS) is the most important piece
- UTILITY SOFTWARE = small tools (antivirus, backup, compression)

> 🧑‍🏫 The OS manages hardware, memory and the user interface. Contrast with application software (the games and apps the user runs).
