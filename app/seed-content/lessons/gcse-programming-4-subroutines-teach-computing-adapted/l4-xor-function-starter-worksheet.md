## 🟢 Support
| Question | Choose |
|---|---|
| `A AND B` is True only when… | (  ) both A and B are True (  ) just one is True (  ) neither is True |
| `A OR B` is True when… | (  ) at least one of A or B is True (  ) both must be False (  ) never |
