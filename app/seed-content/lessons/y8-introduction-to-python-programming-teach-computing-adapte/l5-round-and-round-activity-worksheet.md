## Build the countdown  (we do)
These lines count down from 10 and then print "Lift off", but they are jumbled. Drag them into order.

```parsons
count = 10
while count >= 1:
  print(count)
  count = count - 1
print("Lift off")
```

## Inside or outside the loop?
Some lines run once, and some run every time round the loop. Drag each line into the right group.

```sort
Runs once (outside the loop): count = 10, print("Lift off")
Runs every time round the loop: print(count), count = count - 1
```
