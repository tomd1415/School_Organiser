## Build the multi-branch program  (we do)
These lines give clothing advice for four kinds of weather, but they are jumbled. Drag them into the right order.

```parsons
print("What's the weather like?")
weather = input()
if weather == "cloudy":
  advice = "No sunglasses"
elif weather == "rainy":
  advice = "Get an umbrella"
elif weather == "snowy":
  advice = "Mittens and earmuffs"
else:
  advice = "No particular advice"
print(advice)
```

## How a while loop repeats
A `while` loop checks a condition before each round. Put the steps of ONE pass of a `while` loop in order.

```order
Check the condition
If the condition is true, run the lines inside the loop
Go back to the top of the loop
Check the condition again
When the condition is false, stop and carry on below the loop
```
