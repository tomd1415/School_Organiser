## if / elif / else
`if` checks the first condition, each `elif` checks another, and `else` catches anything left.

> 🧑‍🏫 Show the flow with your hands: Python checks each condition in order and runs the FIRST one that is true, then skips the rest.

## Something missing — repeating
What if we want the program to ask again and again? That is iteration — a `while` loop.

> 🧑‍🏫 Recall Scratch loops (repeat / forever). A `while` loop keeps going while its condition is true and checks the condition before each round.

## How a while loop works
Check the condition → run the block → go back → check again → stop when false.

> 🧑‍🏫 Use the "order the steps" task on the worksheet. Walk through ONE pass slowly. Common error: thinking the loop checks the condition in the middle — it checks at the top, before each round.
