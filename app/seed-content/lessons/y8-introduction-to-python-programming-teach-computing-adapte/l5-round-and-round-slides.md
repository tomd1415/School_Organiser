# Round and round

## Today we are learning
- use a `while` loop with a counter variable
- start, change and check a counter
- build a countdown that stops at the right number
- use a loop to ask several questions

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: iteration · counter · condition · while. Same routine as every lesson.

## Starter — trace the counter
`count = 3` then `count = count - 1` makes count one smaller each time.

> 🧑‍🏫 Use the trace task on the starter worksheet. Walk through it line by line, writing the value of count on the board. This is the building block of every counter.

## A countdown loop  (we do)
A counter starts at a value, changes each round, and the condition decides when to stop.

> 🧑‍🏫 Use the Parsons task. Then the card-sort: which lines run once (set-up, "Lift off") vs every round (print, change the counter). Indentation shows what is inside the loop.

## Your turn — count and stop  (you do)
Build a countdown, then a game that asks exactly 3 questions.

> 🧑‍🏫 Support: choose the right condition + fill the gap. Core: countdown with Lift off. Challenge: 3-question game and explain where the counter changes. Likely error: forgetting to change the counter → loop never ends — fix-words "every loop needs the counter to change."
