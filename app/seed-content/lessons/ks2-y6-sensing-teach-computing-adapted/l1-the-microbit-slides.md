# The micro:bit

## Today we are learning
- name the main parts of a micro:bit
- make a simple program in MakeCode and test it on the emulator
- put a program onto a real micro:bit (flash it)

> 🧑‍🏫 Read the three "I can…" aloud, slowly. Words on the board: micro:bit · MakeCode · input · output · emulator · flash. Same routine as every lesson.

## Starter — what is a micro:bit?
![A micro:bit with its parts labelled]({{res:l1-microbit-parts.png}})

- a micro:bit is a tiny computer
- it takes inputs (button, shake) and makes outputs (lights, sound)

> 🧑‍🏫 Hand out the micro:bits (no wires yet). Let pupils look at the front and back. On the starter worksheet, Support pupils choose the part that senses movement (the accelerometer).

## The parts of a micro:bit
![The micro:bit board]({{res:l1-microbit-parts.png}})

- buttons A and B, LED screen, USB, pins
- the accelerometer senses movement

> 🧑‍🏫 Likely error: calling the whole board "the accelerometer." Fix-words: "the accelerometer is one PART inside the micro:bit — it senses movement." Challenge pupils give an input and an output.

## MakeCode — where we build programs
![The MakeCode screen]({{res:l1-makecode-environment.png}})

- we drag blocks together to make a program
- blocks must go inside on start or forever

> 🧑‍🏫 Open makecode.microbit.org and make a New Project. I-do: drag a show string block inside on start. Likely error: blocks left grey/outside a wrap-around block — they will not run.

## Make and test your program
![The on start and forever blocks]({{res:l1-onstart-forever.png}})

- run your program on the emulator (the pretend micro:bit on screen)
- check it works before using a real device

> 🧑‍🏫 We-do then you-do: pupils scroll text or light the LEDs and press play to test on the emulator. TA: prompt, do not build it for them.

## Put it on a real micro:bit
![A micro:bit with its parts labelled]({{res:l1-microbit-parts.png}})

- Download → copy the file onto the micro:bit → wait for the light to stop blinking
- this is called flashing

> 🧑‍🏫 Pupils order the flash steps on the activity worksheet, then flash a real micro:bit. Plenary: an accelerometer is also in phones, smart watches and game controllers.
