## What we are learning

## MakeCode — where we build programs
We build programs by dragging blocks together. First, make a **New Project**.

![The MakeCode home screen with New Project]({{res:l1-makecode-home.png}})

Blocks are grouped by colour (Basic, Input, Loops, Variables…).

![The MakeCode programming screen]({{res:l1-makecode-environment.png}})

Every program needs a wrap-around block like **on start** or **forever**. Your blocks go *inside* it.

![The on start and forever blocks]({{res:l1-onstart-forever.png}})

## Test before the real device
The **emulator** is a pretend micro:bit on the screen. You can run your program on the emulator to check it before you use a real micro:bit.

## Putting your program on a real micro:bit
Put these steps in order. This is how you "flash" your program onto the micro:bit.

```order
Build your program in MakeCode
Click Download to save the file
Plug the micro:bit into the computer with the USB lead
Copy the file onto the micro:bit
Wait for the light to stop blinking — your program is on the micro:bit
```
