# Why cloud computing?

## Today we are learning
- describe what cloud computing is
- sort the functions and features of the cloud
- judge advantages and disadvantages of the cloud
- choose the best use of cloud computing

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: cloud · online software · server · scalability · security. Same routine as every lesson.

## Starter — what is the cloud?
![A cloud linked to laptops, tablets and phones]({{res:l4-the-cloud-and-many-devices-teach-computing.png}})

The cloud = software and storage you use over the internet, on any device.

> 🧑‍🏫 Concrete examples first: streaming, web email, shared docs. Then the starter sort: function (a service) vs feature (a good quality).

## Functions and features
- FUNCTIONS (services): SaaS, IaaS, PaaS
- FEATURES (good qualities): easy access, low cost, security, scalability

> 🧑‍🏫 Keep SaaS/IaaS/PaaS light — "software / infrastructure / platform you rent online". The sort is the goal, not the jargon.

## Good things about the cloud
- reach files anytime, anywhere
- the provider manages security and back-ups
- add more storage when you need it

> 🧑‍🏫 Matches the activity card sort. "Scalability" = grows and shrinks with your needs.

## Watch out
- you need a good internet connection
- extra storage can cost more
- less control over where your data sits

> 🧑‍🏫 Likely error: thinking the cloud always works. Fix-words: "no internet, no cloud." Link to the rural-worker predict question.

## Which is best?
- Scenario 1: data in the cloud, weak password, patchy internet → risky
- Scenario 2: collaborate in the cloud, sensitive data on secure servers, access levels → best

> 🧑‍🏫 This is the Challenge task. Push them to use "security" and "access" in their answer.
