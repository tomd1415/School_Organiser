## Access tools
- SCREEN READER — reads the screen aloud (helps low vision)
- ALT TEXT — describes a picture (helps low vision)
- SPECIALIST KEYBOARD and SWITCHES — help a physical disability
- VOICE RECOGNITION — type by speaking

> 🧑‍🏫 Matches the "who needs what?" matching task. Stress that the SAME technology can open up a job for many different people.

## Collaboration and communication
![Two heads sharing ideas]({{res:l2-collaboration.png}})

- COLLABORATION = working together on the SAME thing
- COMMUNICATION = sending MESSAGES to each other

> 🧑‍🏫 Common mix-up. Anchor it: editing one doc together = collaboration; emailing/chatting = communication. Both are everyday work skills.

## Working together online — good and bad
- GOOD: edit together, reach files anywhere, track progress, finish faster
- BAD: ideas can clash, skill levels differ

> 🧑‍🏫 This matches the Challenge card sort. Keep it balanced — the tools are powerful but not perfect.
