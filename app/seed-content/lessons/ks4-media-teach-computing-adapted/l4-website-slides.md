# Creating a multi-page website

## Today we are learning
- describe features of a good website
- plan a multi-page website
- build a multi-page website with HTML
- add your own logo and video to the site

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on the board: navigation · call to action · hosting · SSL certificate · HTML · CSS. Same routine as every lesson.

## Starter — what makes a website good?
- some features visitors SEE: home page, navigation, logo, call to action
- some run on the SERVER: hosting, SSL certificate, content management system

![A website hosted on a server, used by several computers]({{res:l4-website-hosting.jpg}})

> 🧑‍🏫 Take three "good website" ideas, then sort front end vs back end. This is the card-sort on the starter worksheet.

## HTML and CSS
![Lines of HTML code]({{res:l4-html-code.jpg}})

- HTML = the content and structure (tags inside `< >`)
- CSS = the design (colours, fonts, layout)

> 🧑‍🏫 Show one `<a>` link tag and one `<img>` tag. Don't overload — they edit a template, they do not write from scratch.

## Build from a template
- copy the template folder
- change the headings and text on each page
- check it in a browser

> 🧑‍🏫 The activity worksheet has these steps to drag into order. Stress: open a COPY in your own area, then refresh the browser after each change.

## Add your logo and video
- save your logo as logo.png in the images folder
- add the video you made last lesson to the video page

> 🧑‍🏫 This links the whole unit together: their Lesson 2 logo and Lesson 3 video go into the site. Likely error: filename not exactly logo.png — fix-words: "the code looks for logo.png exactly."
