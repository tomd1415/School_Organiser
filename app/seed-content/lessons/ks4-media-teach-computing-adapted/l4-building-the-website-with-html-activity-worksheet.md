## What we are learning
Today I will build a multi-page website from a template using HTML, and add my own logo and video.

## HTML — the code behind a web page
![Lines of HTML code on a screen]({{res:l4-html-code-on-a-screen-teach-computing.jpg}})

HTML is made of **tags** inside `< >`. A link uses an `<a>` tag. An image uses an `<img>` tag. CSS controls the **design** (colours, fonts).

## Order the website build steps
Drag the steps into the right order.

```order
Copy the template folder to your work area
Open index.html in a text editor
Add your logo by saving it as logo.png in the images folder
Make the extra pages (mower, video, contact) by copying index.html
Change the headings and text on each page
Open the site in a browser to check it works
```
