- name file formats for each type
- make a graphic with open-source software

## Starter — raster vs vector
- RASTER = made of pixels (tiny squares), like a photo. Goes blurry if enlarged.
- VECTOR = made of maths paths. Stays sharp at any size, smaller file.

> 🧑‍🏫 Demo: zoom into a photo (blocky pixels) vs zoom into a vector logo (still sharp). The starter worksheet sorts file types into raster/vector.

## File formats
- raster: .jpg · .png · .gif
- vector: .svg · .eps · .ai

> 🧑‍🏫 Quick check on fingers: raster or vector for each. Keep it light — recognition, not memorising all of them.
