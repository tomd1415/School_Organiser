## What we are learning
To move in four directions, each arrow key needs its own block stack. We make one, then copy it and change the direction.

## Look at the code
Two arrow keys, each with its own event and action.

![Two block stacks: when up arrow pressed point in direction 0 move 10; when right arrow pressed point in direction 90 move 10]({{res:l2-four-direction-code.png}})

## Put it in order
Drag these steps into the right order to program one direction.

```order
Add a sprite to the maze
Resize the sprite so it fits the paths
Add a "when arrow key pressed" event block
Add a "point in direction" block
Add a "move 10 steps" block
Test the sprite moves
```
