# Translate this! (Part 1)

## Today we are learning
- explain what decomposition means
- break a large problem into smaller tasks
- translate a word using the Translate extension
- start building my translation quiz

> 🧑‍🏫 Read the four "I can…" aloud. Vocabulary on board: decomposition · subproblem. This is a paired project across Lessons 11 and 12.

## Starter — decomposition
![A learner at a laptop with hello in several languages]({{res:l5-hello-in-many-languages-teach-computing.png}})

Think · write · pair · share: give an example of decomposition.

> 🧑‍🏫 Remind them: if a problem feels too hard, it usually means it hasn't been decomposed enough yet.

## The scenario  (we do)
![The translation-quiz scenario brief]({{res:l5-translation-quiz-scenario-brief-teach-computing.png}})

A quiz that tests ten common words. The big problem is already broken into tasks.

> 🧑‍🏫 In pairs, read the scenario and predict where they'll get stuck — most will say Task 4. That's fine: it just needs more decomposing.

## Break Task 4a down further  (we do)
"Translate the word into the desired language using the Translate extension."

> 🧑‍🏫 Reassure them it's OK to feel confused. Do the Decompose-4a steps together: install the extension → say a translation → store a word in a variable → translate the variable.

## Start the project  (you do)
Work in your pair. Use the support sheets if you need them.

> 🧑‍🏫 Pair programming — swap roles regularly. Hand out support sheets (Tasks 1–6) as scaffolding; they slowly remove help. Go through the rubric so pairs know what "good" looks like. Likely error: the Translate block stays in English → fix-words: "check the language dropdown is set."

## I can…
Tick your four "I can…". Tell your partner one thing you found easy and one you struggled with.

> 🧑‍🏫 Plenary: pairs swap advice. Review the summary slide. Note who reached a working translation.
